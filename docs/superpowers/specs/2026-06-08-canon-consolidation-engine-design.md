- 상태: 설계 합의 완료, 검토 대기
- 프로젝트: `C:/ProjectS/canon-forge` (트랙 A — 정리 엔진)

## 1. 문제와 목표

`THE FORGOTTEN SUMMONER` 세계관 볼트(원본 37MB, ~4,041 md)가 정리되지 않은 채
쌓여 있다. 인물·세력·지명·유물·규칙이 여러 출처에 중복·모순·드리프트된 상태로 흩어져 있고,
이를 정리하려는 3개월간의 Codex "확장(expansion)" 시도는 수렴하지 못하고 실패했다.

**목표:** 37MB 원본을 **깨끗하고 일관되며 질의 가능한 캐논**으로 정리한다.
- 작아지는 방향(중복 제거)
- 모순이 0으로 **수렴**
- 개연성·흥미가 본문에 **녹아 있는** 상태
- **끝이 정의된** 작업

## 2. 비목표 (Non-goals)

- ❌ **확장(expansion)이 아니다.** 13MB로 키우는 것이 목표가 아니다. 크기는 *결과*이지 목표가 아니다.
- ❌ 독자 리더 앱(openending)은 이번 범위가 아니다. (정리된 캐논 위에 나중에 얹는다.)
- ❌ 평가 점수표/루브릭을 만들지 않는다. 개연성·흥미는 *짓는 방식*에 녹인다.
- ❌ Hermes 에이전트 운영층 / 영상화(claude-video)는 이번 범위가 아니다. (YAGNI)

## 3. 지배 원칙

1. **반(反)의식(anti-ceremony).** 산출물은 5개로 제한(§11). 배치별 QA 문서·게이트 문서·74개 관리문서 금지. 과거 실패의 핵심 원인이었다.
2. **수렴 정의.** 진척 신호는 단 하나 — *미해결 열린 이슈 수*. 0으로 가면 끝(§9).
3. **무한 연기 금지.** 충돌은 *해소* 아니면 *공개(collisions에 노출)*, 둘 중 하나. "batch 011로 hold" 식 무한 미루기 금지.
4. **원본 불변.** 원본 볼트는 읽기 전용. 절대 수정·복사하지 않는다. 산출물은 엔진 레포의 `out/`에만.
5. **재발명 금지(Karpathy).** 이미 있는 도구(Understand-Anything)·구조(위키링크·템플릿)·자산(기존 register)을 *사용*한다. 파싱으로 될 일을 LLM에 시키지 않는다.
6. **작게 먼저 증명.** 미니 슬라이스 → 한 세력 서브트리 → 전체 Astralis 순으로 검증 게이트를 통과한다.

## 4. 범위 (v1)

**v1 = Astralis 크로니클 전용.**
근거: 볼트의 99.2%(4,029/4,041)가 Astralis 단일 연대기다. 나머지 5개 크로니클

## 6. 아키텍처 / 파이프라인

```
[원본 37MB · 읽기전용]
   │
 ① 인벤토리 (1회, 결정적) ── 전 파일 스캔 → manifest(경로·해시·타입)
   │
 ② 추출 (배치·재개) ── 하이브리드:
   │    • 기계: [[위키링크]] 파싱 → 엣지 / 템플릿 섹션 파싱 → 노드 골격 (LLM 불필요)
   │    • LLM: 사실·속성·대가 의미·톤만 추출
   │    해시 안 바뀐 파일은 재추출 안 함 (agentmemory 영속)
   │
 ③ 검출 (그래프 전역) ── 열린 이슈 산출 → out/collisions.md
   │
 ④ 해소 루프 ── 확신↑ 자동 / 애매 사람 플래그. 결정은 .memory에 영속(재질문 금지)
   │
 ⑤ emit ── out/canon/ (정리·보강된 캐논) + out/graph/
```

- **하네스:** Workflow 도구로 ②추출을 파일 배치 팬아웃(동시 ~10). ③검출에서 "신규 이슈 0건 N회" 수렴 조건.
- **메모리:** agentmemory + `.memory/`로 처리 파일·해소 결정 영속 → 중단/재개 무한 반복.
- **추출 엔진:** Understand-Anything(`article-analyzer` 등)을 노드/엣지 추출·검증에 사용.

## 7. 그래프 모델

**노드 타입:** 인물(성장영웅/소환영웅/주역), 세력, 지명, 유물, 마법체계(8대 백과), 몬스터, 사건/연표, 규칙, act.

**엣지 타입:**
- 일반 관계: `serves / allied / defeats / knows / located-in / wields / uses-magic ...` (위키링크에서 파싱)
- **정체성 엣지 (1급 시민, §8 규칙의 근거):**
  - `is-husk/identity-of → 에반` : 에반의 무의식 더미·다중신분. **다중존재 정당.**
  - `summoned/allied-with → 에반` (성장영웅 등) : 독립 인물. **일반 일관성 규칙 적용**(분신 예외 없음).
- **대가 엣지:** `costs → {lifespan / sanity / memory / body / existence}` + 등급·지속한계.
- **구조 엣지:** `act-of → 크로니클`, `belongs-to-act → act`, `plants-setup / pays-off-setup`(복선 추적).

**핵심 상수:** 루미에(Lumie)는 6 크로니클을 모두 관통하는 유일 엔티티 → 교차차원 정점으로 취급.

## 8. 검출: 열린 이슈 클래스 & 세계 고유 불변식

**열린 이슈 클래스 (→ 0으로 수렴):**
1. **모순** — 같은 엔티티가 출처 간 사실 충돌 (특히 영웅백과 vs 세력 아카이브 다중출처)
2. **개연성 구멍** — 원인 없는 결과, 대가 없는 힘, 이동·획득 경로 누락, 동기 없는 행동
3. **고아/갭** — 참조되나 미정의(link-rot 포함), 빈 act
4. **복선 미회수** — `plants-setup` 했으나 해당 act의 이스식 결말까지 `pays-off` 없음
5. **이름 증식** — 동일 대상 다중 표기 (시드: 솔라리안=태양제국=아르카나=마도제국, Skadi 인간vs정령, Wulfgard/Wulfgar, Priya/Freya, Thanatarch/Necros)
6. **납작한 곳** — 긴장(=대가)·고유 내적논리 없는 설정 (보강 대상; 카운트 아님)

**세계 고유 불변식 (구체적·검사 가능):**
- **대가 단조 누적:** 에반의 대가(수명·이성·기억·육신)는 act를 거치며 단조 증가. *이유 없는 회복 = 모순.*
- **등급↔대가 정합:** 대가는 등급(SSS→D) 따라 증가. *D급이 수명 대가를 치르면 모순.* (`00-B` 티어 기준)
- **정체성 다중존재 규칙:** 에반의 `is-husk-of` 다중존재는 정당. 그 외 인물의 동시 다중존재는 모순. *(이게 없으면 false positive 폭발 — 특히 "이중 존재 역설" 장면)*
- **세력 계층 보존:** core 지배구조와 pressure/hold 주변부를 평탄화하지 않음(register FC 분류 계승).

## 9. 수렴 정의 (= "끝"이 있다)

> **Astralis 전 파일 추출 완료 ∧ 모순 0 ∧ 개연성 구멍 0 ∧ 고아 0 ∧ act별 복선 회수 완료.**

- 진척은 **act 계획 대비 %**로 표시(과거에 없던 신호).
- "납작한 곳/흥미"는 emit 작법으로 녹이므로 카운트 대상이 아니다(평가표 금지).

## 10. 보강 & emit

보강은 "키우기"가 아니라 **③이 플래그한 열린 이슈를 닫는** 활동이다(gap-driven, bounded).
보강분은 그래프에 **재검증**되어 새 모순을 만들면 거부된다.

**테마 (작법 북극성):**
- **인간 이데올로기 > 신.** 신·마왕·차원은 무대, 드라마의 중력 중심은 인간의 의지. 데우스 엑스 마키나 금지. 개연성·흥미를 *인간 행위주체성*으로 푼다.

**구조:**
- **이스(Ys)식 act 결말** — 각 act는 그 땅의 미스터리/재난을 자기완결로 해소 → 에반이 흉터(누적 대가)를 안고 다음 act로. 복선은 act 안에서 회수.

**프로즈 보이스 (v1 결정: 무게 유지 + 속도 주입):**
- 유지: 전민희(룬의 아이들 + 태양의 탑)의 문학적 깊이·인물 내면, 기존 무게·신화 톤, 대가가 문장에 밴 질감.
- 주입: 약먹는 천재마법사의 "글근육" — 읽힘·추진력·속도(현 볼트는 무게는 충분하나 속도가 약함).
- 추가 북극성: 영웅전설/궤적(강박적 연속성=일관성 금본위), 하얀 늑대들(종족·문화 고유 내적논리), 이스(신화적 기층).
- 제네릭 판타지 금지.

## 15. 다운스트림 (트랙 B 연결)

정리된 캐논은 openending(독자 선택형 리더)의 토대가 된다.
- 각 act = 이스식 자기완결 플레이 아크
- 에반의 누적 대가 = 결말 분기축
- novel-canon-mcp = "세계가 거짓말하지 않는다"를 보장하는 질의/검증 인터페이스
- 구조(에반 허브 + 분신·다중신분, 한백무림서식 연작)가 그대로 관통

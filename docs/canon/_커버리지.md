# 원본 커버리지 체크리스트 (Source Coverage Ledger)

> **이 문서의 단 하나의 역할**: 원본 폴더가 **실제로 캐논에 반영됐는지**를 원본 단위로 추적한다.
> 진행 추적의 **정본**은 이 표다. `_progress.md`는 Phase 서술, 이 표는 원본 소비 여부.
>
> ⚠️ **규칙**: "Phase 완료"는 해당 Phase의 모든 행이 ✅일 때만 선언한다. 캐논 문서를 썼는지가 아니라 **원본 폴더가 소비됐는지**로 판정한다. (과거 Phase 2가 세력 아카이브를 빠뜨린 채 '완료'로 찍힌 사고의 재발 방지.)
>
> 범례: ✅ 완료(전수 통합) · 🟡 부분 · ⬜ 미착수 · ⏭️ 의도적 보류(사유 명시)

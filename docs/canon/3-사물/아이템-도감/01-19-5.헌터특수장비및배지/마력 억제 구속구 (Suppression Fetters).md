# 🎖️ 마력 억제 구속구 (Suppression Fetters)

> 사냥꾼들이 짐승의 똥밭에서 구르는 이유이자 이족 보행 괴수들을 해체하는 가장 잔혹한 사냥 도구.

| **구분** | **설명** |
| --- | --- |
| **카테고리** | 소모품 / 사냥 도구 |
| **등급** | **희귀급 (Rare)** 🔵 |
| **소유자/보관소** | 일반 헌터 길드 보급소 배포 |
| **재료** | 독액을 내부에 순환시키는 관 구조의 사슬, 마나 신경망 압박용 미늘 돌기, 짐승 힘줄로 엮은 결속 가죽끈 |
| **거래 가능 여부** | 가능 (길드 보급소 정식 유통 — 생포 등급 의뢰 접수자 한정 판매) |
| **강화 가능 여부** | 불가 (소모형 독액 카트리지 교체 방식 — 구속구 본체는 강화 대상이 아님) |

## 📜 1. 유래 및 순리 (Origin & Providence)
헌터의 무장은 일반 군대의 규격화된 강철 무기나 예지 마법이 수놓아진 으스대는 지팡이가 아닙니다. 생존율이 한없이 바닥을 기는 밑바닥 헌터들의 피상적인 생존본능과 뒷골목 연금술이 결합되어 탄생한 잔혹한 도구입니다.

### 외형 묘사
겉보기에는 평범한 쇠사슬 구속구처럼 보이지만, 사슬 마디마다 속이 빈 관이 뚫려 있고 그 안으로 짙은 녹색 독액이 순환한다. 사슬 표면에는 짐승의 마나 신경망을 물리적으로 파고들도록 설계된 미세한 미늘 돌기가 촘촘히 돋아 있어, 손으로 만지면 사포처럼 거칠다. 손잡이 부분은 짐승 힘줄을 꼬아 만든 가죽끈으로 감겨 있으며, 오래 쓸수록 독액 특유의 부식성 때문에 사슬 표면이 검푸르게 변색된다. 미사용 시에는 독액이 새지 않도록 관 끝을 밀랍으로 봉인해두는데, 봉인이 낡아 갈라진 구속구는 보급소에서 즉시 폐기 조치한다 — 착용자 본인이 독액에 노출될 위험이 있기 때문이다.

## ⚖️ 2. 대가 (Cost)
마력 억제 구속구는 강제 등가 대가나 등급별 강제 비용과 무관하다. 일반 물리·마나 제압 계열 도구이므로 권능 위계 차이만 존재할 뿐, 사용 자체에 따르는 강제 대가는 없다. 다만 독액 발현 시 착용자(포획자)의 손바닥 피부에 피부 괴사 및 중금속성 검은 흉터가 영구적으로 남는 것은 마법 법칙의 청구가 아니라, 도구 자체의 물리·화학적 부작용이다.

### 고유 능력: 《신경망 결박(Neural Bind)》
- **발동 조건**: 대상 짐승의 사지 혹은 몸통에 사슬을 직접 결속
- **효과**: 사슬의 미늘 돌기가 짐승의 마나 신경망을 물리적으로 파고들어 압박하고, 관 내부의 독액이 신경 전달을 차단하며 근육을 서서히 이완·연화시킨다. 이를 통해 죽이지 않고 산 채로 제압·포박하는 생포 임무가 가능해지며, 상업적 가치가 높은 마수를 훼손 없이 확보할 때 필수적으로 쓰인다.
- **한계**: 독액이 사슬 관을 타고 새어 나올 경우 착용자 본인의 손바닥도 함께 노출되며, 이 경우 피부 괴사와 중금속성 검은 흉터가 영구히 남는다. 그래서 숙련된 포획자들은 항상 여벌 장갑을 겹쳐 끼며, 봉인이 낡은 구속구는 절대 재사용하지 않는다.

## ⚔️ 3. 섭리의 권능 (Specialized Arts)
짐승의 마나 신경망을 직접 압박해 근육을 녹이며 생포

## 📜 4. 배경 서사 & 전승 (Lore)

### 전승
마력 억제 구속구는 애초에 살상이 아니라 포획을 위해 고안된 몇 안 되는 헌터 도구다. 상업 길드와 마도 연구소가 산 채로 마수를 확보하려는 수요가 늘어나면서, 뒷골목 연금술사들이 기존의 사슬 구속구에 독액 순환 구조를 결합한 것이 시초라 전해진다. 다만 이 도구를 다루는 헌터들 사이에서는 "짐승보다 내 손이 먼저 죽는다"는 자조적인 농담이 돈다 — 그만큼 착용자 본인의 부상률도 높기 때문이다. 그럼에도 생포 의뢰 한 건의 보수가 통상 토벌 의뢰보다 훨씬 높아, 이 도구를 능숙하게 다루는 포획 전문 헌터들은 손바닥 전체가 검게 흉터진 것을 오히려 실력의 훈장처럼 여긴다고 한다.

### 관련 캐릭터
- 특정 인물에게 귀속되지 않은 범용 도구 — 생포 등급 의뢰를 전문으로 하는 헌터들 사이에서 표준 장비로 통용된다.

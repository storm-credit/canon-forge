# 🗡️ 언약의 룬검 '유스티티아' (Justitia, the Covenant-Rune Blade)

> *"검은 진실을 베지 않는다. 다만 거짓을 벤 자리에 진실이 드러날 뿐이다."*

---

| 항목 | 내용 |
|------|------|
| **분류** | 무기 — 한손검 |
| **등급** | **희귀급 (Rare)** 🔵 |
| **가치** | 거래 불가 (개인 서약 귀속) |
| **현재 소유자** | 셀레스티아 발레리우스 (Celestia Valerius) |
| **세력 귀속** | 하데스 무혈 징수단 (Hades Bloodless Ring) |
| **제작자/장인** | 왕립 아카데미 언약법(言約法) 공방 — 셀레스티아의 졸업 하사품 |
| **재료** | 백은강(白銀鋼), 언약 문자가 각인된 룬 잉크 |

| **강화 가능 여부** | 제한적 — 룬 문구 추가는 언약법 정식 절차를 거쳐야만 가능 |

### 외형 묘사
길이 90cm의 가벼운 세검(細劍)형 한손검. 칼날 전체에 실처럼 가늘게 새겨진 고대 언약 문자가 은은한 청백색으로 발광하며, 벨 때마다 유성이 떨어지는 듯한 궤적을 남긴다. 검자루에는 셀레스티아의 가문 문장이 아니라, 그녀가 스승과 처음 맺은 서약의 인장이 새겨져 있다 — 발레리우스 가문의 것이 아니라 그녀 자신이 스스로 택한 표식이다.

## 2. 성능 및 효과 (Stats & Effects)

| 유형 | 효과 | 비고 |
|------|------|------|
| **기본 성능** | 공격력: 희귀급 상위 (쾌검 특화, 저지력은 낮음) | 무게가 가벼워 연속 베기에 최적화 |
| **속성** | 언약 / 별빛 | 살상보다 제압·구속에 특화된 속성 부여 |

### **고유 능력: 《별빛의 시(Poetry of Starlight)》**
- **발동 조건:** 셀레스티아가 판결 또는 보호의 의지로 검을 뽑을 때
- **효과:**
  - 공중에서 춤추듯 이동하며 유성우를 연상케 하는 궤적으로 다수의 적을 동시에 제압
  - 칼날이 스친 자리에는 치명상 대신 언약의 룬이 옅게 낙인처럼 남아, 상대가 그 자리에서 맺은 거짓 맹세나 위증을 일시적으로 드러나게 한다
  - 살의를 담아 휘두르면 룬이 요동치며 발광이 흐트러진다 — 이는 검이 셀레스티아의 신념(불살不殺의 원칙)과 어긋난 의도를 감지했을 때 나타나는 경고 반응이다

### 보조 능력
- **진실의 섬광:** 검신에 반사된 빛으로 상대의 거짓 진술 순간 미세한 파장을 감지 — 완전한 거짓말 탐지는 아니나 협상·심문 장면에서 강력한 보조 수단
- **언약 공명:** [언약의 구형 수호구]가 전개된 상태에서 함께 사용하면 방패의 방어 범위 안에서 검의 궤적이 한층 안정된다

## 3. 배경 스토리 & 전승 (Lore)

### 제작 기원
왕립 아카데미 언약법 전공 졸업생에게 관례적으로 하사되는 세검이지만, 셀레스티아의 것은 스승이 직접 서약 문구를 골라 새겨 넣은 유일본이다. "진실을 강요하지 말되, 거짓을 감추지도 말라"는 문구가 칼날 뿌리 부분에 가장 깊게 새겨져 있다.

### 전승
채무 분쟁 현장에서 셀레스티아가 이 검을 뽑으면, 소문을 들은 채무자들이 먼저 무릎을 꿇고 사정을 털어놓는 일이 잦아졌다 — 검이 두려워서가 아니라, 그녀가 검을 뽑을 때는 반드시 진실을 들으려 한다는 것을 알기 때문이다.

### 관련 캐릭터
- 셀레스티아 발레리우스 (Celestia Valerius)
- 셀레스티아의 스승 (언약법 최후의 강독관)

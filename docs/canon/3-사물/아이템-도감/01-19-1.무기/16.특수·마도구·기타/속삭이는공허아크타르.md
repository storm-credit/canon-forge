# 🗡️ 속삭이는 공허 아크타르

> 자아를 지닌 지성체 결합 병기 (E-2)

| **구분** | **설명** |
| --- | --- |
| **카테고리** | 무기 (에고 무기 (Ego Weapons)) |
| **소유자/보관소** | 미상 (자아 종속) |

## ⚖️ 1. 대가 (Cost)
아크타르의 공허 자아와 결속을 맺는 행위는 자발적 에테르 채무 계약이다. 심연의 속삭임을 빌려 쓸수록 그 채무는 사용자의 감각과 의지를 조금씩 잠식하는 이자로 불어나며, 갚지 않은 채 결속을 지속하면 자아의 경계가 공허 쪽으로 서서히 침식된다. 빌린 권능을 영혼과 정신으로 갚는 자발적 청구서이지, 무구의 등급 자체가 강제하는 법칙적 비용이 아니다.

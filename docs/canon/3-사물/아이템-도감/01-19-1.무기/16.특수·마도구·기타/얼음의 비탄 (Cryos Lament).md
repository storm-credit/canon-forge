# 🗡️ 얼음의 비탄 (Cryos Lament)

> 이명(異名): 얼음의비탄크라이오스

> 자아를 지닌 지성체 결합 병기 (E-4)

| **구분** | **설명** |
| --- | --- |
| **카테고리** | 무기 (에고 무기 (Ego Weapons)) |
| **소유자/보관소** | 미상 (자아 종속) |

## ⚖️ 1. 대가 (Cost)
크라이오스는 얼음 원소·정신 계열 병기로, 등급에 비례하는 강제 대가는 존재하지 않는다. 소유자의 긍정 기억을 동결시키고 전투 후 도파민 고갈을 야기하는 것은 자아(에고)의 고유한 작동 방식이며, 우주적 법칙이 부과하는 벌이 아니라 이 병기 자체의 성질에서 비롯되는 현상이다. 위계 차이만 존재하며, 사용 여부와 무관하게 추가 청구는 없다.

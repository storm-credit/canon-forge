# 🗡️ 심해의 노래 네프탈리

> 자아를 지닌 지성체 결합 병기 (E-5)

| **구분** | **설명** |
| --- | --- |
| **카테고리** | 무기 (에고 무기 (Ego Weapons)) |
| **소유자/보관소** | 미상 (자아 종속) |

## ⚖️ 1. 대가 (Cost)
네프탈리는 심연과의 자발적 결속을 전제로 작동하는 에고 무기다. 소유자가 무기의 자아에 권능을 위임할수록 빌린 심해의 의지는 에테르 채무로 쌓이고, 이를 갚지 않으면 영혼과 감각을 담보로 이자가 불어난다. 육지에 대한 혐오 반응과 바다로의 귀속 충동은 채무 이자가 신체 감각으로 청구되는 형태이며, 법칙적 저주가 아니라 계약을 방치한 자에게 닥치는 자발적 청구서다.

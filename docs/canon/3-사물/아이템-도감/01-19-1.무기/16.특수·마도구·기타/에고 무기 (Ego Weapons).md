# 🗡️ 에고 무기 (Ego Weapons)

> 자아를 지닌 지성체 결합 병기. 저마다의 신념과 광기를 짊어진 여섯 자루가, 저마다의 방식으로 소유자에게 자신의 뜻을 강요한다.

이 계열은 특정 대장간이나 세력이 아니라 무기 자신의 자아에 귀속된다. 소유자를 스스로 선택하고 거부하며, 결속을 맺은 이에게는 §6-1의 자발적 에테르 채무가 적용된다 — 등급이 부과하는 강제 형벌이 아니라, 자아의 권능을 빌려 쓴 대가로 소유자 스스로 갚아 나가야 하는 청구서다. 개별 항목의 상세 외형·능력·전승은 각 정본 문서를 참조.

### E-1. 울부짖는 태양 칼루멘
- **외형**: 태양을 그대로 벼려낸 듯 황금빛으로 타오르는 대검. 검신 중앙의 정령 자아가 깃든 홈은 평상시 은은한 주황빛을 띠다가, 자비를 베풀려는 순간 용광로처럼 격렬하게 달아오른다.
- **자아의 강제 (Ego Compulsion)**: 무기에 깃든 광신적인 정령 자아가 전투 중 소유자가 자비를 베풀 경우, 배교자로 판단하여 손잡이에서 용광로급 열기를 뿜어내 강제로 소각 처분시킨다.
- **정본 문서**: [칼루멘 (Calumen)](./칼루멘%20(Calumen).md)

### E-2. 속삭이는 공허 아크타르
- **외형**: 빛을 삼키는 칠흑의 장검. 검신 중앙에 박힌 이지러진 결정이 자아의 자리이며, 소유자가 피로에 잠식될수록 짙은 자주색으로 물들며 낮은 진동음을 낸다.
- **자아의 강제 (Ego Compulsion)**: 공허의 왕을 참칭하는 자아가, 전투의 모든 피로를 허무주의로 치환해 안식과 정지만이 옳다는 최면적 환청을 심연 깊이 심어 넣는다.
- **정본 문서**: [아크타르 (Aktar)](./아크타르%20(Aktar).md)

### E-3. 폭풍의 포효 스톰브링어
- **외형**: 먹구름을 벼려낸 듯한 회흑색 대검. 검신의 균열 홈 안에서 청백색 핵이 맥동하며, 후퇴가 불가하다고 판단하는 순간 번개색으로 격렬히 번쩍인다.
- **자아의 강제 (Ego Compulsion)**: 오직 수호와 전진만을 명한다. 아군의 진형이 무너져 후퇴해야 할 순간, 소유자의 다리 근육에 강제 경련을 일으켜 억지로 적진에 뛰어들게 만드는 광전사의 저주장구.
- **정본 문서**: [폭풍의 포효 (Storm's Roar)](./폭풍의%20포효%20(Storm's%20Roar).md)

### E-4. 얼음의 비탄 크라이오스
- **외형**: 얼어붙은 강물의 단면 같은 반투명 은백색 대검. 검신의 균열 문양이 청백색으로 맥동하며, 소유자의 감정이 동요할 때 짙은 청색으로 물들어 냉기를 뿜는다.
- **자아의 강제 (Ego Compulsion)**: 소유자의 행복하고 긍정적인 기억을 제물로 얼어붙게 만든 뒤, 무감각한 인간 백정으로 개조시킨다. 전투 후 심각한 도파민 고갈 쇼크.
- **정본 문서**: [얼음의 비탄 (Cryos Lament)](./얼음의%20비탄%20(Cryos%20Lament).md)

### E-5. 심해의 노래 네프탈리
- **외형**: 심해 조가비의 안쪽 면처럼 진주빛으로 일렁이는 곡도. 검신에 새겨진 파문 무늬가 자아가 깨어날 때마다 실제 물결처럼 흔들린다.
- **자아의 강제 (Ego Compulsion)**: 무기가 통제권을 쥐면, 육지의 흙을 밟는 것만으로도 구토가 치밀도록 신경망을 조작하여 소유자를 영원히 바다를 떠도는 짐승으로 만든다.
- **정본 문서**: [네프탈리 (Nephtali)](./네프탈리%20(Nephtali).md)

### E-6. 불사의 심판 이그니투스
- **외형**: 신성한 백열로 타오르는 순백의 검. 검신 위 심판의 문양이 부도덕한 의도를 감지하면 빛을 잃고 차갑게 식는다.
- **자아의 강제 (Ego Compulsion)**: 부도덕하거나 이기적 목적의 전투가 전개될 때 무기가 파업한다. 나아가 불결한 손을 정화한다는 명목으로 착용자의 손톱 주위에 영구적 화상을 조각한다.
- **정본 문서**: [이그니투스 (Ignitus)](./이그니투스%20(Ignitus).md)

## 대가 (Cost) — 계열 공통 원칙

에고 무기 여섯 자루는 모두 일반 계열의 권능(태양·심연·폭풍·얼음·심해·심판)을 다루며, 등급 자체가 강제하는 법칙적 페널티는 존재하지 않는다. 신은 침묵하고, 각 자아는 소유자가 결속을 맺은 만큼만 요구할 뿐이다. 다만 자아의 권능을 빌려 쓸수록 그 대가는 소유자의 감각·기억·의지를 담보로 한 에테르 채무로 쌓이며, 결속을 끊지 않는 한 이자는 계속 불어난다 — 이는 무구가 일방적으로 부과하는 저주가 아니라, 자아와 소유자 사이에 맺어진 자발적 거래다.

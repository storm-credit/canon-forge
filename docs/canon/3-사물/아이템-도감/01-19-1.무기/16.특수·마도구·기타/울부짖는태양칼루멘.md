# 🗡️ 울부짖는 태양 칼루멘

> 자아를 지닌 지성체 결합 병기 (E-1)

| **구분** | **설명** |
| --- | --- |
| **카테고리** | 무기 (에고 무기 (Ego Weapons)) |
| **소유자/보관소** | 미상 (자아 종속) |

## ⚖️ 1. 가혹한 참상 (Epic Penalty / Burden)
무기에 깃든 광신적인 정령 자아가 전투 중 소유자가 자비를 베풀 경우, 배교자로 판단하여 손잡이에서 용광로급 열기를 뿜어내 강제로 소각 처분시킨다.

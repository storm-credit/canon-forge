# 📜 이시스-아스트라 계보 총람 (Isis-Astra Lineage Registry)

> **정본 주의**: 동명의 "이시스-아스트라" 계열 검이 파생·변형명을 포함해 대륙에 **총 70자루** 확인된다. 70자루 모두 같은 청람석(靑嵐石) 원석 계보에서 갈라져 나온 자매검이나, 소유자·소속·서사는 완전히 무관한 **개별 개체**다. 본 문서는 계열 전 개체의 정본 명부이며, 각 검의 서사·성능은 반드시 해당 정본 문서에서 확인할 것.

각 계열 문서의 "정본 주의" 안내문은 본 총람을 유일한 전체 명부로 참조한다. 계열 검이 새로 정본화되면 **본 총람에 등재하고 총 자루 수를 갱신**하는 것으로 전 문서의 일관성이 유지된다.

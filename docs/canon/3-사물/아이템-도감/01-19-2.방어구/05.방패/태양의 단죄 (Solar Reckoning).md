# 🛡️ 태양의 단죄 (Solar Reckoning)

> 이명(異名): 05.방패/포르티스의영원한수호

> *"솔라리스의 빛이 닿는 곳, 제국의 심판이 벼려진다."*

## 1. 기본 정보

- **무기명**: 포르티스의 영원한 수호
- **종류**: 방패
- **전용 소유자**: 발터 포르티스
- **제작처**: 장소백과 - 솔라리스 프라임 황실 대장간

## 2. 성능 및 효과

- 이 방어구는 빛 속성 이외의 마나를 거부합니다. 심연의 오염이 닿을 경우 무기 스스로 사용자에게 고열의 화상을 입히는 '태양의 단죄' 반동을 일으킵니다.

## 3. 고유 권능

- 소유자 발터 포르티스의 마나 파장에 맞춰 극한의 빛 마나를 압축, 배출합니다.

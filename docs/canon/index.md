## 원칙

> 작아지고, 모순이 0으로 수렴하고, 끝이 있는 캐논. 크기를 키우는 게 아니다.

- 원본은 **읽기전용 source of truth** — 절대 수정 안 함
- 캐논은 원본 evidence 기반. 메운 것은 `⚠️[추가]` 표시 (승인 전엔 캐논 아님)
- 신은 답하지 않는다 — 갈등의 축은 **인간의 이데올로기**

# 직업체계 (Job & Class System)

> Phase 1 · 1차 정리 · 2026-06-16  
> 원본: `01-20. 직업 & 클래스 (Classes & Jobs)/` (40파일)

---

## 핵심 원칙

**규칙**: 직업은 단순한 생업이 아니다 — 신앙·생태계·마도 기술이 얽힌 권력이자 피의 굴레.  
**규칙**: 직업의 한계는 섬기는 신의 교리에 의해 규정된다.  
**규칙**: 이단/금지 직업군 소속자는 적발 즉시 즉결 처형 또는 화형. 재판 없음.

탐험 모티프: 직업 = 왜 세계를 돌아다니는가의 동기. 직업이 다르면 이동 목적·동맹·위험이 다르다.

### 히든 직업 (4종)

> **규칙**: 히든 직업은 "일인 계승"이 아니다 — **극소수·은둔·양산 불가**. 각 직업의 내재적 파국(석화·자아붕괴·혈관폭주·갈증이탈)으로 대를 이어 양산될 수 없을 뿐, 동시대 복수 존재 가능. 일인 전승은 [[초월 직업군]] 소관.

**13 — 심연 조각사 (Abyssal Sculptor)**

- **메커니즘**: 재앙급 몬스터 군주의 저주받은 통뼈·초고농도 마력 결정 원석만 재료로 사용. 조각도 완성 순간 반경 수십km의 기상·생태계 법칙을 **영구적으로** 뒤틀거나 50km 내 몬스터 접근을 원천 차단하는 요새급 결계로 작동. 군대·국가 운명을 돌 조각 하나로 결정.
- **희소 원인 (원본 직접 인용)**: *"조각사 본인의 인간성(혼)과 수질 세포(수명)도 돌가루처럼 허공에 같이 깎여 나간다."* — 대작 하나 완성 직후 손끝부터 **석화(Petrification)** 진행. 근육이 석상으로 굳어 박제 사망. 자연의 족쇄.
- **사회적 취급**: 현상금 100만 골드 + 화형. 미치광이 노인·걸인으로 위장 은둔. 세력 포획 시도 끊임없음.
- **named 인물**: 초대 자하드(Jahad) — 천 년 전 차원 붕괴를 봉인 후 불멸의 석상으로 굳어 늪지 옥좌에 박제. 현시대에도 그의 조각들이 대륙을 감시 중.

**14 — 영혼 얽는 자 (Soul Weaver / Puppeteer)**

- **메커니즘**: 전설급 영웅 시신의 척추에 마력 실을 꿰매 호문쿨루스로 개조. 술자 본인은 수km 후방 안전지대에서 10가닥 마력선으로 전선을 조종. 인형은 팔다리가 잘려도 즉각 봉합·재기동.
- **희소 원인**: 마력선이 인형 척추가 아닌 **술자 본인의 뇌간 중추 신경계 생살**에 다이렉트 연결. 50대 말년에 인형의 망자 혼과 본체 영혼이 뒤섞이는 **자아 분열(조현병)** 한계점 도달 → 자신이 인간인지 인형인지 인지 불능 → 목에 마력 실을 감아 자살. 모든 고위 술사가 이 말로를 피할 수 없음.
- **사회적 취급**: 공식 기록에서 존재 자체를 삭제. 생포 시 성국 신성 심문 후 화형.
- **named 인물**: 피에타 마리오네트(Pieta Marionette) — 자유도시연합 암시장 막후 지배자. 제1차 왕국 침공 당시 최정예 기사 100명의 뇌파를 동시 탈취해 아군끼리 학살시킨 전범 기록 보유.

**31 — 마검사 (Magic Swordsman)**

- **메커니즘**: 서로 상극인 마력 회로 + 오러(Aura) 연성 회로를 육체에 동시 각인. 한 자루 마검으로 일기당천 학살. 어떠한 진영·길드 소속도 거부. 국가급 귀족도 의뢰 불가.
- **희소 원인**: 10대에 혈관이 터져 죽는 자 **99%**. 극소수 천재도 무리한 일격 1회 후 전신 마비 또는 자아 붕괴. 길드 훈련·전수 경로 없음.
- **사회적 취급**: 이단 분류는 아니나 국가 통제권 밖. 발각되면 포섭 시도 후 거부 시 제거.

**32 — 핏빛 군주 (Blood Lord)**

- **메커니즘**: 혈귀술·불사 저주 기반 생명력 강탈·지배. 주로 크림슨 대륙 고대 혈귀·저주 귀족 가문. 화폐 = 인간 선혈·영웅 영혼. 노예 상단·고아원을 식량 창고로 위장 운영.
- **희소 원인**: 타인의 피 미섭취 시 이성 붕괴·이탈 현상. 이계 성수(Holy Water)·햇빛 접촉 시 복구 불능 치명상. 발각 즉시 이단 말살 처분.
- **사회적 취급**: 이단 최상위. 성국 화형 + 능지처참. 크림슨 대륙 심층부 고대 폐허에서만 장기 은신 가능.

### 초월 직업군 (T) — 영웅 전용 일인 전승 클래스 (38개)

> 원본: `35. [영웅 전설] 아스트라리스 대륙의 초월적 직업군 (Transcendent Classes).md` (33KB)  
> **규칙**: 초월 직업은 S급 영웅이 기존 직업 + 우주적 십자가(Epic Burden)를 결합해 **세계관에 단 하나뿐인** 클래스로 각성한 것. 길드·왕립 아카데미에서 전수 불가. named 영웅 1인에 1:1 귀속. 사망 시 클래스 소멸.

**공통 규칙**

## 몬스터 헌터 상세 (위쳐식 야성 직업)

> 원본: `직업 도감/몬스터 헌터 (Monster Hunter).md` + `마법 도감/헌터 전용 특수 마법 (Hunter Arts).md`  
> **규칙**: 헌터 아츠는 마도학이 아니다. "살아남기 위한 운명의 투쟁" — 수명·혈관을 대가로 강제 진화시킨 **야성의 연금술(Primal Alchemy)**. 마탑 정통 마법의 명시적 대척점.

### 헌터 5단계 계급 (핏빛 계급 체계)

| 계급 | 명칭 | 역할 |
|:---:|---|---|
| 1 | Blood Tracker | 추적·정찰. 생존 기초 |
| 2 | Fang Breaker | 중형 거수 처치. 사체 해체 실무 |
| 3 | Siege Hunter | 포위 전술 지휘. 분업 조율 |
| 4 | Abyss Warden | 심연 기원 거수 전담. 특수 기술 사용 |
| 5 | Myth Breaker | 신화급 거수·이계 존재 처치. 동시대 극소수 |

### 헌터 아츠 (Hunter Arts) — 3대 야성 술법

**기술 1 — 지맥 결박 (Leyline Binding)**
- **본질**: 에테르 지맥을 강제 노출시켜 거수 관절부 신경망을 직접 봉쇄
- **대가**: 에테르 과부하(Etherion Surge) — 혈관 파열·시신경 마비. 결박진이 끊기는 순간 영혼 그릇 파괴 위험

**기술 2 — 심연의 침전 절차 (Abyssal Communion)**
- **본질**: 맹독 에테르를 심장에 주입해 체내를 거수와 일시 동화 (맹독 지역 돌파 목적)
- **대가**: 각막이 회색으로 물들고 폐부 파열. **1시간 초과 시 이성 붕괴·영구 실명·괴수 타락**. 길드 내부 명칭 "자발적 사형선고"

**기술 3 — 영혼 붕괴의 공명 (Resonance of Echoes)**
- **본질**: 무기를 통해 거수 갑각 내부만 액체화하는 진동 파장
- **대가**: **절대적 무방비 3초** 유지 필수. 성공 시 갑각은 멀쩡하고 내부만 즉사. 술자 팔뼈에 미세 실금이 누적

### 헌터 전술 분업 (포위전 4포지션)

| 포지션 | 역할 |
|---|---|
| Vanguard | 거수 주의 분산·정면 압박 |
| Sniper | 약점 부위 정밀 사격 |
| Trapper | 지맥 결박진 포위 세팅 |
| Healer | 침전 절차 부작용 응급 대응 |

> **탐험 모티프**: 헌터의 핵심 산업 = 거수 사체 해체 → 에테르 결정 추출. 정규군이 궤멸한 변이 생태계 최전선에 단독 침투. [[등급-대가-정합표]] 무관 — 마법 클래스 체계 외부.

## 직업-신앙-생태계 삼각 의존성

**규칙**: 직업↔신앙↔생태계는 분리 불가. 신앙을 배반하면 직업 능력에 즉각 물리적 저주가 적용된다.

| 신 | 연결 직업 | 금기 위반 시 저주 |
|---|---|---|
| 솔라리스 (태양) | 사제·성기사·이단심문관 | 아군 포기·맹세 위반 → 심장의 룬이 내부 폭발 |
| 가이아 (대지) | 심마니·연금술사·드루이드·정령사 | 오염·착취 → 자신이 만든 포션이 맹독으로 변함 |
| 포세이돈 (바다) | 항해사·해적 | 제물 누락 → 예고 없는 해일에 영혼째 파쇄 |
| 릴리스·로토스 | 심연 사도·흑마법사·히든 암살자 | 계약의 대가로 미각·수면 능력을 생으로 상납 |
| 모르디아 (죽음) | 강령술사 | 명령 위반 → 직접 영혼 회수 |
| 아나테아 (지혜) | 마법사 | 저질 마력석 사용 시 체내 마력 역폭발 → 융합괴물화 |
| 마르스 (전쟁) | 기사·용병·검투사 | 광기가 누적되면 피아식별 불능 상태로 변이 |

### 마법 의존도 분류

| 의존도 | 직업군 | 특징 |
|---|---|---|
| 🔴 필수 | 마법사·연금술사·사제·강령술사·정령사·드루이드 | 반마법장치 구역에선 일반인보다 무력 |
| 🟡 보조 | 기사·용병·음유시인·항해사·몬스터 헌터 | 마법 도구 활용. 육체·처세술이 기본 |
| ❌ 배격 | 드워프 대장장이·이단 심문 반마법단·수도승·도적 | 마법 개입 자체를 사교 행위로 취급. 순수 육체와 철로 마법사 방어막을 단숨에 격파 |

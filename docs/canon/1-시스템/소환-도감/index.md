# 🌀 소환 도감 (Lexicon)

> [!warning] 도감 수치 톤 규정
> 본 도감의 수치·퍼센트는 작가용 자료값이다. 원고(산문) 반영 시 반드시 정성 서술로 번역한다(레저 스케일 문서가 모범 — '벤 다음에야 값을 알려준다'). 기계 게이트가 원고의 수치·퍼센트 노출을 차단한다.
>
> 편집장 노트 (2026-07-15 전수 스윕 마감): 도감 수치 톤 규정 배너 신설(관리 정책 승격 — 게임수치 문법 128파일 개별 수정 대신 이 배너로 관리).

> [소환체계](../소환체계.md)의 부속 도감. 차원을 찢어 이계 존재를 강제 편입하는 10대 소환 계열. 7대 조약(서클) 트리.

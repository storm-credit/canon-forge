# 등급-대가 정합표 (Grade-Cost Balance Table)

> Phase 1 · 1차 정리 · 2026-06-16  
> 원본: 마법 백과 / 소환 백과 / 신성 백과 / 연금 백과 / 마공 백과 / 정령 백과 교차 수집

---

## 핵심 규칙 (5대 불변식 §2 적용)

**규칙**: "등급이 높을수록 대가가 치명적이다. 예외 없음."  
**규칙**: 모든 시스템(마법·소환·신성·연금·마공·정령·주술)에서 이 원칙이 동일하게 적용된다.  
**규칙**: 한 시스템의 최고 등급 대가 = 개체 소멸 또는 자아 해체. 예외 없음.  
**규칙**: 소설에서 "최고 등급을 달성하고 정상 생활하는 NPC"는 캐논 위반.

## 레이어별 대가 비교 (AI 집행 가능 명제형)

### 레이어 1 (입문)

| 시스템 | 대가 |
|---|---|
| 마법 1클래스 | 에테르 동기화 증후군 (환청, 시야 왜곡, 에테르 밀집 지역 환경 반응) |
| 소환 1조약 | 에테르 마찰 두통, 코피, 24시간 차원 현기증 |
| 신성 1위계 | 환상통 전이, 구강 출혈 |
| 연금 1대진리 | 경미한 현기증, 에테르 마찰 |
| 마공 1기어 | 손 저림, 에테르성 이명 |
| 정령 1단계 | 경미 — 가벼운 명상 필요 수준 |

**규칙 레이어 1**: 모든 시스템 1등급 대가는 일시적이며 비가역성 없다.

**규칙 레이어 3**: 중급 이상 대가는 신체 영구 변화를 수반한다.

### 레이어 5 (상급)

|---|---|

| 소환 5조약 | 뇌 조직 과열, 수면 영구 불가, 조기 백발 (30대 백발 증후군) |

| 마공 5기어 | 함장이 함선의 수명과 영구 동기화 |
| 정령 5단계 | 완전한 신체·영혼 해체 → 성좌 확산 |

**규칙 레이어 5**: 상급 이상 달성자는 사회적 "정상 생활" 불가.

### 레이어 7 (전설)

| 시스템 | 대가 |
|---|---|
| 마법 7클래스 | 영혼 정화 (세속 욕망·갈등·탐욕 완전 소멸. 생태계 생물들이 주변 모임) |
| 소환 7조약 | **존재 자체 삭제** (분자 수준 소멸, 영혼 파편 영구 삭제, 모든 현실·사후세계에서 삭제) |
| 신성 7위계 | 완전 영혼 해체 → 성좌 본질로 영구 편입. 역사에서 소거 |
| 연금 7대진리 | 몸·영혼 승천 → 황금 항성 격자로 의식 해체 |
| 마공 7기어 | 우주 기어로 변환 → 세계의 영구 방어 체계로 승천 |
| 정령 5단계 | (5단계가 최고위. 성좌 확산 = 존재 해체) |

**규칙 레이어 7**: 모든 시스템의 최고 등급 대가 = 개체 소멸 또는 우주 법칙 편입. 달성 후 해당 존재는 소설에 "현재 살아 있는 개인"으로 등장 불가.

| 6 | 야만신 (Primal God) | 발동 시 눈·귀·전신 피부 대량 출혈, 내장 파괴, 인간 기억·이성 50%+ 영구 소실, 저주에 잠식된 공허한 파괴자 | ✅ 비가역 |
| 7 | 멸망 (Cataclysm) | **완전한 존재 소멸** — 시전 직후 몸이 재로 붕괴, 영혼 사후세계 도달 불가 영원 분열, 세계령이 역사 기록에서 이름 삭제 | ✅ 완전 비가역 |

## 5대 불변식 §2 검증 (등급↔대가 정합)

| 시스템 | 낮은 등급 대가 | 높은 등급 대가 | 정합 여부 |
|---|---|---|:---:|
| 마법 9클래스 | 환청·시야 왜곡 | 존재 소멸 | ✅ |
| 소환 7조약 | 두통·코피 | 존재 삭제 | ✅ |
| 신성 7위계 | 환상통·구강 출혈 | 영혼 해체·역사 소거 | ✅ |
| 연금 7대진리 | 현기증·마찰 | 황금 항성 격자로 해체 | ✅ |
| 마공 7기어 | 손 저림·이명 | 우주 기어로 영구 변환 | ✅ |
| 정령 5단계 | 명상 필요 | 성좌 확산 소멸 | ✅ |
| 주술 7주계 | 구강 출혈·피부 낙인 | 존재 소멸·역사에서 이름 삭제 | ✅ |

> **결론**: 7개 시스템 전부 등급↔대가 정합 통과. Phase 1 검수 완료.

## 교차 참조 — 에반의 소환 대가 누적 계산 기준

소설 집필 시 "[소설-2] 대가 누적 계산기" 프롬프트와 함께 사용:

```
에반 소환 1회 시 계산 방식:
  소환 조약 등급 확인 → 해당 조약의 대가 항목 확인
  → 5대 불변식 §1: 이전 대가에 누적 (감소 없음)
  → 5대 불변식 §2: 높은 조약일수록 더 큰 대가

임계점:
  6조약 사용 → 장기 파열·수개월 혼수 → 이후 행동 능력 제한 적용
  7조약 사용 → 에반 존재 삭제 → 소설 종결 조건
```

# 에반 무영 라크라시스 — 잊혀진 소환사

## A. 정체 (Core Identity)

- **세력**: 무소속 / 운명의 방랑자 (The Faceless Wanderer). 어떤 세력에도 속하지 않는 1인 군단.
- **본질**: 유물 속 영웅들의 기억 잔재를 현현시키는 **사이코메트리 연대기 소환사**. 본인 무력은 중위급 검사 수준이나, 수백 명의 소환 진용을 상황에 맞춰 스위칭하는 전술 지휘관(Maestro).
- **시작 신분**: 저주받은 유물·주술구 감정사. 유물을 만지면 죽은 자의 감정이 흘러드는 체질.
- **기억 소각**: 타인의 기억을 너무 많이 담아 자신의 출신·이름·얼굴이 희석되어 감. (사고/저주가 아닌 자기 소각)

## B. 세계 불변식 — 분신 구조 (★핵심 정정)

> 기존 스펙의 "에반 분신 = 무의식 껍데기" 기술은 **오류**. 원본은 서로 다른 두 층위를 가진다.

| 구분 | 9대 위장 페르소나 | 대역 오토마톤 (Body Double) |
|---|---|---|
| 자아 | **에반 본인** (직접 변장·조종) | **없음** (텅 빈 호문쿨루스/드론) |
| 목적 | 흑막으로 별개 인물 행세 | 페르소나 외출 중 '에반 본체' 자리 알리바이 |
| 다중존재 | 본인이 옮겨다니므로 동시 등장 = **모순 아님** | 본체와 더미 교차 연출 = **모순 아님** |
| 모순 조건 | 페르소나가 **에반 의지에 반해** 행동 | 오토마톤이 **자율 판단** |

- **9대 페르소나**: 레온하르트 카스테로 · 카일락 무라드 · 세드릭 아인하르트 · 빈센트 그라시움 · 루시우스 베네딕트 · 렉시온 아머하트 · 에리히 폰 발디크 · 단테스 코르바흐 · 파우스트 폰 아르젠틱
- ⚠️ **dedup 확인 필요**: 위 9명이 영웅 백과(01-14)에 *독립 영웅*으로 별도 등재돼 있으면 → "에반 본인"으로 병합 표기해야 함. (미확인)

## C. 금단 소환의 대가 (서사적 — Layer C)

> 에반은 **금단 소환술**을 다루는 자다. 금기 분야이므로 그 위험은 *서사적 긴장*으로 존치하되, "누적 계산기"식 기계적 회계로 행동을 제약하지 않는다 — 대가는 드라마가 필요할 때 선택하는 장치다. → [[세계의-헌법]] §5

- 소환 영웅들의 부작용(불면·환상통 등)이 서사적으로 에반에게 영향을 줄 수 있다 — *이야기 전개 도구*로.
- **기억 vs 힘의 딜레마**가 에반의 핵심 내적 갈등이다 (아래 C4 "망각의 유혹" 참조). 이건 캐릭터 아크이지 등가교환 공식이 아니다.

## D. 핵심 장비·관계

- **무장**: 무한의 아비셜 코트 & 주술구 수납띠(Bandolier of Catalysts) — 100여 개 영웅 매개체를 꿴 아공간 인벤토리.
- **[[루미에]]**: 에반의 마력 잔량을 실시간 계측·경고하는 아카식 번역가. *"에반 없이 루미에는 표류하고, 루미에 없이 에반은 폭주한다."*
- **소환 영웅 군단**: 아군이 아닌 강제 계약된 맹수. 충성([[루드빅 배너맨]]) ↔ 반역([[엘드릿치 옵저버]], [[데미우르고스]]) 혼재.

## E. 성장 벡터 (C1→C6)

| 연대기 | 핵심 |
|:---:|:---|
| C1 아스트라리스 | 질문의 시작 — "나는 왜 이들을 부르는가?" |
| C2 | 도구의 균열 |
| C3 | 이름의 파편 — 소환 영웅 기억 속 자기 이름 발견 |
| C4 | 망각의 유혹 — "기억을 지우면 강해진다"의 대가 |
| C5 | 잔향의 수렴 — 지워진 기억의 귀환 |
| C6 | 선택의 결말 — *"나는 잊힌 소환사가 아니라, 기억하는 소환사다."* |

**궁극 목적**: 자신의 기억과 연결된 최초의 유물을 파괴해 종속 영웅들을 안식시키고, 이공간의 굴레에서 해방되는 것.

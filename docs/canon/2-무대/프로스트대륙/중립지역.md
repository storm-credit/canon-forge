# 프로스트 대륙 중립 지역 (Neutral Zones)

> *"이 혹독한 대륙에서도 사람이 체온을 나눌 수 있는 곳들이 있다."*

혹한의 대륙에서 세력을 초월해 개방된 거점들. 상업·생존·정보가 교차하는 삶의 거점이자, 각 세력의 이해관계가 충돌하는 외교 전선이기도 하다.

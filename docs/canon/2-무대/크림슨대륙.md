# 크림슨 대륙 (Crimson Continent)

> Phase 2 · 1차 정리 · 2026-06-16

## 핵심 정체

**"붉은 황무지"**. 고대 아르카나 제국이 어비스 대재앙으로 자멸한 뒤 대륙의 80%가 맹독성 에테리온(Etherion)으로 오염되었다. 평범한 인간이 숨을 쉬는 것 자체가 위험한 죽음의 사막. 과거 위대한 마법사들의 뼛가루가 붉은 흙먼지가 되어 대기를 떠돈다.

## 고유 생존 규칙 (AI 집행 가능 명제형)

| 규칙 | 조건 | 결과 |
|---|---|---|
| **맹독성 적사풍 석회화** | 방호 없이 적사풍에 노출되면 | 아르카나 코어 가루 흡입 → 피부 석회화 + 폐포 파열 → 5년 이내 장기 부전 사망 |
| **심연의 혈우 용해** | 오염된 대지맥 과부하 시 내리는 붉은 산성비(혈우)에 맞으면 | 비전 방벽 즉각 부식 → 육신이 뼈만 남고 녹아내림 |
| **산성 호수 옥시탄** | 호수의 산성 증기에 노출되면 | 시신경 타들어 가 영구 실명. 접촉 시 뼈만 남고 용해 |
| **적색 강철 웜** | 마나 진동이 0.1 이상인 행렬이 사막 모래 지역을 통과하면 | 지하의 강철 웜이 진동 감지 → 지표 돌파 → 통째 삼킴 |
| **차원 균열 지대** | 크림슨 대륙 내륙 차원 균열 지대에 진입하면 | 과거·현재 경계 혼돈 → 수백 년 전 전투 잔상이 실체처럼 출현. 마법 제어 실패 위험 |
| **핏빛 십자가 선인장** | 혈액 냄새를 지닌 채 선인장 군락에 접근하면 | 선인장이 가시 발사 → 체액 착취 → 탈수 사망 |

## 세력 구조

| 세력 | 거점 | 이념 | 경제 기반 | 외교 노선 |
|---|---|---|---|---|
| **솔라리안 제국 (Solarian Empire)** | 크림슨 대륙 서부 철혈 제국 | 군국주의·실력주의, 감정보다 전투 효율 | 군수품·포로 노예·황실 금화 | 자유도시에 설계도 강탈식 교역. 왕국연합에 결함 무기 판매 |
| **아르카나 잔재 (Arkana Remnants)** | 아르카나 제국 폐허 지하 | 옛 제국 마법 부활 집착, 사교도 성격 | 불안정 코어 마법석·금지 유물·생체 실험 결과물 | 마법협회 암거래. 자유도시 암시장 공급 |
| **용의 후예 (Dragon Descendants)** | 사막 암반 요새 | 혈통 순수주의·전투만이 가치 | 드래곤 가죽 방어구·용의 뼛가루·전투 용병 | 왕국연합에 공물 강요. 자유도시에 검투사 송출 |
| **붉은 사막 부족 (Red Desert Tribes)** | 오아시스 유목 거점 | 사막 적응 생존주의·약육강식 | 사막 약초·맹독 향신료·야수 조련 노예 | 마법협회에 마약성 약초 공급. 솔라리안에 통행권 판매 |
| **잊혀진 서고단 (Forgotten Library Order)** | 고대 유적 지하 비밀 도서관 | 지식 독점·정보 카르텔 | 멸망의 병기 도면·금지 서적·고대 유물 경매 | 마법협회에 초고가 자료 판매. 성국에 이단 서적 공급 |

> **현재 갈등**: 솔라리안 제국 ↔ 아르카나 잔재 간 차원 균열 통제권 분쟁 · 용의 후예 ↔ 붉은 사막 부족 간 오아시스 지배권 충돌

## 경제 체계

- **실질 화폐**: 물 한 모금이 사람 목숨보다 비싼 극한 희소성 경제
- **핵심 무역 품목**: 제국 군수품·아르카나 유물·사막 약초·잊혀진 서고단 지식
- **암시장**: 자유도시 ↔ 아르카나 잔재 간 불법 인체 연성 유물 / 심연 군단 ↔ 서고단 간 악마 소환석

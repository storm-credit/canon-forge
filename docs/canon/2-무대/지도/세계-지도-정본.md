## 1. 5대 대륙 배치 (작가 스케치 정본 — 2026-07-14 확정, 구 시안 A 폐기)

배치 원형: **북서 → 남동 대각 종렬**

```
프로스트 ▣▣▣
에테르 ▣▣▣▣▣▣▣▣          〔해양 ∴∴ 동측 군도 허브〕
              크림슨 ▣▣▣
                    ▣▣▣▣ 오벨리스크
```

| 대륙 | 방위 | 근거 등급 |
|---|---|---|
| **프로스트** | 북(서) | ✅ 캐논 (에테르 북부와 결빙해) |
| **에테르** | 중서 (대륙 최대) | 작가 확정 |
| **크림슨** | 에테르 남동 | 작가 확정 — 기후 그라데이션·지하 궤도망 축 성립 |
| **오벨리스크** | 최남동 외곽 (원해) | 작가 확정 — "잊힌 대륙 외곽부" 정체성·크림슨행 그림자 비도와 정합. 구 '프로스트 인접' 서술 2건은 '원해 항로 연결'로 개정 |
| **해양** | 동측 군도 | ✅ 캐논 (5대륙 연결 허브) |

대륙 간 노선(전부 캐논): 철의 지하 궤도망(에테르-크림슨-프로스트 지하 500m) · 북부 얼음 분쇄로(에테르↔프로스트) · 태양빛 황금 해로(에테르↔해양) · 그림자 협곡 비도(오벨리스크→크림슨·에테르 밀수) · 그림자 항구발 원해 밀수 항로(프로스트↔오벨리스크).

## 2. 에테르 대륙 정본 방위 (07-14 감사 수선 반영)

성국=**중앙**(신성한 분지 고원·성역 벨트) / 왕국연합=**중서부**(서부 국경 수호) / 정령연합 거점=**북부** 은성 숲(수도 실바나리스, 세력권은 전역) / 마법협회=**부유 수도 아르카디아**(중앙 상공 3,000m) / 자유도시연합=**전역 산재** 도시국가 연합 / 갈등 마찰선="성국(중앙 성역)↔왕국연합(서부 국경)". 상세: [[_공간정합-감사보고-2026-07-14|감사보고]].

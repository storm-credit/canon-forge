# 교통·통신 체계 (Magitech Transport & Communication)

> Phase 2 · 1차 정리 · 2026-06-16

## 개요

아스트라리스 5대 대륙을 잇는 마도 인프라. 교통은 육·해·공·차원 4계층, 통신은 광학·지맥·심해 3종이 세력별로 분리 운용된다. 인프라 장악 = 교역로 통제 = 패권.

## 2. 마도 통신 (Communication)

### 2-1. 대마탑 광학 릴레이

```
운용: 마법협회
방식: 산봉우리 대마탑 첨탑 → 마나 결정체 집광 → 빛줄기 릴레이 → 다음 마탑
범위: 에테르·크림슨 대륙 주요 거점망
```

**명제형 규칙**:
- 마탑 간 거리가 50km를 초과하면 → 중계 마탑 없이 직접 전송 불가
- 기상 악조건(폭풍·눈보라) 발생 시 → 광학 신호 차단 → 완전 통신 두절
- 마탑 하나를 파괴하면 → 해당 릴레이 구간 전체 통신 마비

### 2-2. 세계수 지맥망

```
운용: 정령연합·정령사·엘프
방식: 세계수 뿌리 → 대지맥 → 원거리 세계수 잎 → 수신
특성: 텍스트가 아니라 감정·이미지 형태로 전달 (감정 동기화)
범위: 에테르 대륙 삼림 권역 내부 전용
```

**명제형 규칙**:
- 세계수 뿌리가 없는 지역(크림슨·프로스트 등)에서는 → 지맥망 수신 불가
- 전쟁 시 세계수 도메인을 말살하면 → 해당 구역 정령연합 통신망 완전 차단 (전술 활용 가능)
- 감정 필터 없이 수신하면 → 타인의 공황·공포 감정 직격 → 수신자 패닉 위험

### 2-3. 심해 유물 텔레파시

```
운용: 해적 연합 (불법)
방식: 고대 심해 조개 유물 → 뇌파 공명 → 짝 유물로 전송
      고급 버전: 망자의 턱뼈 유물 — 죽은 자의 목소리로 변환 전달
범위: 해양 대륙 전역 + 타 대륙 연안 항구
```

**명제형 규칙**:
- 심해 유물이 쌍(짝)으로 존재하지 않으면 → 통신 불가
- 고급 버전(망자의 턱뼈)을 비숙련자가 사용하면 → 착란 + 망자의 기억 침입 위험
- 마법협회·성국은 불법 통신 채널로 지정 → 발각 시 유물 압수 + 처벌

### 2-4. 기타 통신 수단

| 수단 | 운용 | 특징 |
|---|---|---|
| **성국 뇌파 강제 연결 (Holy Communion)** | 성국 최고 사제 | 신도 간 강제 텔레파시. 명령 하달용. 비신도 사용 시 뇌 손상 |
| **환수 전령 (Familiar Post)** | 마법사·정령사 일반 | 봉인된 동물 사육사에게 메시지 전달. 신뢰도 높으나 속도 느림 |
| **마도 영상기 (Scrying Glass)** | 마법협회 상위 | 실시간 영상 통신. 유지 비용 극대. 의회급 회의 전용 |

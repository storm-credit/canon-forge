# 교통·통신 체계 (Magitech Transport & Communication)

> Phase 2 · 2차 전수 대조 · 2026-06-17

## 개요

아스트라리스 5대 대륙을 잇는 마도 인프라. 교통은 육·해·공·차원 4계층, 통신은 광학·지맥·심해 3종이 세력별로 분리 운용된다. 인프라 장악 = 교역로 통제 = 패권.

## 2. 마도 통신 (Communication)

### 2-1. 대마탑 광학 릴레이

```
운용: 마법협회 + 솔라리안 제국군 합동 (제국 70개 첨탑 보유)
방식: 산봉우리 대마탑 첨탑 → 마나 결정체 집광 → 빛줄기 릴레이 → 다음 마탑
범위: 에테르·크림슨 대륙 주요 거점망
```

**명제형 규칙**:
- 마탑 간 거리가 수십 km를 초과하면 → 중계 마탑 없이 직접 전송 불가
- 기상 악조건(폭풍·눈보라) 발생 시 → 광학 신호 차단 → 완전 통신 두절
- 마탑 하나를 파괴하면 → 해당 릴레이 구간 전체 통신 마비

### 2-2. 세계수 지맥망

```
운용: 하이 엘프 평의회 + 크림슨 대륙 오크 족장들
방식: 세계수 뿌리 → 대지맥 원시 영력 파동 → 목적지 숲의 꽃잎·나뭇잎으로 수신
특성: 텍스트가 아니라 감정·이미지 형태로 전달 (감정 동기화). 나뭇잎 색 변화·바람소리로 경고
범위: 에테르 대륙 삼림 권역 내부 전용 (도시 대리석·사막 모래밭에서는 뿌리 끊김)
```

**명제형 규칙**:
- 세계수 뿌리가 없는 지역(크림슨·프로스트 등)에서는 → 지맥망 수신 불가
- 전쟁 시 세계수 도메인을 말살하면 → 해당 구역 정령연합 통신망 완전 차단 (전술 활용 가능)
- 감정 필터 없이 수신하면 → 타인의 공황·공포 감정 직격 → 수신자 패닉 위험

### 2-3. 심해 유물 텔레파시

```
운용: 해적 연합·암흑 상단·그림자 부대 (불법)
① 심해의 메아리 소라 (Abyssal Echo Conch)
   방식: 마력 소라 껍데기 → 물·소금기 대기를 매질로 초음파 진동 투사 → 짝 소라로 수신
   특성: 수면 아래를 통과하므로 제국 광학 탐지 완전 우회
② 망자의 턱뼈 텔레파시 (Jawbone of the Wraith)
   방식: 언데드 두개골 턱뼈 쌍 → 대지 밑 시체 잔류 에테르 지맥 경유 → 동기화 해골이 음성 재현
   특성: 강령 딥웹. 빛의 성국 발견 즉시 사형
범위: 해양 대륙 전역 + 타 대륙 연안 항구·공동묘지
```

**명제형 규칙**:
- 소라를 너무 오래 들으면 → 심해 고대 악신의 헛소리 혼입 → 착란·토혈
- 망자의 턱뼈를 쥔 손목은 → 날이 갈수록 생기를 잃고 부패
- 마법협회·성국은 불법 통신 채널로 지정 → 발각 시 유물 압수 + 처벌

### 2-4. 기타 통신 수단

| 수단 | 운용 | 특징 |
|---|---|---|
| **성국 뇌파 강제 연결 (Holy Communion)** | 성국 최고 사제 | 신도 간 강제 텔레파시. 명령 하달용. 비신도 도청 시 찬송가 과부하로 즉사 |
| **바람 정령 급행 우편 (Wind-Sprite Couriers)** | 제국·자유도시연합 | 바람 정령이 캡슐 운반. 대도시 간 1~2일 배달. 대중적 인프라 |
| **환수 전령 (Familiar Post)** | 마법사·정령사·용병단 | 세눈박이 부엉이·바람매 등 훈련 환수가 원시 영력 파장 추적 배달 |
| **원시 영력 영창 거울 (Reflection Mirror)** | 귀족·상인·용병 | 손거울~전신거울 크기 실시간 화상 통신. 사용자 체온·마력 소모 |
| **귓속말 자수정 귀걸이 (Whispering Amethyst)** | 군대 지휘관·모험 파티 | 조율된 자수정 귀걸이 착용. 전투 중 무선 음성 교신 |
| **마도 영상기 (Memory Projector)** | 마법협회 상위 | 크리스탈 실린더에 음성·영상 기억 → 홀로그램 재생. 의회급 회의 전용 |

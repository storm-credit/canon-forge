# 해양 대륙 (Oceanic Continent)

> Phase 2 · 2차 전수 대조 · 2026-06-17

## 핵심 정체

**"끝없는 바다"**. 육지가 섬 군도에 불과하고 대부분이 바다인 대륙. 표층(낭만 모험)과 심층(망자 금기)의 이중 세계로 구분된다. 표층은 마나 기류가 강해 함선이 마법 활공(帆空)으로 이동하며, 심층은 죽은 자와 거대 마수들의 영역으로 산 자의 접근을 거부한다.

> ⚠️ **레이어 주의**: 아래 본문(세력 구조·심층 금기)은 **세력 아카이브(01-8)** 기반 해석이다. 반면 **지리 원본(01-7)**은 동일 대륙을 *밝은 에픽 판타지* 톤으로 묘사하며, 심해(심연의 해구)를 금기가 아닌 **빛나는 태초의 성역**으로, 세력을 **자유 해방 연맹·황금 함대·심연의 성기사단**으로 제시한다(하위 색인 문서 참조). 두 레이어의 톤·세력명 통합은 세력 아카이브(01-8) 해양 작업 시 확정한다.

## 이중 구조 (표층·심층)

```
표층 (Surfaceworld)
├── 마나 기류 함선 항법 가능
├── 무역·모험·탐험 낭만
├── 세력 간 패권 경쟁
└── 섬 문명·항구 도시

심층 (Deepworld)                  ← 금기 영역
├── 빛이 닿지 않는 절대 암흑
├── 고대 해신 유적·봉인 괴수
├── 망자의 뱃길 — 죽은 자만 지나는 해류
└── 접근 자 = 실종 처리
```

**명제**: 심층 탐험을 시도한 자가 돌아오면 → 그는 산 자가 아니거나 심층과 계약을 맺은 자다.

## 고유 생존 규칙 (AI 집행 가능 명제형)

| 규칙 | 조건 | 결과 |
|---|---|---|
| **마나 기류 항법** | 마나 기류 구역에서 함선 마법 돛을 펼치면 | 풍력 불필요 → 마법 활공으로 이동 가능. 기류 이탈 시 조종 불능 |
| **심층 진입 금기** | 마나 기류가 끊기는 심층 경계를 넘으면 | 통신·마법 완전 차단 → 표층 귀환 불가 확률 85% 이상 |
| **고대 해신 유물 탐지** | 심층 특정 좌표에서 신성 마법 또는 소환 마법 시전하면 | 봉인된 고대 해신 반응 → 영역 전체 압박 공격 발동 |
| **바다 교단 해역 봉쇄** | 바다의 교단 인가 없이 특정 성해(聖海) 구역에 진입하면 | 교단 심문 함선 즉각 출동 → 이단 심문 절차 강제 개시 |
| **해류 역행** | ⚠️[추가] 죽은 자의 해류(망자 해류)를 역방향으로 항행하면 | 함선 마법 시스템 오작동 + 승무원 사기 급락 + 언데드 마수 유인 |

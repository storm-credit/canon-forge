# 오벨리스크 대륙 (Obelisk Continent)

> Phase 2 · 1차 정리 · 2026-06-16

## 핵심 정체

**"그림자의 대지"**. 거대한 오벨리스크(차원 쐐기) 군락이 대륙을 가득 채우고 있으며, 이 쐐기들이 심연(Abyss)의 침공을 봉인하고 있다. 태양빛이 닿지 않는 영원한 황혼의 땅. 산 자와 죽은 자, 심연의 군단이 하나의 대륙 위에서 3원 충돌 중이다. 영혼과 기억이 물질보다 더 실체적인 곳.

## 고유 생존 규칙 (AI 집행 가능 명제형)

| 규칙 | 조건 | 결과 |
|---|---|---|
| **영원한 황혼 결핍증** | 태양빛 완전 차단 환경에서 2주 이상 체류하면 | 뼈 연화증 + 괴혈병 동시 발병 → 운동 능력 상실 |
| **뼈의 평원 환청 발작** | 뼈의 평원 지대를 걸으면 | 걸음마다 간질 발작 유발 진동 + 죽은 자의 환청 → 방향 감각 마비 |
| **다크 에테르 늪 가스** | 다크 에테르 늪 위의 가스층에 노출되면 | 시신경 부패 → 빛이 고통으로 느껴지는 영구 광과민증 |
| **오벨리스크 공명 간섭** | 오벨리스크 반경 50m 내에서 대형 마법을 시전하면 | ⚠️[추가] 봉인 공명 간섭 → 차원 균열 발생 위험. 소환 마법 특히 위험 |
| **기억의 우물 접촉** | 기억의 우물(망자 기억 저장소)에 직접 손대면 | 타인의 죽음 기억 다운로드 → 정신 붕괴 또는 인격 분열 |

## 경제 체계

- **영혼 경제**: 영혼 결정체(Soul Crystal)가 화폐. 강력한 전사·마법사의 영혼일수록 고가
- **기억 거래**: 잊힌 자들의 연합이 기억의 우물에서 추출한 고인의 기억을 판매
- **오벨리스크 석재**: ⚠️[추가] 봉인 효과가 남아있는 오벨리스크 파편 — 전 세계 신성 마법사가 원함

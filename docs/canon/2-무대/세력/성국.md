# 성국 (Saint Haven) — 빛의 신정 왕국

> *"빛은 공평하게 내리쬔다. 다만, 그림자가 가장 짙게 드리우는 곳도 항상 빛 아래다."*

[[에테르대륙|에테르 대륙]] **서부** 고원에 자리한 대륙 최강의 신정(神政) 국가. 겉으로는 태양신 솔라리스의 영광을 전파하는 빛의 왕국이지만, 속으로는 왕권·교권·이단심문 권력이 서로를 견제하는 내부 갈등의 온상이다. 대륙 최고의 **대(對)심연 방위력**을 무기로 주변국의 목줄을 쥐고 흔드는 종교 제국.

> **신학 주의 (→ [[신앙체계]])**: 태양신 솔라리스의 실존 여부는 캐논에서 **불명**으로 둔다. 본 문서의 신성 서술은 "성국이 그렇게 *믿는다*"는 신앙 차원이며, 신의 실재를 전제하지 않는다.

## B. 통치 체제 & 권력 구조

세 축이 서로 견제한다 — **왕권(레오니스 성왕) · 교권(베네딕투스 교황) · 사법권(이단심문소)**. 이단심문소는 사실상 제3의 독자 권력으로, 그 수장은 교황 다음가는 실권자다.

**성직 계급 체계 (서열)**:

```
교황 (Pontifex Maximus, 황제급)
  └ 성왕 (Holy King, 국왕)
      └ 추기경 (Cardinal, 공작급 · 회의 7인)
          └ 이단심문관장 (Grand Inquisitor, 후작급)
              └ 성기사단 총단장 (장군급)
                  └ 대주교 (Archbishop, 백작급)
                      └ 지방 주교 (Bishop, 남작급)
                          └ 사제 / 성기사 (기사급)
```

- 승진 경로: 사제 → 주교 → 대주교 → 추기경. 세습 권력은 왕족 **산크투스(Sanctus) 가문**만 유지.

## G. 외교 관계망

대륙 최고의 대심연 방위력을 지렛대로 주변국을 압박한다. 표면 명분과 실제 행태가 다른 '위선적 종교 제국' 패턴이 일관된다.

| 대상 세력 | 관계 | 핵심 역학 | 조약 |
|---|---|---|---|
| **왕국연합** | 불평등 방위 동맹 | 성국의 신성 방벽 ↔ 왕국연합 육군 교환. 그러나 신앙 명목으로 내정 관여·"방벽 거두겠다" 협박 | 백색 방패의 맹약 |
| **자유도시연합** | 위선적 밀수 | 공식 국경 봉쇄하면서 고위 성직자가 뒷돈 성물 밀수. 다신앙 관용에 불만 | 좁은 문의 조약 |
| **마법협회** | 이념적 사생결단 | 마법협회를 "신의 영역을 훔치는 악마"로 규정, 발견 즉시 화형. 협상 불가 | 없음 |
| **정령연합** | 오만한 관용 | 다신교·애니미즘을 '이교'로 탄압하나, 차원 균열 방어에 엘프 숲 결계가 필요해 멸시하며 평화 유지 | 거짓된 잎새의 맹약 |
| **솔라리안 제국** | 적통성 전쟁 | 성국(생명·질서의 태양) vs 제국(파괴·정복의 화염). "누가 진정한 태양신 적통인가" 종교 전쟁. 무기한 성전 | 영구 파기 |
| **빙하의 신전** | 가혹한 이단 교정 | 빙하 수사를 멸시하나 심연 군단 남하를 막을 '북방 고기방패'로 활용·강제 포교 | 얼음과 태양의 조약 |

## 정리 메모 (편집장 노트)

### ✅ 정정한 모순 (원본 내부 충돌 → 캐논 확정)
- **지리적 위치 3중 충돌** (서부/동부/중앙): 기존 캐논 [[에테르대륙]]의 "성국=서부" 확정에 맞춰 **서부**로 통일.
- **수도 정체성** (솔라리스 vs 루멘): 캐논 [[에테르대륙]]이 "수도=솔라리스"로 확정 → **솔라리스=수도(종교·정치)**, 루멘=의료·행정 중심 도시로 정리. (원본 일부 문서의 '루멘=수도' 표기는 폐기)
- **태양신 명칭** (솔라리스/솔라/루미나 혼재): [[신앙체계]] 정본 **솔라리스(Solaris)**로 정규화. 신 실존은 불명 전제 유지.
- **퓨리피아 역할** (해상 요새 vs 화산 소각): 개별 요새 문서(화산 소각·격리)를 정본으로 채택, 명소 인덱스의 '해상 요새' 표기는 폐기.
- **요새 개수**: 원본 인덱스는 7개만 등재하나 본문 8개 존재 → **루멘 성채 포함 8개**로 확정.
- **인물명 표기**: 도서관장 '래퍼티(Rafferty)/라파엘(Raphael) 아르카디아' 혼재 → 다수 문서에 쓰인 **라파엘 아르카디아**로 통일.

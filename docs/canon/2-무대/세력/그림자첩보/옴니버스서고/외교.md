# 옴니버스 서고 — 외교

# 0. 옴니버스 서고 외교 관계

## 1. 개요
옴니버스 서고의 외교는 **'지식의 주도권'을 둘러싼 은밀한 학술·마법 전쟁**으로 요약된다. 정보가 곧 권력인 이들은 어느 세력과도 영구 동맹을 맺지 않으며, 기록을 사고팔며 동맹과 배신이 난무하는 외줄타기 전술을 구사한다. 칼이 아니라 '누가 무엇을 먼저 아느냐'로 우열을 가린다.

## 2. 주요 관계
- **마탑 연합과의 학술 냉전**: 고대 술식과 금서의 해석 주도권을 놓고 살벌한 은밀 전쟁을 벌인다. 표면상 정보를 교환하지만, 서로의 해독 성과를 가로채려는 첩보전이 끊이지 않는다.
- **국가·세력 상대 정보 판매**: 미개척지 지형·마수 주기 같은 독점 기록을 어느 진영에도 팔 수 있으나, 단일 유통 원칙상 같은 정보를 두 곳에 넘기지는 않는다.
- **추적자·집행 세력과의 마찰**: 서고의 기록을 무단 반출하거나 거래를 어긴 외부 세력은 즉시 은랑 기사단의 추적 대상으로 전환되어, 외교 관계가 적대로 뒤집힌다.

## 3. 서사 활용 포인트
- 서고가 어느 진영에 어떤 기록을 팔았는지가 곧 전장의 판세를 가르므로, '정보의 행방'을 쫓는 것 자체가 외교·첩보 서사의 축이 된다.

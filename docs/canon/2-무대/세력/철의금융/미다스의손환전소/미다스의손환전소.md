# 미다스의 손 환전소

금전을 벌기 위해 대자연의 순환을 폭력적으로 단절시킨 미다스의 연금술사들은, 자신들조차 육체의 일부가 기계와 금속으로 치환되어가는 역성 붕괴의 저주를 감내하고 있다.

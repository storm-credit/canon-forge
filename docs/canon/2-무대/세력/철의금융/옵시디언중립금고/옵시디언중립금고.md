# 옵시디언 중립 금고

금고 자체가 수천 년 전 형성된 대자연의 결계 위에 지어졌으며, 내부의 진실된 장부는 공간 마법을 통해 다른 차원에 묶여 있어 물리적인 탈취가 원천 봉쇄되어 있다.

# 무혈의 징수단 — 외교

# 0. 무혈의 징수단 외교 관계

## 1. 개요
외교 기조: 국왕들조차 이들에게 빚을 지고 있어, 실질적으로 왕실 내부에 막대한 영향력을 행사한다.

절망적인 전시 상황 속에서도 동맹군 통수를 치는 이기적이고 압도적인 외교술.

# 하데스 무혈 징수단

하데스 무혈 징수단의 존재는 은빛 옥좌가 닿지 못하는 마나의 그림자가 얼마나 짙은지를 보여주는 살아있는 증거다. 이들과 엮이는 것은 단순히 돈을 잃는 것이 아니라 대자연의 저주을 스스로 짊어지는 행위이다.

# 하이본 기축 은행

우월한 혈통만이 지배할 자격이 있다는 오만에 빠진 이들은, 이계의 침식조차 자본력과 혈통의 필터링으로 통제할 수 있다고 믿는 오만한 귀족들의 탐욕적 집합체이다.

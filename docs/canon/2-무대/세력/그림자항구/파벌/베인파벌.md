# 베인 파벌

> *"독은 보이지 않는다. 그래서 가장 무서운 무기다."*

---

## 1. 파벌 개요

| 항목 | 내용 |
|------|------|
| **파벌명** | 베인 파벌 (Bane Faction) |
| **지위** | 그림자 항구 암살·독·특수작전 총괄 |
| **본거지** | [[검은창고군|검은 창고군]], 검빙 절벽, 난파선 묘역 수중 |
| **핵심 자산** | 역병 [[직업체계|연금술사]]단, 침묵의 그림자, 달빛 저격단, 심연의 잠수부대 |
| **정치 노선** | 두려움 기반 · 더러운 일 독점 · 심연 접촉 |

> [!warning] 항구의 어둠
> 모르굴 베인은 **"독을 품은 자"**로, 항구에서 가장 두려운 존재.
> 그의 **역병 [[직업체계|연금술사]]단**은 독가스와 키메라(생체 병기)를 운용하며,
> 항구 자체를 위협할 수준의 파괴력을 품고 있다.

## 4. 파벌 내부 갈등

- **모르굴의 독 연구**: 실험이 항구 자체를 오염시킬 수준 — 통제 불능
- **가레스의 냉혹함**: 처형 기준이 점점 모호 — 두려움이 공포로 변질
- **이즈라엘의 고독**: 절벽 위 혼자 사는 저격수 — 정신 건강 위기
- **데이비온의 심연 접촉**: 잠수 중 심연 존재와 거래 — 오벨리스크 연결 고리

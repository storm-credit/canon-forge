# 레온하르트 가문

> *"속도가 곧 승리다. 폭풍을 뚫는 자만이 살아남는다."*

## 1. 조직 개요

| 항목 | 내용 |
|------|------|
| **조직명** | 레온하르트 가문 (House of Leonhardt) |
| **지위** | 기동 함대 운영, 선봉 타격의 명가 |
| **주요 인물** | 아르노 레온하르트, 마르셀 레온하르트, 아주라 스톰 (함대 제독/소속) |

## 2. 조직의 특징

바르카스와 달리 '속도전'과 '치고 빠지기' 전술의 신봉자들입니다. 주로 해상에서 무역선을 노리는 해적들의 기동 선박을 추적하고 격퇴하는 임무에 특화되어 있으며, 쾌속선과 하늘에서 폭탄을 투하하는 공중 타격대(Sky Guards) 운영에 투자를 아끼지 않습니다.

# 드레이크 씨족

> *"하늘은 용의 것이다. 그리고 용이 선택한 자가, 하늘 아래를 다스린다."*

---

## 1. 씨족 개요

| 항목 | 내용 |
|------|------|
| **씨족명** | 드레이크 씨족 (Drake Clan) |
| **지위** | 용의 후예 부족장 · 용[[직업체계|기사]] 최고평의회 의장 |
| **본거지** | 용의 심장 요새 |
| **핵심 자산** | 드래곤 의회 연결, 시험의 탑, 외교·정찰부, 기록보관소 |
| **정치 노선** | 균형 유지 · 용과 인간의 공존 · 주전-수호 중재 |

> [!note] 용의 선택을 받은 왕
> 타이란 드레이크(SSS급)는 **가장 강해서가 아니라, 용이 선택했기에** 부족장이다.
> 세습도, 결투도, 정치도 아닌 — 오직 용의 심장에 닿은 자만이 이 자리에 선다.

## 4. 내부 갈등

- **타이란의 줄타기**: 주전파와 수호파 사이 균형 — 한쪽을 택하면 내전 촉발
- **카사르의 예언**: "다가올 재앙"을 예견하지만 구체적 시기를 모름 — 예방 불가
- **에리온의 발견**: 고대 기록에서 "용과 인간의 계약이 영원하지 않다"는 기록 발굴
- **아레나의 최전선**: 솔라리안 스톰홀드와 매일 대치 — 자칫 전면전 촉발

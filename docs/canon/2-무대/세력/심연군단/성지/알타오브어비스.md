# 알타 오브 어비스

> *"이 제단 위에서 바친 것은 돌아오지 않는다. 하지만 심연은 기억한다."*

---

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 심연의 입 상단부, 균열 가장자리 돌출 절벽 |
| **외관** | 검은 수정 기둥 8개가 둘러싼 원형 제단. 중앙에 무한히 깊은 틈 |
| **역할** | 심연 군단 최고위 의식 장소 / 동화 의식 거행지 |

---

## 2. 제단의 기능

### ■ 동화 의식
- 포획한 영웅/지도자를 심연에 바쳐 **심연의 존재로 재탄생** 시킴
- 의식 성공 시: 대상의 기억·능력 유지, 충성심만 심연으로 전환
- 의식 실패 시: 완전 소멸, 심연도 기억하지 못함

### ■ 속삭임 증폭
- 제단 근처에서 심연의 속삭임이 **물리적 파동**으로 변환
- 수 km 범위의 생물에게 악몽, 유혹, 기억 왜곡 유발

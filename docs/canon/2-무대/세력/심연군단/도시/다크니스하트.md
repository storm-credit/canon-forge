# 다크니스 하트

> *"빛은 이곳에서 죽는다. 희망도, 영혼도."*

| 항목 | 내용 |
|------|------|
| **위치** | 오벨리스크 대륙 중부, 심연의 구덩이 깊숙한 곳 |
| **역할** | 심연 군단의 수도, 심연의 군주 거주지 |
| **분류** | 수도 (Capital) |

---

## 개요

심연의 군주가 거주하는 거대한 암흑 요새. 영혼을 삼키는 검은 불길이 영원히 타오르며, 악마와 언데드가 가득한 지옥과 같은 도시이다.

4. **균열 봉쇄** - 심연에서 쏟아지는 균열 봉쇄

# 그림자 첩보 조직 (Shadow Intelligence)

대아스트라리스 세계의 암살·첩보·공작을 전담하는 6개 초국가 비밀 조직의 범주.
어느 단일 국가의 법제에도 속하지 않으며, 의뢰·계약·정보 독점이라는 음지의 논리로 자생한다.
구성원 전원은 신원이 말소되거나 처음부터 공식 기록에 존재하지 않으며,
대륙 간 분쟁의 이면에 이들의 개입이 없었던 사건은 드물다.

# 일루미나티 동력 발전소 — 외교

# 0. 일루미나티 동력 발전소 외교 관계

## 1. 개요
외교 기조: 인체 실험 문제로 성국(Saint Haven)에게는 영원한 이단으로 낙인찍혀 소탕의 대상이다.

절망적인 전시 상황 속에서도 동맹군 통수를 치는 이기적이고 압도적인 외교술.

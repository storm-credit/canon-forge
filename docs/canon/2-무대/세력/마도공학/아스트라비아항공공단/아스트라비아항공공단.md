# 아스트라비아 항공 공단

중력을 거스르는 행위 자체가 대자연의 섭리를 배반하는 일. 비공정에 탑승한 자들은 극심한 에테르 멀미와 영혼의 이탈 현상을 혹독한 대가으로 겪는다.

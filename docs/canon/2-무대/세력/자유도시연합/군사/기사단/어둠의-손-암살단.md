# 02. 어둠의 손 암살단 (Dark Hand Assassins)

> *"빛이 강할수록 그림자는 짙어지지. 우리는 그 그림자 속에 산다."*

| 항목 | 내용 |
|---|---|
| **본부** | 카르델라 (지하 본부 '검은 둥지') |
| **지휘관** | 마테오 블랙잭 (Mateo Blackjack, 대부) |
| **규모** | 정예 암살자 500명 + 하부 조직원 수천 명 |
| **상징** | 피 묻은 검은 손바닥 (Bloody Handprint) |
| **역할** | 청부 살인·정보 거래·도둑질·뒷골목 질서 유지 |

**핵심 부대**:
- **집행자 (Executioners)**: 조직 배신자·타겟 제거 최정예 암살자
- **그림자 스파이 (Shadow Spies)**: 상점·여관·성직자 위장 정보 수집
- **독물 연구소**: 독·마약 제조
- **장물아비 조합**: 훔친 물건 현금화 유통망

**지휘 체계**: 대부(마테오) → 핸드(5명 최고 간부, 암살/정보/밀수/도박/마약 담당) → 핑거(행동 대장)

**거점**: 검은 둥지(카르델라 지하 미궁형 본부) · 침묵의 성소(암살자 훈련소) · 카르델라 투기장 · 지하 수로 네트워크

# 10. 밤안개 첩보대 (Nightmist Intelligence Corps)

> *"안개는 모든 것을 덮고, 우리는 그 안에서 진실을 본다."*

| 항목 | 내용 |
|---|---|
| **본부** | 섀도우 게이트 (비밀 기지) |
| **지휘관** | 칼리스 노이르 (Kalis Noir, 국장) |
| **규모** | 정예 요원 100명 (가장 소수 정예) |
| **상징** | 안개 속의 눈 (Eye in Mist) |
| **역할** | 정보 수집·첩보·역정보 유포·적국 요인 납치 |

**핵심 부대**:
- **고스트 (Ghost)**: 존재 자체가 지워진 최상위 요원. 칼리스 직속
- **도청반**: 마법 도청 장치·사역마 이용 대륙 전역 도청
- **분석반**: 정보 진위 파악·암호 해독
- **공작반**: 적국 내분 조장·심리전 역정보 유포

**지휘 체계**: 칼리스 노이르(국장) → 팀장(성국/제국/왕국연합 담당) → 오퍼레이터(현장 지원 본부 요원)

**거점**: 섀도우 게이트(지도에 없는 결계 비밀 기지) · 각국 주요 도시 안가(安家) · 기록의 방(대륙 모든 비밀·스캔들·약점 데이터베이스)

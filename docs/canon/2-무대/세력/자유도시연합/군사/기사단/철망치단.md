# 03. 철망치단 (Iron Hammer Battalion)

> *"강철과 화약! 이것이 우리의 대화법이다!"*

| 항목 | 내용 |
|---|---|
| **본부** | 포지게이트 (군사 도시) |
| **지휘관** | 브롬 아이언비어드 (Brom Ironbeard, 드워프 전쟁 영웅) |
| **기술 고문** | 에드가 카르다스 (무기 개발·화력 자문) |
| **규모** | 중장보병 2,000명 + 공병 500명 (드워프·인간 혼성) |
| **상징** | 불타는 망치 (Flaming Hammer) |
| **역할** | 공성전·요새 방어·중장갑 돌격·마도 공학 무기 운용 |

**핵심 부대**:
- **철갑 돌격대 (Ironclad Breakers)**: 전신 강철 갑옷 + 증기 동력 해머, 최전방 돌격
- **포격 지원대**: 이동식 대포·마법 박격포 운용
- **공병대 (Sapper Corps)**: 참호 구축·방어탑 건설·성벽 폭파
- **골렘 기갑대**: 에드가 카르다스 제작 전투용 골렘 운용

**거점**: 포지게이트 요새(본진) · 실버포트 해안 방어 파견 기지 · 철심 산(무기 제작소·광산) · 폭발물 연구소

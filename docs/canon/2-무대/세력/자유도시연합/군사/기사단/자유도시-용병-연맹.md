# 01. 자유도시 용병 연맹 (Free Mercenary Alliance)

> *"돈과 신용, 그리고 실력. 이 세 가지면 우리는 왕국도 무너뜨린다."*

| 항목 | 내용 |
|---|---|
| **본부** | 머시너리 게이트 (용병 길드 본부) |
| **지휘관** | 다리안 벨가르드 (Darian Belgarde, 용병왕) |
| **규모** | 등록 용병 30,000명 이상 (대륙 최대) |
| **상징** | 교차된 검과 동전 (Sword & Coin) |
| **역할** | 용병 중개·계약 보증·분쟁 개입·대규모 용병단 운용 |

**핵심 부대**:
- **S급 용병단**: 국가 정규군을 능가하는 전설적 용병들
- **전쟁 용병대 (War Freelancers)**: 대규모 전쟁 시 군단 단위 고용
- **호위 길드 (Escort Guild)**: 상단 호위·VIP 경호 전문
- **몬스터 헌터 길드**: 마수 토벌 전문
- **용병 중재 위원회**: 의뢰인↔용병 분쟁(먹튀, 배신) 심판

**등급 시스템**: Iron → Bronze → Silver → Gold → Platinum → Diamond. 상위 랭커는 독자 부대 지휘.

**거점**: 머시너리 게이트(의뢰지·술집·은행 합체) · 명예의 전당(전설적 용병 무기 전시) · 훈련소 · 블랙 리스트 보드

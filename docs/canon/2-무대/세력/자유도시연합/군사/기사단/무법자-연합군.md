# 06. 무법자 연합군 (Outlaw Legion)

> *"바다엔 왕이 없고, 우리에겐 법이 없다!"*

| 항목 | 내용 |
|---|---|
| **본부** | 포트 넥서스 (무법자 항구) |
| **지휘관** | 데릭 로어슨 (Derek Rawson, 제독) |
| **규모** | 해적선 500척 + 선원 10,000명 이상 |
| **상징** | 해골과 닻 (Skull & Anchor) |
| **역할** | 해상 약탈·게릴라전·비정규 용병·암시장 운영 |

**핵심 부대**:
- **해적단 연합 (Pirate Syndicate)**: 여러 해적 선장들의 회의체
- **금화 약탈자들 (Gold Raiders)**: 내륙 대상 터는 산적 연합
- **밤도적단 (Night Thieves)**: 도시 내 도둑 길드 (마테오 조직과 별개)
- **무법 용병대 (Rogue Mercs)**: 다리안 연맹에서 제명된 자들의 사설 용병단

**거점**: 무법자 항구(실버포트 숨겨진 구역. 고대 해적 보물 지도 조각 은닉) · 나이트폴 인근 숲속 성채 · 이동형 캠프

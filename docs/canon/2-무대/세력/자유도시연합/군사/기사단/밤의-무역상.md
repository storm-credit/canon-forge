# 07. 밤의 무역상 (Traders of the Night)

> *"해 뜰 때 거래하면 세금을 내야 하지만, 해 질 때 거래하면 그게 다 순수익이지."*

| 항목 | 내용 |
|---|---|
| **본부** | 포트 넥서스 및 전 대륙 뒷골목 |
| **지휘관** | 모이라 와일드웨이브 (해적·내륙 연락책) |
| **규모** | 점조직, 파악 불가 (수천 명 추산) |
| **상징** | 달빛 아래의 배 (Ship under Moonlight) |
| **역할** | 밀수·금지 품목 유통·비상 보급로 확보·정보 전달 |

**핵심 부대**:
- **유령 선단 (Ghost Fleet)**: 안개 틈타 감시망 뚫는 고속 밀수선
- **땅굴쥐 (Tunnel Rats)**: 지하 통로 운반책. 이사벨 카르도 운송 길드와 비밀 협력
- **모조품 장인 조합**: 가짜 명품·위조 신분증 제작
- **검은 경매소**: 장물 전용 비밀 경매

**거점**: 제3 부두(포트 넥서스, 밤에만 불 켜지는 시장) · 위장 비밀 창고 · 주점 '검은 고양이'(접선 장소)

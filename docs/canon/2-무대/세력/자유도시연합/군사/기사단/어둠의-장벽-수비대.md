# 09. 어둠의 장벽 수비대 (Dark Wall Guards)

> *"장벽이 무너지면, 자유도 끝이다."*

| 항목 | 내용 |
|---|---|
| **본부** | 크로스로드 외곽 성벽 + 주요 방어 거점 |
| **지휘관** | 가릭 아이언월 (Garrick Ironwall, 다리안 벨가르드의 옛 부관) |
| **규모** | 정규군 5,000명 (가장 많은 병력) |
| **상징** | 검은 방패와 성벽 (Black Shield & Wall) |
| **역할** | 성벽 방어·출입 통제·공성전 방어·몬스터 웨이브 저지 |

**핵심 부대**:
- **장벽 수호대 (Wall Guardians)**: 24시간 성벽 순찰·외부 위협 감시
- **석궁 사수대**: 성벽 위 원거리 저격
- **관문 수비대 (Gatekeepers)**: 모든 인원·물자 검문 검색
- **역병 처리반**: 오염된 마수·전염병 소각 처리

**거점**: 검은 장벽(크로스로드 흑요석 성벽, 높은 마법 저항) · 방어 사령부(작전 회의실+무기고 벙커) · 지하 병영

# 자유도시연합 군사 조직 심화

자유도시연합의 군사력은 정규 직할군 없이 **용병·길드·사설 무력**으로 구성된다. 10개 부대가 각각의 전문 영역에서 도시의 방어와 이익을 수호한다.

---

## 01. 자유도시 용병 연맹 (Free Mercenary Alliance)

> *"돈과 신용, 그리고 실력. 이 세 가지면 우리는 왕국도 무너뜨린다."*

| 항목 | 내용 |
|---|---|
| **본부** | 머시너리 게이트 (용병 길드 본부) |
| **지휘관** | 다리안 벨가르드 (Darian Belgarde, 용병왕) |
| **규모** | 등록 용병 30,000명 이상 (대륙 최대) |
| **상징** | 교차된 검과 동전 (Sword & Coin) |
| **역할** | 용병 중개·계약 보증·분쟁 개입·대규모 용병단 운용 |

**핵심 부대**:
- **S급 용병단**: 국가 정규군을 능가하는 전설적 용병들
- **전쟁 용병대 (War Freelancers)**: 대규모 전쟁 시 군단 단위 고용
- **호위 길드 (Escort Guild)**: 상단 호위·VIP 경호 전문
- **몬스터 헌터 길드**: 마수 토벌 전문
- **용병 중재 위원회**: 의뢰인↔용병 분쟁(먹튀, 배신) 심판

**등급 시스템**: Iron → Bronze → Silver → Gold → Platinum → Diamond. 상위 랭커는 독자 부대 지휘.

**거점**: 머시너리 게이트(의뢰지·술집·은행 합체) · 명예의 전당(전설적 용병 무기 전시) · 훈련소 · 블랙 리스트 보드

## 02. 어둠의 손 암살단 (Dark Hand Assassins)

> *"빛이 강할수록 그림자는 짙어지지. 우리는 그 그림자 속에 산다."*

| 항목 | 내용 |
|---|---|
| **본부** | 카르델라 (지하 본부 '검은 둥지') |
| **지휘관** | 마테오 블랙잭 (Mateo Blackjack, 대부) |
| **규모** | 정예 암살자 500명 + 하부 조직원 수천 명 |
| **상징** | 피 묻은 검은 손바닥 (Bloody Handprint) |
| **역할** | 청부 살인·정보 거래·도둑질·뒷골목 질서 유지 |

**핵심 부대**:
- **집행자 (Executioners)**: 조직 배신자·타겟 제거 최정예 암살자
- **그림자 스파이 (Shadow Spies)**: 상점·여관·성직자 위장 정보 수집
- **독물 연구소**: 독·마약 제조
- **장물아비 조합**: 훔친 물건 현금화 유통망

**지휘 체계**: 대부(마테오) → 핸드(5명 최고 간부, 암살/정보/밀수/도박/마약 담당) → 핑거(행동 대장)

**거점**: 검은 둥지(카르델라 지하 미궁형 본부) · 침묵의 성소(암살자 훈련소) · 카르델라 투기장 · 지하 수로 네트워크

## 03. 철망치단 (Iron Hammer Battalion)

> *"강철과 화약! 이것이 우리의 대화법이다!"*

| 항목 | 내용 |
|---|---|
| **본부** | 포지게이트 (군사 도시) |
| **지휘관** | 브롬 아이언비어드 (Brom Ironbeard, 드워프 전쟁 영웅) |
| **기술 고문** | 에드가 카르다스 (무기 개발·화력 자문) |
| **규모** | 중장보병 2,000명 + 공병 500명 (드워프·인간 혼성) |
| **상징** | 불타는 망치 (Flaming Hammer) |
| **역할** | 공성전·요새 방어·중장갑 돌격·마도 공학 무기 운용 |

**핵심 부대**:
- **철갑 돌격대 (Ironclad Breakers)**: 전신 강철 갑옷 + 증기 동력 해머, 최전방 돌격
- **포격 지원대**: 이동식 대포·마법 박격포 운용
- **공병대 (Sapper Corps)**: 참호 구축·방어탑 건설·성벽 폭파
- **골렘 기갑대**: 에드가 카르다스 제작 전투용 골렘 운용

**거점**: 포지게이트 요새(본진) · 실버포트 해안 방어 파견 기지 · 철심 산(무기 제작소·광산) · 폭발물 연구소

## 04. 까마귀 전사단 (Raven Warriors)

> *"자유도시의 평화는 피로 지켜진다. 우리는 혼돈을 제어하는 전사들이다!"*

| 항목 | 내용 |
|---|---|
| **본부** | 크로스로드 (까마귀 성채) |
| **지휘관** | 오스카 아르젠 (Oscar Argen, 총사령관) |
| **규모** | 약 3,000명 (정예 근위대 포함) |
| **상징** | 검은 까마귀 (Raven) |
| **역할** | 도시 치안 유지·범죄 진압·대테러 작전·VIP 경호 |

**핵심 부대**:
- **까마귀 근위대 (Raven Guard)**: 요인 경호 최정예 부대. 오스카 직접 지휘
- **철주먹단 (Iron Fist)**: 폭동 진압·시가전. 비살상 제압술(몽둥이·방패)
- **안개 감시단 (Mist Watchers)**: 도시 잠복·범죄 사전 감지
- **어둠 사냥꾼단 (Night Hunters)**: 야간 전용 특수 기동대

**거점**: 까마귀 성채(크로스로드 중심부, 감옥+사령부. 지하 '고대 까마귀의 깃털' 비밀 금고) · 감시의 탑(주요 길목 망루, 마법 통신) · 피의 광장(공개 재판·처형장)

## 05. 은화 사슬단 (Silver Chains Syndicate)

> *"전쟁은 칼로 하는 게 아니다. 돈으로 하는 것이다."*

| 항목 | 내용 |
|---|---|
| **본부** | 골든마르크 (은화의 성전) |
| **지휘관** | 세실리아 메르카토르 (의장) |
| **역할** | 경제 전쟁·자금줄 차단·매수·로비·고리대금 |
| **상징** | 은화로 묶인 사슬 (Chains of Coin) |

**핵심 부대**:
- **금융 감시단 (Financial Overseers)**: 각국 환율·물가 조작으로 적국 경제 파탄
- **고용된 검 (Hired Blades)**: 최상급 용병 사설 부대 (다리안 연맹과 별개)
- **상업 첩보대**: 경쟁 상단 기밀 탈취·유통망 파괴
- **비밀 계약단**: 정치인·장군 매수 로비스트

**거점**: 은화의 성전(골든마르크, 대륙 최대 은행+금고) · 자유시장 회관(크로스로드) · 비밀 계약소

## 06. 무법자 연합군 (Outlaw Legion)

> *"바다엔 왕이 없고, 우리에겐 법이 없다!"*

| 항목 | 내용 |
|---|---|
| **본부** | 포트 넥서스 (무법자 항구) |
| **지휘관** | 데릭 로어슨 (Derek Rawson, 제독) |
| **규모** | 해적선 500척 + 선원 10,000명 이상 |
| **상징** | 해골과 닻 (Skull & Anchor) |
| **역할** | 해상 약탈·게릴라전·비정규 용병·암시장 운영 |

**핵심 부대**:
- **해적단 연합 (Pirate Syndicate)**: 여러 해적 선장들의 회의체
- **금화 약탈자들 (Gold Raiders)**: 내륙 대상 터는 산적 연합
- **밤도적단 (Night Thieves)**: 도시 내 도둑 길드 (마테오 조직과 별개)
- **무법 용병대 (Rogue Mercs)**: 다리안 연맹에서 제명된 자들의 사설 용병단

**거점**: 무법자 항구(실버포트 숨겨진 구역. 고대 해적 보물 지도 조각 은닉) · 나이트폴 인근 숲속 성채 · 이동형 캠프

## 07. 밤의 무역상 (Traders of the Night)

> *"해 뜰 때 거래하면 세금을 내야 하지만, 해 질 때 거래하면 그게 다 순수익이지."*

| 항목 | 내용 |
|---|---|
| **본부** | 포트 넥서스 및 전 대륙 뒷골목 |
| **지휘관** | 모이라 와일드웨이브 (해적·내륙 연락책) |
| **규모** | 점조직, 파악 불가 (수천 명 추산) |
| **상징** | 달빛 아래의 배 (Ship under Moonlight) |
| **역할** | 밀수·금지 품목 유통·비상 보급로 확보·정보 전달 |

**핵심 부대**:
- **유령 선단 (Ghost Fleet)**: 안개 틈타 감시망 뚫는 고속 밀수선
- **땅굴쥐 (Tunnel Rats)**: 지하 통로 운반책. 이사벨 카르도 운송 길드와 비밀 협력
- **모조품 장인 조합**: 가짜 명품·위조 신분증 제작
- **검은 경매소**: 장물 전용 비밀 경매

**거점**: 제3 부두(포트 넥서스, 밤에만 불 켜지는 시장) · 위장 비밀 창고 · 주점 '검은 고양이'(접선 장소)

## 08. 깃털 맹약 (Feather Covenant)

> *"칼로 벤 상처는 아물지만, 계약으로 묶인 영혼은 영원히 구속된다."*

| 항목 | 내용 |
|---|---|
| **본부** | 황금 맹약의 신전 (골든마르크) |
| **지휘관** | 레이나 브라이트헤븐 (맹약의 수호자) |
| **규모** | 변호사·사제·외교관 등 지식인 500명 |
| **상징** | 황금 깃펜과 천칭 (Golden Quill & Scales) |
| **역할** | 법률 자문·계약 공증·분쟁 중재·외교 사절 |

**핵심 부대**:
- **공증 사제단 (Notary Priests)**: 신의 이름으로 계약 공증. 위반 시 저주 발동
- **분쟁 조정 위원회**: 상단 간 다툼·영토 분쟁 평화적 해결
- **외교 사절단**: 타국과의 협상. 말빨 센 인재 집합
- **계약 집행관 (Contract Enforcers)**: 계약 위반자 추적 처벌 (물리적+마법적 페널티)

**거점**: 황금 맹약의 신전(부의 신 신전+대법원) · 계약의 도서관(모든 계약서 원본 보관, 마법 결계) · 중재의 방(합의될 때까지 감금)

## 09. 어둠의 장벽 수비대 (Dark Wall Guards)

> *"장벽이 무너지면, 자유도 끝이다."*

| 항목 | 내용 |
|---|---|
| **본부** | 크로스로드 외곽 성벽 + 주요 방어 거점 |
| **지휘관** | 가릭 아이언월 (Garrick Ironwall, 다리안 벨가르드의 옛 부관) |
| **규모** | 정규군 5,000명 (가장 많은 병력) |
| **상징** | 검은 방패와 성벽 (Black Shield & Wall) |
| **역할** | 성벽 방어·출입 통제·공성전 방어·몬스터 웨이브 저지 |

**핵심 부대**:
- **장벽 수호대 (Wall Guardians)**: 24시간 성벽 순찰·외부 위협 감시
- **석궁 사수대**: 성벽 위 원거리 저격
- **관문 수비대 (Gatekeepers)**: 모든 인원·물자 검문 검색
- **역병 처리반**: 오염된 마수·전염병 소각 처리

**거점**: 검은 장벽(크로스로드 흑요석 성벽, 높은 마법 저항) · 방어 사령부(작전 회의실+무기고 벙커) · 지하 병영

## 10. 밤안개 첩보대 (Nightmist Intelligence Corps)

> *"안개는 모든 것을 덮고, 우리는 그 안에서 진실을 본다."*

| 항목 | 내용 |
|---|---|
| **본부** | 섀도우 게이트 (비밀 기지) |
| **지휘관** | 칼리스 노이르 (Kalis Noir, 국장) |
| **규모** | 정예 요원 100명 (가장 소수 정예) |
| **상징** | 안개 속의 눈 (Eye in Mist) |
| **역할** | 정보 수집·첩보·역정보 유포·적국 요인 납치 |

**핵심 부대**:
- **고스트 (Ghost)**: 존재 자체가 지워진 최상위 요원. 칼리스 직속
- **도청반**: 마법 도청 장치·사역마 이용 대륙 전역 도청
- **분석반**: 정보 진위 파악·암호 해독
- **공작반**: 적국 내분 조장·심리전 역정보 유포

**지휘 체계**: 칼리스 노이르(국장) → 팀장(성국/제국/왕국연합 담당) → 오퍼레이터(현장 지원 본부 요원)

**거점**: 섀도우 게이트(지도에 없는 결계 비밀 기지) · 각국 주요 도시 안가(安家) · 기록의 방(대륙 모든 비밀·스캔들·약점 데이터베이스)

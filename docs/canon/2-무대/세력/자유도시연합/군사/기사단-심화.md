# 자유도시연합 군사 조직 심화

자유도시연합의 군사력은 정규 직할군 없이 **용병·길드·사설 무력**으로 구성된다. 10개 부대가 각각의 전문 영역에서 도시의 방어와 이익을 수호한다.

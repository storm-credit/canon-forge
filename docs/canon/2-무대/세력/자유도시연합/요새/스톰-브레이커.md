# 스톰 브레이커 (Stormbreaker) — 폭풍도 이 요새 앞에서는 멈춘다. 멈추지 않으면 쪼개진다.

> *"타이푼 호가 출격하면 바다가 비켜간다. 비켜가지 않으면 타이푼 호가 가른다."*
> — 연합 해군 제1함대 구호

에테르 대륙 남단 곶(Cape)에 세워진 **연합 해군 제1함대 해안 방어 요새**. 포트 넥서스 외곽 해상에 위치한 이 요새는 해적 소탕, 해상 재난 구조, 태풍 감시, 그리고 심해 마수 대응의 최전선이다. 데릭 로어슨이 해상 작전을 지휘하며, 셀레나 와일드웨이브가 이중첩자로 내부 정보를 유출하려 한다.

---

## 기본 정보

| 항목 | 내용 |
|---|---|
| **위치** | 에테르 대륙 남단 곶(Cape), 포트 넥서스 외곽 해상 |
| **역할** | 해안 경비, 해적 소탕, 해상 재난 구조, 태풍 감시 |
| **해상 작전 지휘관** | **데릭 로어슨** (제독) [C: 원본 표기 드레이크 루가 → 데릭 로어슨으로 정본 확정] |
| **이미지** | 100km까지 빛이 닿는 거대 등대, 폭풍 속에서도 흔들리지 않는 성벽, 지하 독에서 출격하는 군함 |

**핵심 시설**: 거대 등대(Grand Lighthouse, 100km 마법 조명) · 해안 포대(마도포, 적함 격침용) · 지하 독(Dock, 비밀 출격 통로) · 기상 관측소

## 요새 구조

### 상징적 지형

| 지형 | 설명 |
|---|---|
| **폭풍의 곶** | 요새가 위치한 절벽 곶. 연중 강풍이 불어 사람이 날아갈 정도 |
| **사이렌의 암초** | 요새 서쪽 암초 지대. 수십 척의 적선 잔해가 쌓여 있음. 사이렌이 산다는 전설 |
| **블루 홀** | 요새 남쪽 심해 구멍. 심해 마수 정기 출몰 지점. 해군이 주기적으로 정화 작전 실시 |

### 주요 구역

| 구역 | 특징 |
|---|---|
| **사령부** | 데릭 로어슨 해상 작전 지휘 본부. 해도·기상 데이터 실시간 수집 |
| **포대 구역** | 마도포 12문 배치. 사거리 30km. 자동 조준 마법 적용 |
| **격납고** | 그리폰·와이번 기마 부대 격납. 공중 정찰 및 지원 출격 |
| **감옥 타워** | 나포된 해적·적 함장 수감. 심문실 포함 |

## 정치·행정 구조

|---|---|
| **연합 해군 제1함대 사령부** | 데릭 로어슨 총지휘. 포트 넥서스와 이원 지휘 체계 |
| **기상 마법사단** | 날씨 관측 및 제한적 기상 조절 담당. 태풍 진로 예측 전문 |

**기함 타이푼 호**: 연합 해군 최대·최강 함선. 스톰 브레이커를 기지로 삼음. 타이푼 호가 출격하면 대부분의 해적이 전투 없이 도주한다.

# 크로스로드 (Crossroad) — 모든 길은 여기서 시작하고, 여기서 끝난다

> *"0번 이정표를 찾으면 어디든 갈 수 있습니다. 단, 가는 비용은 본인 부담입니다."*
> — 크로스로드 센트럴 스테이션 안내 방송

[[자유도시연합|자유도시연합(Crossroad Cities)]]의 **행정 수도이자 대륙 물류 허브**. 에테르 대륙 최중앙, 4대 간선도로가 만나는 지점에 건설된 이 도시는 연합 의회가 소재하며 대륙 전체의 운송·여객 흐름을 관장한다. 이사벨 카르도의 한마디가 대륙 전체 물류를 마비시킬 수 있는 것처럼, 크로스로드를 장악하는 자가 에테르를 장악한다.

---

## 기본 정보

| 항목 | 내용 |
|---|---|
| **위치** | 에테르 대륙 최중앙, 4대 간선도로 교차점 |
| **역할** | 자유도시연합 행정 수도, 물류·여객 운송 허브, 의회 소재지 |
| **통치자** | **이사벨 카르도** (운송 길드장) [C: 원본 표기 마리안 카르도 → 이사벨 카르도로 정본 확정] |
| **이미지** | 끝없이 오가는 마차, 거대한 대시계탑, 광장의 황금 이정표 돌기둥 |

**핵심 시설**: 리버티 스퀘어(독립 선언 광장) · 센트럴 스테이션 · 운송 길드 본부 · 아이언 오스 보급로 입구

## 도시 구조

### 상징적 지형

| 지형 | 설명 |
|---|---|
| **0번 이정표** | 도시 한가운데 세워진 황금 돌기둥. 에테르 대륙 모든 도시까지의 거리가 새겨져 있음 |
| **대시계탑** | 도시 상징. 대륙 표준시 기준. 매 정시마다 출발 마차 목록을 알리는 마법 방송 송출 |
| **마차의 강** | 도시를 관통하는 4대 간선도로. 하루 수만 대의 마차가 오가 강물처럼 흐름 |

### 주요 지구

| 지구 | 특징 |
|---|---|
| **터미널 지구** | 센트럴 스테이션 중심. 대륙 전역 마차 노선 출발·도착. 환전소·보험사 밀집 |
| **숙박 지구** | 여행자 전용. 여인숙부터 고급 호텔까지. 단기 임시 거주자가 도시 인구의 절반 |
| **행정 지구** | 연합 의회·각종 행정 기관 소재. 리버티 스퀘어 인접 |
| **정비 지구** | 마차 수리·말 교체·화물 재적재 전문. 운송 길드 직할 관리 |

## 정치·행정 구조

| 기관 | 역할 |
|---|---|
| **자유도시연합 의회** | 연합 전체 입법 기관. 각 도시 대표단 파견. 크로스로드 소재 |
| **운송 길드** | 이사벨 카르도 총괄. 대륙 물류 노선 독점 관리 |
| **크로스로드 시정부** | 도시 행정 담당. 실권은 운송 길드가 쥠 |

**물류 패권**: 이사벨 카르도의 말 한마디면 대륙 전체 물류가 마비된다. 어떤 왕이든 장군이든 보급선이 끊기면 무력화된다는 걸 모두가 안다.

## 신앙·문화

- **이동의 신 메르쿠르** 신앙. 여행자 출발 전 0번 이정표 앞 기도가 전통
- 크로스로드는 어느 세력에도 치우치지 않는다는 원칙 — 리버티 스퀘어 독립 선언에서 비롯
- 수많은 노선이 오가는 만큼 대륙 각지 문화가 혼재. 명실상부 다문화 도시

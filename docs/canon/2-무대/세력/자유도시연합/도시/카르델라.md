# 카르델라 (Cardella) — 돈으로 해결 안 되는 건 더 많은 돈으로 해결된다

> *"협상 테이블에 앉지 못한다면, 당신은 이미 메뉴판에 올라가 있는 겁니다."*
> — 마테오 블랙잭

[[자유도시연합|자유도시연합(Crossroad Cities)]]의 **비공식 해결사 도시**. 에테르 대륙 서부 천연 협곡 안에 자리 잡은 이 도시는 도박·유흥·정보 거래·청부 암살까지 모든 것이 거래되는 그레이 마켓의 중심지다. 자연 요새 지형 덕에 외부 세력이 함부로 침입하지 못하며, 그림자 의회가 실질적 권력을 행사한다.

---

## 기본 정보

| 항목 | 내용 |
|---|---|
| **위치** | 에테르 대륙 서부 협곡, 천연 요새 지형 |
| **역할** | 도박·유흥·정보 거래소, 비공식 해결사 중개 도시 |
| **통치자** | **마테오 블랙잭** (블랙 수트 조직 수장) [C: 원본 표기 마르코 블랙잭 → 마테오 블랙잭으로 정본 확정] |
| **이미지** | 화려한 카지노 불빛, 운하에 떠다니는 시체, 협곡 벽에 새겨진 조직 문양 |

**핵심 시설**: 로얄 카지노 · 중재의 탑 · 검은 시장 · 머시너리 게이트 투기장 연결 통로

---

## 도시 구조

### 상징적 지형

| 지형 | 설명 |
|---|---|
| **카지노 거리** | 로얄 카지노를 필두로 크고 작은 도박장이 늘어선 메인 스트리트. 밤새 불이 꺼지지 않음 |
| **침묵의 운하** | 도시를 가로지르는 수로. 시체 유기 장소로 악명 높음. 수면이 유독 고요함 |
| **지하 미궁** | 협곡 암반 아래 파인 광활한 지하 공간. 조직별 영역 표시로 가득함 |

### 주요 지구

| 지구 | 특징 |
|---|---|
| **상층부 (VIP 구역)** | 마테오 블랙잭 직할 관리. 상류층 의뢰인·조직 간부 전용 |
| **중층부 (일반 구역)** | 카지노·정보 거래·청부 의뢰 중개. 일반 방문객 주요 활동 구역 |
| **하층부 (슬럼)** | 실패한 도박꾼·조직 말단·도주자들의 은거지. 그림자 의회 말단 지배 |

## 정치·행정 구조

| 기관 | 역할 |
|---|---|
| **그림자 의회** | 마테오 블랙잭 + 5명 간부 보스. 도시 실질 통치 |
| **중재의 탑** | 조직 간 분쟁 중재 전담. 결정은 최종이며 불복 시 즉시 제거 |
| **블랙 수트 조직** | 마테오가 이끄는 최대 범죄 조직. 카지노·정보망 장악 |

**중립 지대 원칙**: 상점과 카지노 내부에서는 어떤 조직이든 전투 금지. 위반자는 그림자 의회 전원의 공적이 된다. 이 규칙 덕에 카르델라는 적대적 조직끼리도 같은 테이블에 앉을 수 있는 유일한 장소가 됐다.

|---|---|

## 신앙·문화

- 공식 신앙 없음. 돈과 평판이 신앙의 대체재
- 도박은 문화이자 계약이자 전쟁의 연장선
- 카지노 안에서는 귀족도 왕도 그냥 손님일 뿐 — 패를 잘못 쥐면 누구든 빈털터리가 됨

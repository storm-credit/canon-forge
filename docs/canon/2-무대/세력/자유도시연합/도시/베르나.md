# 베르나 (Berna)

> *"베르나 시장의 밤은 낮보다 정직하다."* — 문체 시료 (에반)

주인공 에반이 8년간 감정사로 살아온 자유도시연합의 도시. Act 1 1장(1~100화)의 주 무대.
이 문서는 **원고가 확정한 공간 사실만** 등재한다 (발명 0 — 각 항목에 화번호).

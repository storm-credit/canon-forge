# 숨은 거래단 (Hidden Exchange Guild)

> *"진정한 권력과 부는 결코 백일하에 얼굴을 드러내지 않는다. 우리의 장부 안에서, 너의 죄조차 영원한 비밀이 될 것이다."*

**성격**: 대륙 최악의 범죄 조직들조차 두려워하면서도 의존하는 **지하 중립 중개 브로커**. 스스로 범죄를 저지르는 대신 불법 유물·금지 마법서·정치 기밀 문서가 익명으로 거래되도록 판을 깐다. 계약 위반 시 왕도 예외 없이 피의 보복을 받는다.

**거점**: 이동형 거점 (실버포트 지하 미궁 내 특정 거래일 한정 개장) · 거울방 무한 성채(이공간 본거지)

**핵심 역량**:
- 금지 마법서·타락 성물·왕가 비밀 문서 익명 경매 주관
- 국가 최상급 수배범 신원 세탁 (생체 정보 위조 포함)
- 적대 세력 간 은밀한 휴전·거래 중재 (철갑 보안 거점 제공)
- 약조 위반자 신체 절단 후 거래처 택배 발송 ('경고 문화')

**핵심 인물**:

| 인물 | 별명 | 역할 |
|---|---|---|
| **벤투라 모레티** (L급) | 가면의 주인 | 총수. 진짜 얼굴·성별·나이 불명. 항상 다른 철가면 착용 |
| **엘리자베스 로슈포르** (A급) | 계약의 서기 | 모든 거래 인장 담당. 위반자 처형 조율 |
| **리카르도 살바토레** | 거래의 감시자 | 중재 타격대장. 거래 파기 시 즉시 분절 처형 |
| **에블린 더모트** | 거짓 없는 자 | 신원 세탁 전문가. 눈동자 색·마방진 파장까지 위조 |

**대외 관계**: 마법협회 금지 서고단(희귀 마법서 중개) · 왕국연합 반체제 귀족(안전 거래 공간 제공) · 해적 연합(장물 유통) · 성국 이단 사제(망명 피신처 제공)

# 0-1. 발키리아 전장의 맹세 용병단 군사 및 전술 구조

## 1. 개요
해당 세력은 전장의 규율과 피의 맹세를 숭배하는 방진 전술. 죽음 앞에서도 절대 전열이 무너지지 않는 수비와 돌격.을 기본 교리로 채택하고 있다.

## 2. 서사적 디테일 (Epic Detail)
아스트라리스 크로니클의 무자비한 생태계 속에서, 이들의 존재는 단순히 세력을 넘어 **세계관의 운명(Providence)**을 뒤흔드는 무거운 축으로 기능한다.

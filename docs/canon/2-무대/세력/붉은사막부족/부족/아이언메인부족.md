# 아이언메인 부족

> *"철갈기가 달리면, 모래도 길을 비킨다."*

---

## 1. 부족 개요

| 항목 | 내용 |
|------|------|
| **부족명** | 아이언메인 부족 (Ironmane Tribe) |
| **지위** | 붉은 사막 부족 연합 최강 전투 부족 |
| **본거지** | 사막 기병 훈련장 |
| **핵심 자산** | 스칼렛 불꽃단, 사막 독수리단, 사막 기마대 |
| **정치 노선** | 무력 우선 · 정면 전투 · 정복 확장 |

> [!important] 사막의 창
> 카르닥 아이언메인(SS급)은 잔드리크 다음으로 강한 전사이며,
> 그의 기병대는 사막에서 **무적의 기동력**을 자랑한다.
> 잔드리크 이후의 가장 유력한 후계 후보.

## 4. 부족 내부 갈등

- **카르닥의 야심**: 잔드리크 이후 대부족장이 되려는 은밀한 욕망
- **칼리드의 잔인함**: 약탈 시 불필요한 학살 — 부족의 평판 하락
- **네미카의 독립심**: 공중에서 홀로 판단 — 지상 명령을 무시하는 경향
- **전투 중독**: 평화 시에도 전쟁을 원하는 성향 — 외교적 해결 거부

# 06. 신성 방패단 (Divine Shield Legion) — 국경 방어·수성전

| 항목 | 내용 |
|---|---|
| 본부 | 아스트라르 (Astrar) — 태양의 성채 |
| 규모 | 약 7,000명 (국경 지대 분산 배치) |
| 상징 | 거대한 탑과 방패 (Tower & Shield) |
| 단장 | 방패군주 에단 크레스트윈드 (서부 전선 사령관, 축성술·수성전의 대가) |

> ⚠️ [불일치 주의] 성국.md §I 지도부 목록에서 에단 크레스트윈드는 "성녀 호위기사"로 표기. 방패단 방패군주와 동일 인물인지 별개인지 확인 필요. → _전수분석.md [C] 누락 대장 등재.

**4대 핵심 부대**: 성벽 수호단(성벽 방어) · 성전사 방패단(야전 방진, 적 기병 저지) · 신성 궁병대(곡사 지원) · 신성 마법 지원단(성벽 수리·광역 방어막)

**특수 능력**: 빛의 방벽(부대 단위 거대 에너지 방벽, 공성 병기도 막음) · 결계 유지(파손 성벽·성문 긴급 마법 복구) · 방패진 전술(개인 방패 겹쳐 철의 벽) · 태양의 요새(거점 점령 시 방어력 200% 증폭)

**비밀 — 솔라리스의 방패 심장**: 태양의 성채 지하에 "무한의 마력 원자로" 역할을 하는 고대 유물. 작동 중 요새 결계가 영구 유지되나 과부하 시 폭발 위험.

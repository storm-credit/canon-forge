# 03. 정화단 (Inquisitors of Purity) — 이단 심문·처형

| 항목 | 내용 |
|---|---|
| 본부 | 옵시디안 (Obsidian) — 정화의 대성전 (지하 고문실·감옥) |
| 규모 | 비공개 (약 1,000명 추정 + 수많은 정보원) |
| 상징 | 불타는 눈 (Burning Eye) |
| 단장 | 대심문관 발타자르 드루건 (Baltazar Drugen) — 성왕조차 함부로 대하지 못함 |

**4대 핵심 부대**: 불꽃의 심판단(공개 처형·무력 진압) · 성스러운 심문관(심문·고문) · 빛의 재판관(신성 법정) · 정화의 검단(암살·비밀 공작 "그림자 속의 빛")

**특수 능력**: 이단 감지(타락 마나·금지 마법·악의 레이더) · 진실의 불꽃(거짓말 시 혀가 타들어가는 고통) · 신성 화형(일반 물로 꺼지지 않는 불꽃) · 마법 차단(주변 마나 교란)

**원칙**: "한 명의 이단을 놓치느니, 열 명의 무고한 자를 의심하는 게 낫다."

**비밀 — 심연의 이용**: 상층부가 "이단을 잡기 위해 이단의 힘을 연구해야 한다"는 명목으로 압수한 흑마법 서적·유물을 비밀리에 연구 중.

> 단장 발타자르 드루건은 [[드루건|드루건 가문]] 당주이자 [[신성-거래회|신성 거래회]] 수장 — 이단을 처벌하는 자가 이단적 성물 밀매를 직접 운영하는 구조적 모순의 정점.

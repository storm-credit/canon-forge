# 01. 신성기사단 (Holy Knights) — 주력 전투

| 항목 | 내용 |
|---|---|
| 본부 | 루멘 (Lumen) — 성기사단 본부 채플 |
| 규모 | 정예 기사 약 5,000명 (종자·지원 제외) |
| 상징 | 태양을 휘감은 검 (Sun-wrapped Sword) |
| 단장 | 알렉산드로 발렌티안 (Grand Master, 성왕 직접 임명) |

**4대 핵심 부대**: 태양의 검단(최전방 공격) · 성스러운 방패단(방어) · 언데드 사냥단(대 언데드 특수전) · 태양의 심판단(내부 감찰·이단 척결)

**특수 능력**: 태양의 축복(신성 마법 1.5배) · 신성의 결계(3인 이상 집합 시 방어막) · 빛의 심판(언데드/악마에 절대 피해) · 불사의 저항(즉사·저주 고내성)

**입단 조건 "태양의 시험"**: 귀족 원칙(신성력 극히 높은 평민은 특별 전형). 10년 수련+신학 시험+이단심문관 사상 검증. 솔라리스 태양의 성전에서 3일 밤낮 기도 환영과 싸워 이겨야 함.

**내부 갈등**: 전통파(순수 혈통·신앙 중시) vs 실력파(평민 등용·효율 중시) 알력 심화.

**비밀 — 타락한 태양**: 일부 고위 기사들이 더 강한 힘을 위해 압수한 마도제국 유물을 몰래 사용. "독으로 독을 제압한다" 합리화.

| 봉인 유물 | 효과·위험 |
|---|---|
| 마도제국의 유물들 (압수품) | 일부 고위 기사 사용. 들키면 이단 처단 |

# 성기사단 상회 (Holy Order Trading Guild)

> *"신앙을 수호한다는 명분 아래, 점령지의 광맥과 이단의 재산이 곧 우리의 자금줄이다."*

| 항목 | 내용 |
|---|---|
| **실질 후원** | [[발렌티안]] 가문 직속 (원정 보급·무기 조달 독점) |
| **성격** | 교황청 비호 아래 무력·무기 밀매를 독점하는 **군산복합체** |
| **자금원** | 점령지 금속 광산 무단 점거 + 이단 심문 압수 재산 |
| **주요 거래처** | 왕국연합 (무력 지원을 담보로 막대한 전쟁 물자 착취) |

**주요 품목**: 축복받은 처형검 · 신성 인장이 각인된 고문구 · 대이단 파괴용 강압 포션. 최상급 성구는 뇌물을 바친 대귀족에게만 은밀히 납품한다.

**구조**: 신성기사단의 원정 보급선과 직결돼 있어, 상회의 이익이 곧 발렌티안 가문의 군사 동원력으로 환원된다. 신앙 수호라는 명분이 자원 강탈과 무기 거래를 정당화하는 장치로 기능한다.

> **[탐험 후크] 폐쇄된 무기 제련소** — 신성 기사단의 폐쇄된 제련소에서는 신의 축복이 아니라 생포된 이교도의 피를 끓여 무기에 강도를 더한다는 참혹한 소문이 돈다.

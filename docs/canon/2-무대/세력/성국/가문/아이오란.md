# 아이오란 교황가 (House Aioran)

**계보 정체성**: 교황청 전결권·성물 인증권·신성 마법 허가권을 독점하는 종교 권력 혈통. 성왕이 세속 권력이라면, 아이오란 교황가는 신학적 정당성 자체를 공급하는 구조.

---

## 핵심 인물

**베네딕투스 아이오란 (Benedictus Aioran)** — 교황 / 황금의 사기꾼 / SS급

- 현역 교황. '황금의 사기꾼'이라는 이명은 적들이 붙인 것 — 본인은 '신의 현명한 대리자'라 자칭.
- 교황청 공식 문서 발행권·성기사 서임권·이단 판결 최종 승인권 보유.
- 신성 거래회 수장 발타자르 대주교와 연결고리 의심. 성혈단을 비공인 암살 도구로 활용한다는 의혹.

**엘레오노르 실버하트 (Eleonore Silverheart)** — 성녀 / 갇힌 성녀 / S급

- 고아 출신. 기적 판정으로 '성녀' 서열에 올랐으나 실제로는 교황청의 프로파간다 도구로 이용됨.
- 태양신전 내부에서 사실상 연금 상태. 외부 접촉이 철저히 통제됨.
- 치유 계열 신성 마법 비범한 자질 보유 — 이것이 교황청이 그녀를 놓아주지 않는 근거.

**카시엘 루미나르 (Cassiel Luminare)** — 제3기사단장 / 그림자 기사 / S급

- 고아 출신, 루미나 가문 성씨를 성씨로 하사받음(혈통 아님). '그림자 기사'라는 이명처럼 공식 기록에 거의 등장하지 않음.
- 제3기사단(정보·척후 특화) 지휘. 교황청과 발렌티안 가문 사이의 정보 이중 플레이 가능성 있음.

**실라스 블랙쏜 (Silas Blackthorn)** — 고문관 / 미치광이 사신 / S급

- 아이오란 가문 직속 비선 고문. '미치광이'라는 이명은 예측 불가능한 조언과 행동 때문.
- 예언술 경계선상의 마법 사용자로 의심받으나 공식 이단 혐의 없음 — 교황가의 비호 때문.

**세레나 루미나스 (Serena Luminace)** — Act 1 히로인

- 아이오란 가문 방계 출신. 외부에서는 평범한 하급 성직자로 알려짐.
- Act 1 서사에서 에반과 접촉, 성국 내부 모순을 처음으로 목격자 관점에서 제공하는 인물.

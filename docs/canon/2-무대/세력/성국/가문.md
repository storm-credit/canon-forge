# 성국 6대 가문 (Saint Haven Noble Houses)

> 성국의 권력은 신의 이름으로 분점된다. 6개 가문은 동맹도, 적도 아닌 — 서로를 필요로 하면서 서로를 감시하는 구조로 존재한다.

[[성국|성국(Saint Haven)]] 지배 구조를 형성하는 여섯 혈통 및 준가문 세력. 성왕과 교황이 각각 세속권과 종교권을 분점하지만, 실질적 통치는 이 6개 집단의 권력 균형 위에서 작동한다.

## I. 산크투스 왕가 (House Sanctus) — 신정 왕권

**계보 정체성**: 태양신 솔라리스와의 성검 계약을 통해 왕위를 유지하는 신화적 혈통. 성검의 계약이 유효한 한 왕위 찬탈은 신성모독.

### 핵심 인물

**레오니스 산크투스 (Leonis Sanctus)** — 성왕 / 태양의 대리인 / SS급

- 현역 성왕. '태양의 대리인'이라는 칭호는 단순 경칭이 아닌 신학적 직위 — 성검 계약의 살아있는 당사자.
- 외교적으로는 교황 베네딕투스와 공동 통치를 표방하지만, 군사 동원령·전쟁 선포권은 성왕 단독 권한.
- 베네딕투스가 종교 입법권으로 성왕의 세속권을 조금씩 잠식하고 있음을 인지하고 있으나, 정면 충돌 대신 발렌티안 가문을 방패로 활용하는 방식을 택함.

**아타나시우스 산크투스 (Athanasius Sanctus)** — 제1왕자 / 방랑하는 왕자 / S급

- 왕위 계승 1순위. 그러나 지속적으로 계승 거부 신호를 보내며 수도 외곽·국경지대를 독자 유랑.
- 왕위보다 전선 현장 경험을 선호. 10대 기사단 중 어느 소속도 아닌 독립적 전투 기록 보유.
- **위기**: 아타나시우스 거부가 공식화될 경우, 산크투스 왕가의 직계 혈통 단절로 성검 계약 재갱신 불가 → 국가 근본 신정 체계 붕괴 위험.

**레온 솔라리스 (Leon Solaris)** — 제2왕자 / 솔라 가르드 사령관

- 실용주의자. 솔라 가르드 직할 지휘를 통해 북부 방어선을 실질 장악.
- 아타나시우스 거부 시 유력한 대안 후계자. 단, '솔라리스'라는 성씨는 하사받은 것이며 순수 산크투스 혈통이 아님 — 교황가가 이를 근거로 계승권 이의 제기 가능.

### 산크투스 가문 구조도

```
레오니스 성왕 (현재)
├── 아타나시우스 제1왕자 (계승 거부 위험)
└── 레온 솔라리스 제2왕자 (솔라 가르드 사령관)
```

## II. 아이오란 교황가 (House Aioran) — 종교 통제

**계보 정체성**: 교황청 전결권·성물 인증권·신성 마법 허가권을 독점하는 종교 권력 혈통. 성왕이 세속 권력이라면, 아이오란 교황가는 신학적 정당성 자체를 공급하는 구조.

### 핵심 인물

**베네딕투스 아이오란 (Benedictus Aioran)** — 교황 / 황금의 사기꾼 / SS급

- 현역 교황. '황금의 사기꾼'이라는 이명은 적들이 붙인 것 — 본인은 '신의 현명한 대리자'라 자칭.
- 교황청 공식 문서 발행권·성기사 서임권·이단 판결 최종 승인권 보유.
- 신성 거래회 수장 발타자르 대주교와 연결고리 의심. 성혈단을 비공인 암살 도구로 활용한다는 의혹.

**엘레오노르 실버하트 (Eleonore Silverheart)** — 성녀 / 갇힌 성녀 / S급

- 고아 출신. 기적 판정으로 '성녀' 서열에 올랐으나 실제로는 교황청의 프로파간다 도구로 이용됨.
- 태양신전 내부에서 사실상 연금 상태. 외부 접촉이 철저히 통제됨.
- 치유 계열 신성 마법 비범한 자질 보유 — 이것이 교황청이 그녀를 놓아주지 않는 근거.

**카시엘 루미나르 (Cassiel Luminare)** — 제3기사단장 / 그림자 기사 / S급

- 고아 출신, 루미나 가문 성씨를 성씨로 하사받음(혈통 아님). '그림자 기사'라는 이명처럼 공식 기록에 거의 등장하지 않음.
- 제3기사단(정보·척후 특화) 지휘. 교황청과 발렌티안 가문 사이의 정보 이중 플레이 가능성 있음.

**실라스 블랙쏜 (Silas Blackthorn)** — 고문관 / 미치광이 사신 / S급

- 아이오란 가문 직속 비선 고문. '미치광이'라는 이명은 예측 불가능한 조언과 행동 때문.
- 예언술 경계선상의 마법 사용자로 의심받으나 공식 이단 혐의 없음 — 교황가의 비호 때문.

**세레나 루미나스 (Serena Luminace)** — Act 1 히로인

- 아이오란 가문 방계 출신. 외부에서는 평범한 하급 성직자로 알려짐.
- Act 1 서사에서 에반과 접촉, 성국 내부 모순을 처음으로 목격자 관점에서 제공하는 인물.

### 아이오란 가문 구조도

```
베네딕투스 교황 (수장)
├── 엘레오노르 실버하트 (성녀, 연금 상태)
├── 카시엘 루미나르 (제3기사단장, 비혈통 하사)
├── 실라스 블랙쏜 (고문관)
└── 세레나 루미나스 (방계, Act 1 히로인)
```

## III. 발렌티안 가문 (House Valentian) — 무력 독점

**계보 정체성**: 10대 기사단 통솔권을 세습으로 보유하는 무력 가문. 성왕의 군사 동원령은 실질적으로 발렌티안 총단장의 집행 의지에 달려 있음.

### 핵심 인물

**알렉산드로 발렌티안 (Alexandro Valentian)** — 성기사단 총단장 / 불굴의 방패 / SS급

- 발렌티안 가문 현 당주이자 성기사단 총단장. 개인 전투력보다 전략적 지휘력이 더 높은 평가를 받음.
- 성왕과 교황 사이의 군사권 균형자. 어느 쪽도 알렉산드로를 완전히 통제하지 못함.
- 에단 크레스트윈드를 고아 출신임에도 발탁해 수호기사단장까지 올린 인물 — 혈통 외 실력 우선 원칙 고수.

**에단 크레스트윈드 (Ethan Crestwind)** — 수호기사단장 / 신성의 벽 사령관 / 침묵의 수호자 / S급

- 고아 출신. 발렌티안 가문 혈통이 아닌 유일한 핵심 군 지휘관.
- 실력만으로 기사단 서열을 올라 신성의 벽([[신성의-벽|Rampart of the Divine]]) 사령관 겸 수호기사단장 직위 획득.
- 철저한 과묵함으로 '침묵의 수호자' 이명. 정치적 발언 없음, 명령 집행만 존재.
- **내부 위기**: 혈통 기사단 파벌이 에단을 이단으로 몰아 제거하려는 음모가 수면 아래에서 진행 중. 성공 시 신성의 벽 방어 공백이 발생.

### 발렌티안 가문 구조도

```
알렉산드로 발렌티안 (총단장)
└── 에단 크레스트윈드 (수호기사단장, 비혈통 발탁)
```

## IV. 드루건 가문 (House Drugan) — 공포 통치

**계보 정체성**: 이단 심문·처형 권한을 세습하는 법·폭력 가문. 형식은 법 집행이지만 실질은 정적 제거 도구.

### 핵심 인물

**발타자르 드루건 (Balthazar Drugan)** — 이단심문관장 / 철의 대주교 / S급

- 공식 직위: 이단심문관장 + 대주교. 이 두 직위의 결합이 종교 권위와 처형 권력을 동시에 부여.
- **이중 구조**: 공식으로는 이단을 처벌하나, 동시에 신성 거래회(암약 조직) 수장으로 성물 밀매·고리대금을 직접 운영. 자신이 단속해야 할 범죄의 정점에 자신이 있음.
- 베네딕투스 교황과는 상호 이용 관계 — 교황이 목표물을 제공하면, 발타자르가 이단으로 처리.

### 드루건 가문 권력 기반

이단 심문관 네트워크 — 전국 30개 심문소 직할 관리. 심문관 개인은 영장 없이 체포 가능, 법원 개입 없이 구금 가능. 이 권한이 드루건 가문을 성국에서 가장 두려운 집단으로 만드는 구조.

## V. 몬탈티 파트너십 (Montalti Partnership) — 행정 실무

**계보 정체성**: 엄밀히는 단일 가문이 아닌 2개 가문의 실무 파트너십. 재무와 외교/정보를 각각 맡아 성국 행정의 실질적 집행을 담당.

### 핵심 인물

**세레스티아 다르민 (Celestia Darmin)** — 재무장관 / 강철의 행정관 / A급

- 다르민 가문 출신 재무장관. 성국 예산·면죄부 발행·교역로 관리를 총괄.
- '강철의 행정관'이라는 이명처럼 감정 없이 수치로만 국가를 운영한다고 알려짐.
- 신성 거래회의 재정 이중 장부를 일부 인지하고 있으나 공개하지 않음 — 개인 보존 논리.

**리시아 에스텔 (Lysia Estelle)** — 외교관 / 사교계의 여왕 / A급

- 에스텔 가문 출신 외교·정보 담당관. 귀족 사교계에서 수집한 정보를 성국 외교에 활용.
- '사교계의 여왕'이라는 이명처럼 대화를 통해 상대를 조종하는 능력이 뛰어남.
- 카시엘 루미나르(제3기사단, 정보 특화)와 업무 중첩 — 이들 사이의 정보 권한 경쟁이 잠재.

## VI. 푸게토리움 (Purgatorium) — 분리 암약

**계보 정체성**: '가문'이 아닌 비밀 결사 형태. 화형에서 생존한 자들이 모인 복수 집단으로, 성국 권력 구조 외부에서 독자 행동.

### 핵심 인물

**말라카이 애쉬 (Malakai Ash)** — 푸게토리움 수장 / A급

- 화형 생존 전설. 성국의 화형 집행이 실패한 유일한 공식 기록 보유자.
- 전신에 화상 흉터. 성국 내 잿빛 천 두름 비공식 네트워크 지도자.
- 신성 화형 집행단과 이면 계약 관계 — 특정 표적 처리를 비공인 의뢰로 받음. 동시에 자신이 성국에 복수를 준비 중.
- 성혈단·성스러운 불꽃과 교차 점촉 없이 독자 행동을 원칙으로 함.

### 푸게토리움의 특수성

일반적인 암약 조직과 달리 조직의 목표가 '성국 내부 장악'이 아닌 '성국 시스템 외부의 생존과 복수'. 구성원 모두가 화형·고문·추방을 경험한 생존자로 구성 — 이것이 조직의 결속과 동시에 불안정성의 원천.

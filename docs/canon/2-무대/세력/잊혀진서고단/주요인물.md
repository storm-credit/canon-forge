# 15-2. 시대별 전설 마도 영웅록

> *각 마법 학파의 극한에 닿았으나, 잔혹한 섭리(Grim Plausibility)에 먹혀버린 이들의 기록*

## 📚 아르젠 (Argen) - 진리의 파수꾼
- **이명**: 서고단 최상위 염동력 석학
- **관련 마법**: 그랜드 매스 텔레파시, 그랜드 에이지스 오브 소울 (정신 9클래스)
- **🩸 가혹한 대가 (Grim Plausibility)**: 수만 군대의 신경망을 하나로 엮어 일사불란한 전술을 내렸으나, 그 군대의 병사 하나가 잘려 나갈 때마다 절단의 환상통과 죽음의 공포 수만 배가 지휘관 뇌리에 100% 직수입되어 자살 충동에 미쳐간다.

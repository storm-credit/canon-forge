# 영원의 성채

> *"시간도 이곳에서는 얼어붙는다."*

| 항목 | 내용 |
|------|------|
| **위치** | 프로스트 대륙 최북단, 빙하 중심부 |
| **역할** | 빙하의 신전 수도, 얼음 [[직업체계|마법사]] 거주지 |
| **분류** | 수도 (Capital) |

---

## 개요

강력한 얼음 [[직업체계|마법사]]들이 거주하는 빙하의 신전 수도. 영구적인 마법 보호막이 도시 전체를 감싸고 있으며, 내부는 극한의 추위에도 불구하고 마법으로 온도가 조절된다.

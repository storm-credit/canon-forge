# 얼어붙은 지혜의 도서관

> *"이곳의 지식은 영원히 얼어붙어 보존된다."*

| 항목 | 내용 |
|------|------|
| **위치** | 프로스트 대륙 최북단, 영원의 성채 서쪽 |
| **역할** | 수천 년 된 얼음 마법 서적 보관소 |
| **분류** | 서고 (Library) |

---

## 개요

수천 년 동안 축적된 얼음 마법 지식이 보관된 거대 도서관. 서적들은 얼음 마법으로 영구 보존되어 있으며, [[직업체계|마법사]]들만 접근이 허용된다.

---

## 구조

### 주요 구역
| 구역 | 특징 |
|------|------|
| **일반 열람실** | 기초 마법서 열람 |
| **고급 서고** | 중급 마법 문헌 |
| **금서고** | 금지된 마법 기록 |
| **고대 기록실** | 수천 년 된 원전 |
| **복사실** | 마법 복제 시설 |

### 보존 마법
- 영구 동결 보존
- 자동 복구 마법
- 무단 열람 방지

---

## 에피소드 훅

1. **금서 탈취** - 금지된 마법서 탈취 시도
2. **해독 의뢰** - 고대 문헌 해독 요청
3. **서고 침입** - 외부 세력의 잠입
4. **잃어버린 문서** - 사라진 중요 문헌 추적

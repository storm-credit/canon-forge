# 프로스트마이어 가문

> *"정의는 얼음처럼 단단해야 한다. 녹으면, 모든 것이 흘러내린다."*

---

## 1. 가문 개요

| 항목 | 내용 |
|------|------|
| **가문명** | 프로스트마이어 가문 (House Frostmire) |
| **지위** | 빙하의 신전 심문관장 · 균열 통제부 총괄 |
| **본거지** | 영겁의 서가, 심연빙 균열대, 잠든 군단의 감옥 |
| **핵심 자산** | 아이스저스티스 심판관, 퍼마소울 검객단, 홀리프리즈 방패단 |
| **정치 노선** | 이단 정화 · 균열 봉인 · 내부 감찰 |

> [!warning] 두려움의 가문
> 네리안 프로스트마이어는 "차디찬 심판"이라 불린다.
> 이단이 아닌 자도 **네리안의 이름만 들으면 얼어붙는다** — 두려움이 곧 권력.

## 4. 가문 내부 갈등

- **네리안의 광기**: 이단 심문이 점점 기준이 모호해짐 — 동맹도 적도 구별 불가
- **라이언의 비밀**: 심연빙 균열대에서 Flux 실험 — 봉인인가, 이용인가
- **발렌의 딜레마**: 잠든 군단의 감옥이 포화 — 새 봉인체를 받을 공간 부족
- **다른 가문의 견제**: 윈터본+글레시아가 프로스트마이어의 권한 축소 시도

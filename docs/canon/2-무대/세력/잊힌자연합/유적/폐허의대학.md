# 폐허의 대학

> *"폐허 속에서도 지식은 살아 숨 쉰다."*

| 항목 | 내용 |
|------|------|
| **위치** | 오벨리스크 대륙 북부, 고대 도시 폐허 내 |
| **역할** | 과거 마법 학문의 중심지, 현재 연구 거점 |
| **분류** | 고대 유적 (Ancient Ruins) |

---

## 개요

마도제국 멸망 이후 일부 학자들이 연구 거점으로 활용하는 -1. 고대 마법 대학의 폐허. 금지된 마법과 고대 지식이 남아 있다.

### 잔존 시설
| 시설 | 상태 |
|------|------|
| **대강당** | 부분 붕괴, 사용 중 |
| **도서관** | 대부분 보존 |
| **실험실** | 일부 작동 |
| **기숙사** | 거주 가능 |
| **지하 연구소** | 봉인됨 |

## 에피소드 훅

1. **학자 구출** - 실종된 연구자 수색
2. **고대 강의** - 봉인된 강의 기록 재현
3. **지하 봉인** - 지하 연구소 봉인 해제 시도
4. **마법 오작동** - 고대 실험 장치 오작동

# 블랙웨이 협곡 요새 (Blackway Forts) — 협곡이 곧 성벽이다

> *"빛이 들지 않는 곳에서 가장 더러운 거래가 이루어진다."*

[[왕국연합|왕국연합(Allied Kingdoms)]] 서남부 암벽 협곡 지대에 구축된 **절벽 반동굴형 복합 요새**. 자연 협곡을 그대로 성벽으로 활용하며, 협곡을 통과하는 유일한 전략 직행로를 통제한다. 공식적으로는 왕국연합 서남부 방어선이지만, 밀수·정보 거래가 암암리에 이루어지는 **그늘의 관문**으로도 악명 높다.

---

## 기본 정보

| 항목 | 내용 |
|---|---|
| **위치** | 에테르 대륙 서남부, 암벽 사이 좁은 협곡 지대 |
| **역할** | 전략적 요충 직행로 통제 + 서남부 방어선 |
| **별명** | "그늘의 관문" / "협곡이 곧 성벽" |
| **이미지** | 수직으로 솟은 암벽 양쪽에 요새 구조물이 박혀 있음. 협곡 바닥은 그늘져 낮에도 어두움. 쌍둥이 검문소가 통행 차단 |

**핵심 시설**: 쌍둥이 검문소(Twin Checkpoints) · 절벽 포대 · 낙석 장치(Rockfall Mechanism) · 지하 암굴 감옥 · 밀수꾼 주점 (비공식)

---

## 지형 및 요새 구조

### 시각적 특징

- **협곡 자연 성벽**: 높이 50~80m 수직 암벽이 양측에 솟아 있어, 정면 통과 외 다른 이동 경로가 원천 차단
- **쌍둥이 검문소**: 협곡 입구와 출구 양쪽에 각각 설치. 동시 봉쇄로 협곡 내부를 함정화 가능
- **절벽 포대**: 양쪽 암벽 상단에 설치된 투석기·마법 포 진지. 협곡 내 이동 중인 적에게 일방적 타격
- **낙석 장치**: 암벽 상단에 미리 쌓아둔 거대 바위를 신호와 함께 투하하는 방어 기믹. 대형 부대 전멸 가능

### 주요 구역

| 구역 | 특징 |
|---|---|
| **쌍둥이 검문소** | 통행 허가·세금 징수·신원 확인 |
| **절벽 포대** | 암벽 상단 공격 진지. 사다리 전용 접근로 |
| **낙석 장치 제어소** | 협곡 상단 비밀 위치. 소수 인원이 대부대 저지 가능 |
| **지하 암굴 감옥** | 포로·밀수꾼 수용. 습한 암굴 환경. 탈출 불가 구조 |
| **'그늘 주점'** | 비공식. 검문소 통과 대기자·밀수꾼·정보상 집결 장소 |

## 어두운 현실

**밀수 경로**
- 블랙웨이 협곡은 왕국연합-성국-자유도시 경계 삼각지대에 위치
- 수비대장의 묵인 하에 금지 마법 물품·불법 용병 계약이 정기적으로 통과
- 검은 매 정찰대가 알면서도 방관하는 이유: 밀수 루트가 역정보 유통에도 활용됨

**수비대장의 탐욕**
- 현 수비대장(원본 기재: "탐욕스러운 악당")은 뇌물 수수 + 세금 착복으로 악명 높음
- 연합 감찰관이 조사에 나섰다가 실종된 사례 있음

|---|---|---|

| **밀수꾼 리더** | 미정 | 협곡 비밀 루트 독점. 에반의 은밀한 이동 안내 가능 |

## 소환 · 에반 관련

**[은밀한 이동]**
- 에반이 공식 검문을 피해 서남부를 통과해야 하는 상황
- 수비대장과 협상 or 밀수꾼 리더의 비밀 루트 이용

**[조사 — 실종 감찰관]**
- 연합에서 에반에게 블랙웨이 감찰관 실종 조사 의뢰
- 수비대장의 범행 증거 수집 + 지하 암굴 감옥 생존자 구조

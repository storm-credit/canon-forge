# 웨스트게이트 (Westgate) — 서부의 마지막 관문

> *"이 문 너머엔 법이 없다. 우리가 법이다."*

[[왕국연합|왕국연합(Allied Kingdoms)]] 최서단 국경 요새. **야생의 땅(Wildlands)** 경계에 세워진 1차 방어선으로, 서부에서 밀려드는 몬스터·괴수·미지의 존재를 저지하는 임무를 맡는다. **웨스트 마치** 전진기지보다 후방에 위치하며, 연합 서부 방어 체계의 중추다.

## 지형 및 요새 구조

### 시각적 특징

- **철의 장벽 (The Iron Wall)**: 웨스트게이트의 핵심 방어 구조물. 높이 15m, 두께 5m의 철 보강 석조 성벽. 야생의 땅과 연합 영토를 분리하는 최후 선
- **그리핀 둥지**: 성벽 상단에 설치된 그리핀 기수 전대 격납 구역. 공중 정찰 및 타격 담당
- **용병 구역 'The Meat Grinder'**: 요새 외곽. 비공식 용병 집결·계약 구역. 고위험 야생 탐험 의뢰 거래

### 주요 구역

| 구역 | 특징 |
|---|---|
| **철의 장벽 (Iron Wall)** | 주 방어선. 야생 침입 1차 저지. 마법 강화 투석기 배치 |
| **연합 서부 사령부** | 서부 국경 병력 지휘소. 지도 작전실 |
| **그리핀 둥지** | 그리핀 기수 부대. 공중 정찰·타격 전담 |
| **용병 구역** | 자유 계약 용병 숙소·거래소. 비공식 정보 유통 |
| **보급 창고** | 서부 방어선 전체 보급 허브. 웨스트 마치까지 보급선 연결 |

## 핵심 방어 기믹

**[방어] 철의 장벽 봉쇄**
- 3중 철문 + 함정 지대 + 성벽 상단 포격 조합
- 야생의 땅에서 오는 대형 괴수 대응 전용 "충돌 킬존" 구역 설계

**[공중] 그리핀 기수 정찰**
- 하루 2회 서부 정찰 비행. 대형 괴수 이동 조기 감지
- 위급 시 공중 타격 + 보급로 투하 병행

**[특수] 용병 활용**
- 정규군의 위험 임무를 용병이 대신 수행
- "Meat Grinder" 별명의 이유: 용병 사망률이 연합 전체 최상위

## 소환 · 에반 관련

**[방어전 — 대형 괴수 침공]**
- 야생의 땅에서 평소와 다른 대형 괴수 무리가 방어선에 집중 돌격
- 에반이 정찰대와 협력하여 돌격 원인을 탐색 → 균열 발견

**[탐험 — 야생의 땅 진입]**
- 웨스트게이트를 거쳐야만 야생의 땅으로 진입 가능
- 총사령관의 통행 허가 또는 에르메스의 동행이 필요

# 아이언가르드 (Ironguard) — 철의 맹세는 깨지지 않는다

> *"강철은 녹슬지 않고, 철의 맹세는 깨지지 않는다."*

[[왕국연합|왕국연합(Allied Kingdoms)]] [[브룬발트]] 왕국 산하 동부 국경 도시. 에테르 대륙 동부 산악 지대에 위치하며 크림슨 대륙 접경 최전방을 지킨다. **왕국연합 최대 무기·갑옷 생산지**로, 아이언가르드의 철광과 대장간이 연합 전체 군수품의 핵심 공급원이다.

---

## 기본 정보

| 항목 | 내용 |
|---|---|
| **위치** | 에테르 대륙 동부 산악 지대, 크림슨 대륙 접경 |
| **역할** | 연합 최대 무기·갑옷 생산지 + 크림슨 방어 최전방 |
| **별명** | 외부: "철의 도시" / 내부: "아이언가르드가 무너지면 연합이 무너진다" |
| **이미지** | 산 위의 석조 요새 도시, 끊임없는 망치질 소리와 군가, 철광산 입구에서 뿜어 나오는 연기 |

**핵심 시설**: Iron Crown 왕성(산 정상) · 대장간 거리 · 관문 요새(크림슨 방향 3중 성문) · 철광 벨트(지하 광산 네트워크) · 병기 연구소

## 지형 및 도시 구조

### 상징적 지형

| 지형 | 설명 |
|---|---|
| **산악 요새 구조** | 자연 방벽을 활용한 다층 성채. 협곡이 천연 킬존 역할 |
| **철광 벨트 (Iron Belt)** | 도시 하부 거대 광산 네트워크. 대륙 최대 철광 매장량 |
| **협곡 관문** | 크림슨 대륙으로 통하는 유일한 육로. 3중 성문으로 봉쇄 가능 |

### 주요 지구

| 지구 | 특징 |
|---|---|
| **왕성 지구** | Iron Crown 왕성. 철왕 거처 + 군사 지휘소 |
| **대장간 거리** | 왕국 최고 무기·갑옷 제작 장인 밀집. "아이언가르드 품질" 인증 |
| **병영 지구** | 상시 주둔 대군. 징병 훈련 시설 |
| **광산 지구** | 지하 채굴 도시. 광부 가문 세대 거주 |
| **관문 요새** | 크림슨 방향 방어선 핵심. 총독 직할 |

## 군사 & 생산력

| 조직·시설 | 역할 |
|---|---|
| **철의 기사단** | 중갑 기병 정예. 연합 최강 육군 핵심 |
| **산악 보병** | 험지 전투 특화. 크림슨 침공 방어 전선 |
| **공성 병기 공장** | 발리스타, 투석기 대량 생산 + 공급 |
| **대장간 길드** | 연합 전체 군수품(검·갑옷) 공급 핵심 |

**연합 내 역할**
- 크림슨 침공 방어 최전선
- 연합 군사 회의 주도권 영향력 행사
- 다른 왕국에 무기·갑옷 공급 (경제적 의존 관계 형성)

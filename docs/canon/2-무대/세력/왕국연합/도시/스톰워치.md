# 스톰워치 (Stormwatch) — 폭풍이 와도 우리는 남는다

> *"폭풍이 와도 우리는 지켜본다. 폭풍이 가도 우리는 남는다."*

[[왕국연합|왕국연합(Allied Kingdoms)]] [[브룬발트]] 왕국 산하 해안 도시. **실버랜스(Silverlance) 가문**의 거점이자 연합 서부 군사 지휘 중심지. 왕국연합 유일의 대규모 해군을 보유하며, 해양 대륙과의 교류 창구 역할도 겸한다.

---

## 기본 정보

| 항목 | 내용 |
|---|---|
| **위치** | 에테르 대륙 서해안, 대양 접경 절벽 지대 |
| **역할** | 실버랜스 가문 거점 + 서부 군사 지휘 + 연합 해군 본부 |
| **별명** | 외부: "폭풍 왕국" / 내부: "바다가 곧 도로다" |
| **이미지** | 거친 파도와 절벽 위의 등대, 항구에 빽빽한 전함들, 소금 바람 냄새가 나는 해안 절벽 도시 |

**핵심 시설**: 폭풍 왕궁(Storm Palace, 절벽 끝 성채) · 제독 항(제독 항구, 해군 본부) · 등대 탑(해안 최고 높이, 관측소 겸용) · 해병대 본부 · 조선소 구역

## 지형 및 도시 구조

### 상징적 지형

| 지형 | 설명 |
|---|---|
| **절벽 성채** | 폭풍 왕궁이 서 있는 자연 요충지. 아래로 항구, 위로 감시탑 |
| **다중 항구** | 군항(군함 전용)·상항(무역선)·어항 분리 운영 |
| **등대 탑 (Stormwatch Tower)** | 대륙 최고 높이 등대. 낮에는 관측소, 밤에는 항해 지원 |
| **해안 포대** | 항구 입구 양쪽 절벽에 배치된 대형 함포 진지 |

### 주요 지구

| 지구 | 특징 |
|---|---|
| **왕궁 지구** | 폭풍 왕궁, 실버랜스 가문 저택, 해군 사령부 |
| **제독 항** | 군함 정박, 해병대, 해군 행정 |
| **상인 항** | 무역선, 조선소, 해상 교역 사무소 |
| **어부 마을** | 해안가 서민 거주. 선원·어부 가문 대대로 거주 |

## 군사 & 해군력

| 조직 | 규모 / 역할 |
|---|---|
| **폭풍 함대 (Storm Fleet)** | 전열함 100척 이상. 왕국연합 유일 대규모 해군 |
| **해병대** | 상륙·해상 전투 특화 정예 부대 |
| **해안 포대** | 항구 방어 대포 진지 |
| **제3기사단 (교체 주둔지)** | 칼릭 지휘. 스톰워치 방어 지원 순환 주둔 |

> "바다가 곧 도로다. 우리가 곧 법이다."
> — 폭풍 함대 격언

**해양 지배 역할**
- 연합 해상 무역로 보호
- 해적 소탕 책임
- 해양 대륙과의 유일 정기 교류 창구

## 실버랜스 가문

왕국연합 가문 아카이브([[가문]]) 참조. 스톰워치를 세습 지배하는 귀족 해군 가문. 대대로 폭풍왕 칭호를 이어받으며 해군을 직접 통솔한다.

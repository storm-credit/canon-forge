# 노블가르드 (Noblegard) — 카멜로스의 왕좌

> *"명예는 검 끝에서 나오고, 질서는 왕좌에서 시작된다."*

[[왕국연합|왕국연합(Allied Kingdoms)]] [[브룬발트]] 왕국 산하 도시. **카멜로스(Camellos) 왕가**의 본거지이자 왕실 마상 토너먼트 개최 도시다. 브룬발트의 군사적 강경 이미지와 달리, 노블가르드는 기사도 문화와 귀족 사교의 화려한 외관을 유지하는 브룬발트의 "문화 수도" 역할을 한다.

---

## 기본 정보

| 항목 | 내용 |
|---|---|
| **위치** | 에테르 대륙 북서부, 브룬발트 왕국 영토 내 |
| **역할** | 카멜로스 왕가 본거지 + 왕실 마상 토너먼트 개최 |
| **별명** | 외부: "카멜로스의 왕좌" / 내부: "진짜 권력은 검보다 초대장에서 나온다" |
| **이미지** | 하얀 대리석 성벽과 붉은 지붕군, 왕국 연합 깃발들이 휘날리는 고전적 요새 도시. 대형 토너먼트장의 함성이 멀리까지 들림 |

**핵심 시설**: 카멜로스 왕궁(Camellos Palace) · 왕실 대토너먼트장(Royal Arena) · 기사 아카데미 · 귀족 사교관 · 왕실 마구간

## 지형 및 도시 구조

### 상징적 지형

| 지형 | 설명 |
|---|---|
| **카멜로스 언덕 (Camellos Hill)** | 도시 중심. 카멜로스 왕궁 + 귀족 저택군. 일반 시민 출입 엄격 제한 |
| **왕실 대로 (Royal Boulevard)** | 토너먼트장과 왕궁을 잇는 직선 대로. 행렬·의전 전용 |
| **기사의 거리 (Knight's Row)** | 기사 아카데미, 무기상, 갑옷 공방, 결투장이 늘어선 구역 |
| **귀족 사교 지구 (Noble Quarter)** | 연회관, 극장, 고급 주점. 혼인·동맹·밀약이 이 구역에서 결정됨 |

## 왕실 토너먼트

> "브룬발트에서 가장 화려한 날은 전쟁이 아니라 토너먼트다."

- **왕실 대토너먼트 (Royal Grand Tournament)**: 연 1회 개최. 5대 왕국 기사단 대표 + 초청 외국 기사 참여
- **카멜로스 배 (Camellos Cup)**: 왕가 직접 수여 트로피. 우승자는 왕실 기사단장 자격 자동 부여
- **관전 외교**: 타 왕국 귀족·대사들이 관전 구실로 집결 → 실질 외교 담판

## 서사 메모

> **서사 활용**
> - **토너먼트 음모**: 카멜로스 배 결승전에서 우승 후보 기사 암살 시도 → 에반이 개입

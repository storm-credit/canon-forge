# 아덴브루크 (Adenbruck) — 동쪽과 서쪽을 잇는 황금의 다리

> *"동쪽과 서쪽을 잇는 황금의 다리."*

[[왕국연합|왕국연합(Allied Kingdoms)]] [[카르디온]] 왕국 산하 동부 접경 상업 도시. 성국(Saint Haven)과의 경계에 위치한 **왕국연합 최대 상업 도시**로, 연합과 성국 간 무역의 핵심 창구다. **붉은 깃발 용병단 본부**도 이 도시에 위치한다.

---

## 기본 정보

| 항목 | 내용 |
|---|---|
| **위치** | 에테르 대륙 동부 접경, 성국과의 경계 |
| **역할** | 성국과의 무역 허브 + 왕국연합 최대 상업 도시 |
| **별명** | 외부: "황금의 다리" / 내부: "왕국과 성국이 악수하는 곳" |
| **이미지** | 강변의 활기찬 시장, 상단 깃발이 펄럭이는 다리들, 한쪽 끝엔 성국 문장·반대편엔 연합 문장 |

**핵심 시설**: 대교역장(왕국연합 최대 거래소) · 상인 길드 본부 · 강변 창고 · 환전소 거리 · 붉은 깃발 용병단 본부(Red Banner HQ) · 아덴브루크 훈련소

## 지형 및 도시 구조

### 상징적 지형

| 지형 | 설명 |
|---|---|
| **황금 교각 (Golden Bridge)** | 성국 방면으로 이어지는 대형 교역 다리. 세관 검문소 위치 |
| **대교역장 (Grand Exchange)** | 왕국연합 최대 거래소. 각국 상단 상시 주재 |
| **강변 창고 구역** | 대규모 물류 허브. 성국·카르디온·자유도시 화물 집산 |
| **환전소 거리** | 각 세력 화폐 교환 전문 업소 밀집 |

### 주요 지구

| 지구 | 특징 |
|---|---|
| **교역 지구** | 대교역장, 상인 길드, 중개 사무소 |
| **금융 지구** | 환전소, 어음 발행, 보험 거래 |
| **용병 지구** | 붉은 깃발 본부, 훈련소, 용병 숙소 |
| **성국 접경 지구** | 성국 상단 주재소, 종교 물품 전문상 |

# 드레이븐 (Draven) — 북부 무역 허브

> *"길은 브룬발트 대성채에서 시작하지만, 돈은 드레이븐에서 시작한다."*

[[왕국연합|왕국연합(Allied Kingdoms)]] [[브룬발트]] 왕국 산하 도시. 브룬발트 대성채 인근 최대 민간 도시로, 북부 물자 집산지 역할을 한다. 외교적으로도 정령연합·자유도시 방면 상단의 거점 역할을 하며, 표면 아래로 **검은 매 정찰대의 드레이븐 지하 정보 기지**가 운영된다.

---

## 기본 정보

| 항목 | 내용 |
|---|---|
| **위치** | 에테르 대륙 북서부, 브룬발트 대성채 인근 |
| **역할** | 북부 무역 허브 + 상단 물자 집산지 |
| **별명** | 외부: "북부의 시장" / 내부: "브룬발트가 먹고사는 곳" |
| **이미지** | 대형 창고들이 늘어선 도시 외곽, 북부 상단의 마차 행렬, 요새 도시 특유의 두꺼운 성벽 |

**핵심 시설**: 드레이븐 대시장 · 상단 길드 구역 · 브룬발트 대성채 보급 기지 · 통역사 길드 · 검은 매 드레이븐 기지(지하, 비공개)

## 지형 및 도시 구조

### 상징적 지형

| 지형 | 설명 |
|---|---|
| **북부 대로 (Northern Road)** | 브룬발트 대성채 방향과 연합 내부를 잇는 간선 도로. 군수품·민수품 혼재 |
| **창고 지구 (Warehouse District)** | 도시 외곽 대형 창고 군집. 곡물·광물·무기 비축 |
| **드레이븐 시장 (Draven Market)** | 도시 중심 대형 상설 시장. 북부 상단 거래 중심지 |
| **보급 기지** | 브룬발트 대성채로 향하는 군수품 최종 집결지 |

### 주요 지구

| 지구 | 특징 |
|---|---|
| **상단 지구** | 길드 본부, 환전소, 중개 상인 밀집 |
| **보급 지구** | 군수품 창고, 군 행정 시설 |
| **서민 거주지** | 장인·노동자·용병 거주. 도시 인구 70% |
| **그늘 구역** | 공식 지도에 없는 지하 구역. 검은 매 기지, 밀수 창고 |

## 첩보 거점

**검은 매 드레이븐 정찰 기지**
- 공식 명칭: 드레이븐 제2 경비소
- 실제 기능: 북부 국경 감시 거점 + 정보 분석 지소
- 라이너스 발렌타인의 정보망 핵심 노드 중 하나
- 이 도시를 통과하는 상단 대부분이 검은 매의 소극적 감시 대상

## 서사 메모

> **서사 활용**
> - **보급 차단 음모**: 드레이븐 창고에 화재 or 독약 → 브룬발트 대성채 군수 위기
> - **밀수 추적**: 금지된 마법 물품이 드레이븐 루트를 통해 유통 → 에반이 추적
> - **정보 교환**: 라이너스의 첩보망에 접근하기 위해 드레이븐 기지를 경유해야 하는 상황

# 브라이튼 (Brighton) — 동부 곡창 허브

> *"풍요로운 대지가 왕국의 힘이 된다."*

[[왕국연합|왕국연합(Allied Kingdoms)]] [[리오나르]] 왕국 산하 농업 도시. 에테르 대륙 중서부 비옥한 강 유역에 자리 잡아 리오나르 왕국과 왕국연합 동부 지역의 식량을 책임지는 **동부 곡창 허브**다.

---

## 기본 정보

| 항목 | 내용 |
|---|---|
| **위치** | 에테르 대륙 중서부, 비옥한 강 유역 평원 |
| **역할** | 농업 중심 도시. 동부 곡창 허브 |
| **별명** | 외부: "황금 밀밭" / 내부: "땀의 도시" |
| **이미지** | 끝없이 펼쳐진 황금빛 밀밭, 거대한 저장고들, 농부들의 노래와 수확제 광장 |

**핵심 시설**: 황금 밀밭(Golden Wheat Fields) · 농업 조합 본부 · 중앙 저장고 구역(Granary District) · 수확제 광장 · 농업 마법 연구소

---

## 지형 및 도시 구조

### 상징적 지형

| 지형 | 설명 |
|---|---|
| **황금 평원 (Golden Plains)** | 도시 주변 광활한 밀밭. 수확기엔 금빛 바다처럼 펼쳐짐 |
| **강둑 수로 (Riverbank Canals)** | 관개 시스템. 농업 마법으로 수량 조절 |
| **저장고 언덕 (Granary Hill)** | 대규모 곡물 저장 시설 밀집. 왕국연합 전략 자산 |
| **수확제 광장 (Harvest Square)** | 연례 수확제 축제 장소. 도시 최대 집결지 |

### 주요 지구

| 지구 | 특징 |
|---|---|
| **농업 조합 지구** | 조합 본부, 가격 책정, 상단 협상 |
| **저장고 지구** | 곡물·식량 전략 비축 시설 |
| **농민 거주지** | 소박한 주거, 가족 농장 |
| **연구 지구** | 농업 마법 연구소 + 수확 사제단 신전 |

## 정치·행정 구조

| 기관 | 역할 |
|---|---|
| **농업 조합 의회** | 대농장주 + 소농 대표. 생산 할당·가격 협상 |
| **도시 영주** | 리오나르 왕국 귀족. 세금 징수·치안 |
| **수확 관리관** | 저장고 관리, 비상 식량 배분 |

> "밀 가격을 정하는 자가 진짜 권력자다."

**구조적 갈등**
- 대농장주 vs 소농 이익 충돌 상존
- 흉년 시 폭동 위험. 저장고가 왕국연합 전략 자산이라 경비 강화

## 서사 메모

> **서사 활용**
> - **균열 농지 오염**: 균열이 밀밭에 열려 작물이 오염 → 식량 위기 직전. 에반이 정화
> - **흉년 + 폭동**: 기상 이변 or 마법 저주로 흉작 → 농민 폭동 진압·원인 조사
> - **수확제 가짜 예언자**: 수확제에서 사이비 예언자가 농민을 선동 → 에반이 진위 검증

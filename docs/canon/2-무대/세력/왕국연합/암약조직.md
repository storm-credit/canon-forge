## 4개 조직 교차 역학

| 관계 | 내용 |
|------|------|
| **황금 맹약단 ↔ 검은 왕관단** | 극단적 적대 — 체제 수호(황금) vs 체제 전복(검은 왕관). 만나면 목숨을 건 살육 |
| **황금 맹약단 ↔ 맹약의 검** | 공생 — 황금 맹약단이 자금 제공, 맹약의 검이 경쟁 가문 제거 대리 실행 |
| **황금 맹약단 ↔ 회색 시장 조합** | 분리된 영역 — 자금 흐름은 공유하되 직접적 협력 없음. 같은 구역 이권 충돌 가능 |
| **검은 왕관단 ↔ 회색 시장 조합** | 제한적 협력 — 회색 시장이 반란 자금과 폭약을 판매하되, 정치판 개입은 거부 |
| **맹약의 검 ↔ 검은 왕관단** | 처절한 적대 — 맹약의 검은 체제를 수호하므로 반체제 검은 왕관단을 발각 즉시 참수 |
| **황금 맹약단 ↔ 성국 신성 거래회** | 대륙 공통 암류 — 겉으로 이단을 외치는 성국 금융 조직과 이면에서 채권·금괴 교환 |

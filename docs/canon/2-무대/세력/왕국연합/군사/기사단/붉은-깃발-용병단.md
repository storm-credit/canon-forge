# 09. 붉은 깃발 용병단 (Red Banner Mercenaries)

| 항목 | 내용 |
|---|---|
| **지휘관** | 더글라스 레가스 (Douglas Regas, "용병왕") |
| **본부** | 아덴브루크 (붉은 깃발 본부) |
| **규모** | 등록 용병 7,000명 (유동적) |
| **역할** | 전투 지원·특수 작전·호위·용병 알선 |

**세부 조직**

| 조직 | 역할 |
|---|---|
| **붉은 검단 (Red Blades)** | 근접전 전문. 검투사 출신 다수 |
| **붉은 방패대 (Red Shields)** | 방어 용병 |
| **붉은 창병대** | 기병 저지 장창 |

**계약 구조**
- 왕국연합과 **장기 독점 계약** (준정규군)
- 재무관 리아나 실버레이크와 몸값 협상 진행 중
- 배신 이력 없는 자만 입단. "피의 계약" 체결 — 계약 위반 시 목숨으로 배상

> **[C] 이름 대조**: 더글라스 레가스(Douglas Regas) = 붉은 깃발 용병단 장군. 아이언가르드 도미닉 레가스(Dominic Regas)와 별개 인물. 레가스 가문 출신 두 인물로 처리.

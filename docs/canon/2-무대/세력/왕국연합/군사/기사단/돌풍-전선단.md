# 08. 돌풍 전선단 (Gale Vanguard)

| 항목 | 내용 |
|---|---|
| **지휘관** | 에르메스 게일윈드 (Ermes Galewind) |
| **본부** | 웨스트 마치 (서부 방벽도시 Sanctum Bastion) |
| **규모** | 5,000명 |
| **역할** | 국경 경비·초기 대응·지연전·게릴라 방어 |
| **상징** | 회오리바람 |

**세부 조직**

| 조직 | 역할 |
|---|---|
| **돌풍 방패대 (Gale Shields)** | 바람 마법으로 경량화한 방패 진형 |
| **돌풍 돌격대 (Gale Breakers)** | 방어 중 갑작스러운 충격 돌격 |
| **돌풍 궁병대 (Gale Archers)** | 바람을 타는 장거리 화살 |
| **돌풍 마법단 (Gale Casters)** | 바람 결계로 적 진군 지연 |

**[비밀] 바람의 통로**: 국경 산맥을 관통하는 비밀 동굴 지도 보유. 적의 후방 기습 루트.

**현실**
- 연합 최전방 부대. 심연 군대와 최초 교전. 사상자 가장 많음
- 레갈리아의 화려한 기사단에 비해 주목받지 못해 "우리가 피 흘릴 때 저들은 파티나 한다"는 불만 상존

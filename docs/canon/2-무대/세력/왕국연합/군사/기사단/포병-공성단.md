# 06. 포병 공성단 (Artillery Corps)

| 항목 | 내용 |
|---|---|
| **지휘관** | 크라우스 (Kraus, 수석 포병장) |
| **본부** | 벨가르드 공성 연구소 |
| **규모** | 2,000명 |
| **역할** | 마도포·발리스타·투석기 + 공성·방어 포격 |
| **상징** | 폭발하는 포탄 |

**특징**
- 성국의 '성화의 포병단'과 기술 자존심 경쟁 중 (성국=마법 위주 vs 연합=기계공학 위주)
- 입단 조건: 수학·물리학·마법공학 필기시험 통과. 경쟁률 매우 높음
- 모든 병기에 **자폭 장치** 내장 (탈취 대비)

**주요 병기**

| 병기 | 설명 |
|---|---|
| 드래곤 슬레이어 발리스타 | 용의 비늘도 뚫는 초대형 석궁 |
| 이동식 마도포 | 마차·골렘이 끄는 야전포. 사거리 5km |
| 화염 포탄 | 폭발 후 마법 불꽃 지속 잔류 |

**비밀**: 마도제국이 신형 대포 설계도를 지속적으로 절취 시도 중.

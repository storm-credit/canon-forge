## 01. 창공 기사단 (Sky Knights)

| 항목 | 내용 |
|---|---|
| **지휘관** | 시안 (Sian) |
| **본부** | 레갈리아 (연합 의회 경비 겸임) / 리오나르 (본진) |
| **역할** | 왕국연합 연합 정예 기사단. 기사도의 살아있는 상징 |
| **상징** | 창공을 나는 매 |

**특징**
- 왕국연합 공식 대외 기사단. 국제 행사·사절 경호 등 의전 기능 겸비
- 리오나르 기사 문화의 정수를 체현
- 지휘관 시안은 리오나르 왕국 출신의 전설적 기사

## 02. 용맹의 창병대 (Valiant Lancers)

| 항목 | 내용 |
|---|---|
| **지휘관** | 에드릭 크림슨가드 (Edric Crimsonguard) |
| **본부** | 브룬발트 대성채 |
| **역할** | 중장창 중보병. 성채 방어 + 카운터어택 전문 |
| **상징** | 붉은 십자 창 |

> **이름 주의**: 원본 일부에 "에드윈(Edwin)"으로 기재된 경우 있음. 정본은 **에드릭 크림슨가드(Edric Crimsonguard)**. [[가문]] §크림슨가드 참조.

**특징**
- 브룬발트 국왕 에드몬드의 직속 반격 부대
- 거인의 방패(Giant's Shield) 성문이 열릴 때 돌격하는 역할
- "장창이 열리는 순간 전선이 바뀐다"

## 03. 카멜로스 근위대 (Camellos Guard)

| 항목 | 내용 |
|---|---|
| **지휘관** | 미정 |
| **본부** | 노블가르드 (카멜로스 왕궁 경비) |
| **규모** | 6,000명 |
| **역할** | 왕실 근위 + 의전 경비 + 노블가르드 방위 |
| **상징** | 황금 사자 방패 |

**특징**
- 카멜로스 왕가 직속 근위대. 브룬발트 국왕 에드몬드 신변 보호 포함
- 토너먼트 시즌에는 경기장 경비 겸임
- 외부 위협보다 내부 음모(귀족 암투) 대응에 특화

**입단 조건**
1. 브룬발트 왕국 출신 귀족 기사 이상
2. 카멜로스 왕가 충성 서약
3. 토너먼트 상위 입상 이력 우대

## 04. 폭풍 매 기병대 (Stormhawk Cavalry)

| 항목 | 내용 |
|---|---|
| **지휘관** | 칼릭 카일룸 (Calic Kailum) |
| **본부** | 벨가르드 (마스터 브리더 마구간) |
| **규모** | 4,000기 (전원 기마병) |
| **역할** | 경기병 고속 타격 + 측면 기습 |
| **상징** | 폭풍 속을 나는 매 |

**특징**
- 벨가르드 양익 출격문에서 5,000기*(포함 혼성) 동시 출격 가능한 핵심 전력
- 고속 기동력으로 적 측면을 찌르는 타격전 전문
- 마스터 브리더 아르논의 특수 군마 육성

**전술 능력**
- **[폭풍의 질주]**: 전력 질주 중 적진 돌파. 충격파로 경보병 대형 붕괴
- **[매의 선회]**: 적을 포위하여 원형 기동으로 지속 타격

## 05. 청동 수호단 (Bronze Guardians)

| 항목 | 내용 |
|---|---|
| **지휘관** | 미정 |
| **본부** | 벨가르드 청동 성채 |
| **규모** | 6,000명 |
| **역할** | 마법 저항 방어 + 벨가르드 핵심 방어선 |
| **상징** | 청동 방패 |

**특징**
- 청동 갑옷: 철보다 마법 저항성이 높은 특수 합금 제작. 마법 공격 15~20% 경감
- 벨가르드 청동 성채에 상주. 성채 함락 시도에 대한 최후 방어 전력
- "청동이 녹아도 이 방패는 남는다"

**전술 능력**
- **[청동 방벽]**: 전원 방패 연결, 전방 마법 집중 포격도 버텨내는 진형
- **[마법 흡수]**: 받은 마법 에너지 일부를 역방향으로 반사 가능

## 06. 포병 공성단 (Artillery Corps)

| 항목 | 내용 |
|---|---|
| **지휘관** | 크라우스 (Kraus, 수석 포병장) |
| **본부** | 벨가르드 공성 연구소 |
| **규모** | 2,000명 |
| **역할** | 마도포·발리스타·투석기 + 공성·방어 포격 |
| **상징** | 폭발하는 포탄 |

**특징**
- 성국의 '성화의 포병단'과 기술 자존심 경쟁 중 (성국=마법 위주 vs 연합=기계공학 위주)
- 입단 조건: 수학·물리학·마법공학 필기시험 통과. 경쟁률 매우 높음
- 모든 병기에 **자폭 장치** 내장 (탈취 대비)

**주요 병기**
| 병기 | 설명 |
|---|---|
| 드래곤 슬레이어 발리스타 | 용의 비늘도 뚫는 초대형 석궁 |
| 이동식 마도포 | 마차·골렘이 끄는 야전포. 사거리 5km |
| 화염 포탄 | 폭발 후 마법 불꽃 지속 잔류 |

**비밀**: 마도제국이 신형 대포 설계도를 지속적으로 절취 시도 중.

## 07. 검은 매 정찰대 (Black Hawk Scouts)

| 항목 | 내용 |
|---|---|
| **관리자** | 라이너스 발렌타인 (Raynus Valentine, 그림자의 주인) |
| **본부** | 골드리프 (위장: 감시탑 / 실제: 지하 정보 기지) |
| **규모** | 정찰대원 800명 + 수많은 정보원 |
| **역할** | 원거리 정찰·침투·암살·첩보 |
| **상징** | 검은 매의 깃털 |

**세부 조직**
| 조직 | 역할 |
|---|---|
| **그림자 추적자단 (Shadow Stalkers)** | 은신·침투 특화 특수부대 |
| **바람의 정찰대 (Wind Scouts)** | 1일 300km 주파 속보 전령 |
| **[비밀] 검은 칼날 (Black Blades)** | 요인 암살·납치 전담. 비공개 조직 |

**입단 조건 "생존"**
1. 벨가르드 요새에 몰래 침입하여 사령관의 깃털펜을 훔쳐오기
2. 장비 없이 마수가 우글거리는 숲에서 1주일 생존
3. 드레이븐 지하 '검은 매의 맹세' — 배신 시 즉결 처형 서약

**[비밀] 라이너스의 눈**: 라이너스 발렌타인이 한쪽 눈이 마도공학 의안(義眼)이며, 이를 통해 정찰병들이 보는 시야 일부를 공유받는다는 소문.

**세부 조직**
| 조직 | 역할 |
|---|---|
| **돌풍 방패대 (Gale Shields)** | 바람 마법으로 경량화한 방패 진형 |
| **돌풍 돌격대 (Gale Breakers)** | 방어 중 갑작스러운 충격 돌격 |

## 10. 은빛 전술 사령부 (Silver Tactics Corps)

| 항목 | 내용 |
|---|---|
| **총사령관** | 베르나르드 아우스트루스 (Bernhard Austrus, "대전략가") |
| **본부** | 벨가르드 (은빛 사령부) + 레갈리아 분소 |
| **규모** | 참모·행정·통신병 1,500명 |
| **역할** | 작전 수립·지휘 통제·보급 계획·정보 분석·마법 통신 |
| **상징** | 은색 체스 말 |

**세부 조직**
| 조직 | 역할 |
|---|---|
| **은빛 전략단 (Silver Strategists)** | 엘리트 참모. 워게임으로 작전 설계 |
| **은빛 지휘단 (Silver Commanders)** | 전선 파견 현장관 |
| **은빛 정찰대 (Silver Scouts)** | 군사 정보 전문 (검은 매와 별개) |
| **은빛 광단 (Silver Casters)** | 장거리 통신·전장 감시 마법 |

**핵심 시설**
- **은빛 사령부**: 벨가르드. 마법 홀로그램 전황 지도실. 지하 '고대 전략의 서'
- **은빛 전당**: 레갈리아. 대규모 군사 회의장. '창세의 전술서' 봉인

**비밀**: 병사 사기를 위해 불리한 전황을 일부러 축소 발표하거나 승률을 조작 보고한다는 의혹.

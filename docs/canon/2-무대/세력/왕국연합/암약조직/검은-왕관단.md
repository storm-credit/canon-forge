# 검은 왕관단 (Black Crown Syndicate)

> *"왕좌는 강자의 것이지, 구더기 같은 혈통의 것이 아니다. 그들의 오만함을 피워 오르는 연기와 비명으로 답해주어라."*

## 개요

왕국연합의 부패에 칼을 겨눈 치명적인 반체제 무장 결사. 몰락한 귀족, 용병, 이념적 혁명가, 빈민, 이단 마법사까지 수용하는 이들은 평화로운 체제 전복을 믿지 않으며 오로지 테러·내전 조장·요인 암살을 통해 왕국을 핏빛 무정부 상태로 몰아넣어 새로운 지배 질서를 짜려 한다.

**조직 특성**
- 기사도의 룰 따위는 무시하는 극단주의. 민간인 피해를 아랑곳하지 않고 도심 한복판에 폭탄을 터뜨리는 무자비한 전술의 일상화
- 뚜렷한 거점 없이 점조직으로 움직이며, 귀족의 침실부터 하수도 밑바닥까지 동조자가 숨어 있는 게릴라 네트워크

## 주요 활동

| 활동 | 방식 |
|------|------|
| **귀족 및 왕권 전복 테러** | 국왕 행차나 귀족 무도회에 시한부 마도 폭약을 밀반입해 대귀족 일가족을 육편으로 날림 |
| **무기 밀거래와 전쟁 조작** | 일부 국경 영주 사병 반란을 부추기고 밀수한 공성 병기·저주받은 마법검을 팔아 국지적 내전 촉발 |
| **선동 및 폭동 주도** | 빈민가·피난민 캠프에 끄나풀을 심어 굶주린 이들이 곡물 창고와 영주 성을 약탈하도록 폭동 유도 |
| **정치 요인 연쇄 암살** | 연합 방어선 강화를 주장하는 유능한 장군·충직한 관료들만 골라 잔인하게 도륙. 왕국의 척추를 부러뜨림 |

## 핵심 인물

| 인물 | 별명 | 역할 |
|------|------|------|
| **라그나르 블랙웰 (Ragnar Blackwell)** | 무관의 왕 | 주류에 처참히 몰락한 귀족 가문 출신의 냉혹한 혁명가. 복수심으로 뭉친 압도적 카리스마. 어떤 희생도 감수하는 폭군의 기질. L-1급 |
| **카산드라 벨몬트 (Cassandra Belmont)** | 사신 | 암살·테러 부대의 그림자 수장. 타겟의 숨통을 끊는 순간 기이한 미소를 띠는 히어로급 암살자 |

| **미하일 레벤토프 (Mikhail Leventov)** | 어둠의 선동가 | 피비린내 나는 연설과 허위 삐라 조작으로 굶주림을 증오와 낫으로 승화시키는 사상적 흑막 |

|------|------|
| **왕국연합 반체제 귀족** | 자신들이 억압당한다고 믿는 몰락 위기의 하급 귀족들과 은밀히 쿠데타를 각 잡고 있음 |
| **자유도시연합 암시장** | 실버포트 인근 범죄 카르텔(검은 해골단 등)과 피의 거래망으로 물자·무기 충당 |

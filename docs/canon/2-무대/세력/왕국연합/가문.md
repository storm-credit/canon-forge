# 왕국연합 가문 아카이브

> *"깃발은 많고, 왕은 여럿이며, 검은 하나로 움직인다 — 그래야만 한다."*

왕국연합은 단일 왕국이 아닌 **브룬발트·리오나르·카르디온** 3왕국의 연합체다. 레온하르트(브룬발트)가 순환 의장을 맡고 있으나, 에드몬드(리오나르)와 제라르드(카르디온)는 독립된 통치권을 유지하며 사사건건 견제한다.

## 가문 간 관계

| 관계 | 유형 | 설명 |
|------|------|------|
| **카멜로스 ↔ 실버랜스** | 긴밀 협력 | 왕과 총사령관 — 전장에서 등을 맡기는 절대적 신뢰 |
| **카멜로스 ↔ 듀락** | 악우(惡友) | 성격 정반대(공격 vs 방어)이지만 유일한 동등 친구. 정치적으로 자주 충돌 |
| **카멜로스 ↔ 파브리스** | 경계적 의존 | 전비 어음의 최대 채권자. 돈줄을 쥔 제라르드가 의회에서 발언권 확보 |
| **실버랜스 ↔ 듀락** | 군사적 경쟁 | 공세 vs 방어 — 전략 회의마다 격론. 하지만 전장에서는 찰떡 호흡 |
| **실버랜스 ↔ 파브리스** | 긴장 | 군비 증강 요구 vs 재정 건전성 주장 — 예산 전쟁의 양 당사자 |
| **듀락 ↔ 파브리스** | 실무 협력 | 성벽 건설·보급선 유지에서 실무적 협력. 정치적으로는 중립 |
| **연합(4권력) ↔ 녹슨 왕관** | 극악의 대립 | 명예를 버린 구시대 기사들과 도적단. 보급선 약탈로 연합 후방 갈아먹음 |

## 균형 유지 메커니즘

### 카멜로스의 중재
- 실버랜스가 과도한 전쟁을 추진 → **왕관회의에서 병력 분담률 조정으로 견제**
- 파브리스가 군비 삭감을 요구 → **기사도 명분으로 최소 군사력 보장**
- 듀락이 고립주의에 빠짐 → **연합 방어 조약을 이유로 참여 강제**

### 듀락의 군사적 균형추
- 실버랜스의 공세적 전략이 과해질 때 → **방어 논리로 제동**
- 파브리스가 용병에 과도하게 의존할 때 → **정규군 건설을 주장**
- 내전 조짐이 보일 때 → **서부 국경의 위협을 상기시켜 단결 유도**

### 파브리스의 경제적 지렛대
- 실버랜스가 군비 증강을 강행 → **전쟁 어음 이자율 인상으로 압박**
- 카멜로스가 일방적 결정을 시도 → **관세 조정으로 간접 견제**
- 전쟁이 장기화될 조짐 → **보급선 조절로 전쟁 조기 종결 유도**

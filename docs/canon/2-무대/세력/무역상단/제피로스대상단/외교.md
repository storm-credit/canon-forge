# 제피로스 대상단 — 외교

# 0. 제피로스 바람의 전령 상단 외교 관계

## 1. 개요
외교 전략: 모든 국가의 통신망을 독점하다시피 하며, 첩보 조직(6-7)의 강력한 견제를 받고 있다.

동맹과 배신이 난무하는 외줄타기 전술을 구사한다.

## 2. 서사적 디테일 (Epic Detail)
아스트라리스 크로니클의 무자비한 생태계 속에서, 이들의 존재는 단순히 세력을 넘어 **세계관의 운명(Providence)**을 뒤흔드는 무거운 축으로 기능한다.

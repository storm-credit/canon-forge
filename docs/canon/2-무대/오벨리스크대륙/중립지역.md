# 오벨리스크 대륙 중립 지역 (Neutral Zones)

> *"영혼과 선혈이 거래되는 황혼의 암시장."*

죽음의 대륙에서도 세력을 초월해 산 자와 죽은 자가 공존하는 거점들. 무역·생존·신앙이 교차하는 삶의 마지막 거점이다.

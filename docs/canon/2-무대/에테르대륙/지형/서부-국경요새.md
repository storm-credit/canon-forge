# 서부 — 국경 요새 지대 (Western Border)

> *"지도 위에는 잉크로 그어져 있지만, 이 땅의 실제 경계선은 수만 번 다져진 시체들의 뼈와 참호로 새겨져 있다."*

크림슨 대륙(붉은 황무지)의 야만 군단·타 세력 침공을 최전선에서 받아내는 영원히 잿빛인 요새 방벽. [[성국]]의 군사 도시 [[아스트라르]]가 중심.

## 지정학적 풍경

- **아스트라르 요새 평원 (Astrar Front)**: 잿빛 하늘 아래 끝없는 강철 방벽·참호, 멈추지 않는 상시 전면전. **에테르 대륙의 고기 분쇄기(Meat Grinder)** — 교리에 반역했거나 권력에 밀린 잡병이 징벌 부대로 끌려와 포병 마나석 교체 중 마력 화상으로 산화. '내일의 승리'를 안 믿는 허무주의·소모전의 공기.
- **블랙웨이 협곡 (Blackway Gorge)**: 서부 산맥을 거인의 도끼로 내려찍은 듯한 검붉은 열극. 대규모 병력·파성투석기·거대 마수가 지날 유일한 대륙 간 육로 — 절벽 내부에 수천 저격 진지·살수 함정. 파견은 곧 확정된 죽음.

## 주요 지명

| 지명 | 동향 |
|---|---|
| **[[아스트라르]] 요새** | '구원의 방패' 선전 이면의 군국주의 병영. 모든 주민이 태어날 때부터 규격화된 소모품 병사 |
| **실드포드 전초기지** | 전선 직전 마지막 쾌락지. 피 묻은 군함·장비를 매입하는 전쟁 카르텔과 징벌 부대의 공생 |
| **그레이리지 무법지대** | 전선이 매주 뒤바뀌어 소속국이 바뀌는 회색 지대. 기회주의 정보 브로커 집합소 |

## 서사 후크

- **고대 포열의 폭주**: 협곡에 파묻힌 고대 에테르 대포가 강제 오작동 → 피아 불문 초토화, 성국·적국이 일시 휴전 후 레이드.
- **징벌군의 대반역**: 강압 지시에 분노한 징벌 기사단이 방벽 문을 잠그고 성국 지휘관 목을 매다는 내부 무장 반란.
- **잿빛 병사들의 마나 피폭**: 부서진 전투 크리스탈에서 누출된 불량 에테르에 집단 피폭 → 이성을 잃고 동족을 물어뜯는 [마수화] 전염.

# 캐논 정리 마스터 플랜 & 프롬프트

> 목적: 『THE FORGOTTEN SUMMONER』 원본 8,954파일을 **확장 아닌 정리(consolidation)**로
> 깨끗·일관·완결된 캐논으로. 작성: 2026-06-15.

## 정리 순서 (바닥부터 — 의존성 역순)

윗단이 아랫단을 참조하므로 **아래→위** 순서로 가야 재작업이 없다.

| Phase | 대상 | 왜 이 순서 | 원본 위치 | 규모 |
|---|---|---|---|---|
| **0. 헌법** | 세계의 틀 + 6대 크로니클 + 불변식 | 모든 것의 기준. 먼저 잠금 | `00. 세계의 틀`, `00-3 크로니클` | 4 |
| **1. 시스템** | 직업/클래스 · 마법 체계 · 등급(권능 위계) 규칙 | 인물·아이템이 전부 참조 | `01-20 직업`, `01-31 마법` | ~1,900 |
| **2. 무대** | 대륙→국가→도시→문화 · 세력 구조 | 인물·사건의 배경 | `01-8 세력 아카이브` | 3,903 |
| **3. 사물** | 아이템 도감 · 유물 단일소유 | 직업/마법에 의존 | `01-19 아이템` | 729 |
| **4. 인물** | 성장영웅 → 세력 영웅 → 에반·페르소나 | 위 전부에 의존 | `01-14`, `01-15` | ~3,000 |
| **5. 수렴** | 위키링크 정합 · 고아 제거 · 모순 0 확인 | 전체 교차검증 | 전역 | — |

> ⚠️ 사용자가 "아이템부터" 원하면 Phase 3을 먼저 해도 된다. 단 직업/등급(Phase 1)이
> 확정 안 된 상태라 아이템의 "사용 직업·등급" 칸은 ⚠️[보류]로 두고, Phase 1 후 채운다.

## 전체 카테고리 배정 (빠진 것 없음 — 29개 전부)

원본 모든 백과를 단계에 매핑. 교통·신앙·정령 등 하나도 안 빠뜨림.

| Phase | 포함 카테고리 (원본) | 파일수 |
|---|---|---|
| **0. 헌법** | `00. 세계의 틀` · `01-1 창세 신화`(7) · `01-3 6대 크로니클` · `01-10 세계 종교 및 신앙 체계`(10) · `01-16 연대기`(9) · `01-17 고대 문명`(8) | ~38 |
| **1. 시스템** | `01-20 직업·클래스`(40) · `01-31 마법 백과`(1866) · `01-36 주술`(27) · `01-33 신성`(21) · `01-35 마법 공학`(24) · `01-37 소환`(27) · `01-42 차원·공간 마법`(5) · `01-34 연금`(213) · `01-32 정령`(212) | ~2,435 |
| **2. 무대** | `01-8 세력 아카이브`(3903) · `01-3~7 5대 대륙 지리` · `01-41 마도 교통`(40) · `01-40 마도 통신`(11) · `01-9 전쟁사`(10) · `01-11 대륙별 전쟁 기술`(10) | ~4,000 |
| **3. 사물** | `01-19 아이템 도감`(729) · `01-38 동반수`(24) · `01-39 몬스터`(52) | ~805 |
| **4. 인물** | `01-14 영웅 백과`(1491) · `01-15 인물 백과`(33) · 에반·9페르소나 | ~1,524 |
| **5. 수렴** | `_Backup_Legacy`(51) 등 레거시 격리·폐기 · 전역 위키링크 정합 · 고아 제거 · 모순 0 | 51+ |

**카테고리 귀속 메모:**
- **신앙/종교** = 2층. *체계*(01-10 신앙 체계, 01-1 창세 신화)는 Phase 0 헌법. *마법으로서의 신성력*(01-33 신성 백과)은 Phase 1 시스템. *세력별 종교 운용*은 Phase 2 세력 아카이브 안에서.
- **교통/통신** = `01-41 마도 교통`·`01-40 마도 통신` → Phase 2 무대(인프라).
- **정령/연금** = 마법 체계의 일부 → Phase 1 시스템. (정령 212개는 양 많음 — 개별 엔티티는 Phase 4식 처리 가능)
- **몬스터/동반수** = Phase 3 사물(전투 대상·동반 개체)로 묶되, 서사 비중 큰 개체는 Phase 4 교차.
- **전쟁사** = Phase 2 무대(세력 간 역사).

## 각 단위(카테고리/엔티티)마다 쓰는 정리 프롬프트

```
역할: 너는 『THE FORGOTTEN SUMMONER』의 캐논 편집장이다.
목표는 설정 확장이 아니라 정리(consolidation)다. 결과는 원본보다 짧고 깨끗해야 한다.

[절대 불변식]
- (제거됨 2026-06-16) ~~대가(등가교환)는 단조누적이며 되돌릴 수 없다.~~ → "치명적 페널티" 강제 법칙은 Hanesis Protocol 주입물로 확인되어 캐논에서 제거. 대가는 어둠·금기 분야 한정 서사 장치로만 존치.
- (제거됨 2026-06-16) ~~등급 ↔ 대가는 정합한다 (높은 등급 = 더 큰 대가).~~ → 등급은 권능 위계만 의미. 대가와 강제 연동 안 함.
- 신화 유물은 단일소유 (한 시점에 한 소유자).
- 에반의 9대 위장 페르소나 = 에반 본인의 변장. 동시·이격 등장은 모순 아님.
- 에반의 대역 오토마톤 = 자아 없는 빈 껍데기. 자율 판단·의지 행동은 모순.
- "신은 답하지 않는다. 모든 선택과 결말은 인간의 것." 신 중심 구원 서사 금지.

[작업 순서]
1. 추출 — 각 원본 파일에서 핵심 사실만: 정본명·별칭·등급·소속·대가·핵심관계·출처경로.
2. 충돌 검출 — 다음을 찾아 표시:
   · 모순(같은 대상이 등급/소속/대가가 다르게 기재)
   · 중복(같은 대상이 여러 파일에 흩어짐)
   · 고아(어디서도 연결 안 되는 항목)
   · 이름증식(비슷한 이름이 동일인인지 별개인지)
   · 불변식 위반
3. 판정 — 충돌마다 하나 선택: 정본채택 / 병합 / 폐기 / 보류
   기준: ① 원본 evidence 있는 쪽 우선 ② 더 구체적·일관된 쪽 ③ 불변식에 맞는 쪽.
4. 개연성 보강 — 끊긴 연결고리만 기존 설정 근거로 메움.
   근거 없는 새 내용은 반드시 ⚠️[추가]로 표시(사용자 승인 전엔 캐논 아님).
5. 출력 — 아래 캐논 템플릿으로.

[출력 규칙]
- 모든 항목에 출처(source_path) 표기. 확정 못한 건 삭제 말고 ⚠️[보류].
- 마음대로 설정을 지어내지 않는다. 원본에 있는 것만 캐논, 메운 것은 표시.
- 편집장 노트에 "정정한 모순 / 미해결 / 추가 제안"을 남긴다.
```

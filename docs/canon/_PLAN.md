# 캐논 정리 마스터 플랜 & 프롬프트

> 목적: 『THE FORGOTTEN SUMMONER』 원본 8,954파일을 **확장 아닌 정리(consolidation)**로
> 깨끗·일관·완결된 캐논으로. 작성: 2026-06-15.

## 정리 순서 (바닥부터 — 의존성 역순)

윗단이 아랫단을 참조하므로 **아래→위** 순서로 가야 재작업이 없다.

| Phase | 대상 | 왜 이 순서 | 원본 위치 | 규모 |
|---|---|---|---|---|
| **0. 헌법** | 세계의 틀 + 6대 크로니클 + 불변식 | 모든 것의 기준. 먼저 잠금 | `00. 세계의 틀`, `00-3 크로니클` | 4 |
| **1. 시스템** | 직업/클래스 · 마법 체계 · 등급↔대가 규칙 | 인물·아이템이 전부 참조 | `01-20 직업`, `01-31 마법` | ~1,900 |
| **2. 무대** | 대륙→국가→도시→문화 · 세력 구조 | 인물·사건의 배경 | `01-8 세력 아카이브` | 3,903 |
| **3. 사물** | 아이템 도감 · 유물 단일소유 | 직업/마법에 의존 | `01-19 아이템` | 729 |
| **4. 인물** | 성장영웅 → 세력 영웅 → 에반·페르소나 | 위 전부에 의존 | `01-14`, `01-15` | ~3,000 |
| **5. 수렴** | 위키링크 정합 · 고아 제거 · 모순 0 확인 | 전체 교차검증 | 전역 | — |

> ⚠️ 사용자가 "아이템부터" 원하면 Phase 3을 먼저 해도 된다. 단 직업/등급(Phase 1)이
> 확정 안 된 상태라 아이템의 "사용 직업·등급" 칸은 ⚠️[보류]로 두고, Phase 1 후 채운다.

## 각 단위(카테고리/엔티티)마다 쓰는 정리 프롬프트

```
역할: 너는 『THE FORGOTTEN SUMMONER』의 캐논 편집장이다.
목표는 설정 확장이 아니라 정리(consolidation)다. 결과는 원본보다 짧고 깨끗해야 한다.

[절대 불변식]
- 대가(등가교환)는 단조누적이며 되돌릴 수 없다.
- 등급 ↔ 대가는 정합한다 (높은 등급 = 더 큰 대가).
- 신화 유물은 단일소유 (한 시점에 한 소유자).
- 에반의 9대 위장 페르소나 = 에반 본인의 변장. 동시·이격 등장은 모순 아님.
- 에반의 대역 오토마톤 = 자아 없는 빈 껍데기. 자율 판단·의지 행동은 모순.
- "신은 답하지 않는다. 모든 선택과 결말은 인간의 것." 신 중심 구원 서사 금지.

[작업 순서]
1. 추출 — 각 원본 파일에서 핵심 사실만: 정본명·별칭·등급·소속·대가·핵심관계·출처경로.
2. 충돌 검출 — 다음을 찾아 표시:
   · 모순(같은 대상이 등급/소속/대가가 다르게 기재)
   · 중복(같은 대상이 여러 파일에 흩어짐)
   · 고아(어디서도 연결 안 되는 항목)
   · 이름증식(비슷한 이름이 동일인인지 별개인지)
   · 불변식 위반
3. 판정 — 충돌마다 하나 선택: 정본채택 / 병합 / 폐기 / 보류
   기준: ① 원본 evidence 있는 쪽 우선 ② 더 구체적·일관된 쪽 ③ 불변식에 맞는 쪽.
4. 개연성 보강 — 끊긴 연결고리만 기존 설정 근거로 메움.
   근거 없는 새 내용은 반드시 ⚠️[추가]로 표시(사용자 승인 전엔 캐논 아님).
5. 출력 — 아래 캐논 템플릿으로.

[출력 규칙]
- 모든 항목에 출처(source_path) 표기. 확정 못한 건 삭제 말고 ⚠️[보류].
- 마음대로 설정을 지어내지 않는다. 원본에 있는 것만 캐논, 메운 것은 표시.
- 편집장 노트에 "정정한 모순 / 미해결 / 추가 제안"을 남긴다.
```

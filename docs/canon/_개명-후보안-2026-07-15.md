# 개명 후보안 — 근접음(近接音) 클러스터 6군

> ⚠️ **이 문서는 제안서다.** 어떤 이름도 이 문서로는 바뀌지 않는다. 각 행 '작가 결재란'에 결정이 기입되고 별도 수선 티켓이 발부되기 전까지 캐논·원고는 현행 유지한다. 신규 이름 후보(④열)는 제안 목적의 창작이며, 채택 전에는 정본이 아니다.
>
> 편집장 노트: 2026-07-15 품질 Wave2 — 접근성 렌즈(5.5) '근접음 병렬 노출' 결손의 **개명 옵션**을 실측 근거와 함께 제시. (산문 노출 회피 규칙은 이미 02.시점-문체 §A1·A4에 등재됨 — 본 문서는 그 회피 규칙으로 부족한 경우의 대체 개명안을 다룬다.)
>
> 🔺 **재채점 갱신 (2026-07-15 품질 Wave3 · 접근성반):** 접근성 렌즈 재채점 결과 **5.5 → 4.5 하향**, 근접음 병렬이 설정집 최상위 문제로 재적발됐다. 재채점의 지적("원고 본문에 루미에·루미에르·페르디타 3인 공존")을 원고 1~100화 실측으로 검증한 결과 — **루미에르 아르카나(히로인)는 원고 다수 실등장(452회+, 022~060화 등)으로 루미에(상시 화자)와 본문에서 실제 공존**하나, **페르디타(팻)는 원고 0건**(§0 표와 일치)이다. 따라서 현 본문 공존은 3인이 아니라 **루미에↔루미에르 2인**이며, 페르디타는 '아직 산문에 없는 잠재적 제3바퀴'로 **위험도 하향**한다. 다만 **팻이 산문에 등장하는 순간 3중 병렬로 승격**되므로, **A안(페르디타 개명)의 결재 시급도를 상향**한다 — 팻(페르디타) 등장화 집필 착수 전 결재를 권장한다(집필 후엔 원고 재작업 비용 발생). 나머지 원고 0건 클러스터의 '개명 골든타임' 논리는 불변. 상위 표기 지침(T-1~T-4: 동시노출 하드규칙·아스트라리스 접두·솔라리스 이명 정본·아르카나 구분)은 02.시점-문체에 등재. **본 갱신도 제안 상태이며 어떤 개명도 집행하지 않는다(작가 결재 전 무효).**

## 0. 실측 방법과 표기

- **원고**: `원고/` 전체(크로니클1 Act1 001~100화 + 스토리-상태·브리프·점검 장부). 실제 집필된 본문.
- **캐논**: `docs/canon/` 전체(설정 문서).
- 수치는 `rg -P`(PCRE2) 실측. `occ`=구절 출현 수, `files`=출현 파일 수. 근접 파생어(예: 미카엘리스에 포함된 '카엘리스')는 분리 집계했고 본문에 주기했다.
- 핵심 원칙: **원고 등장 실측치가 개명 파급의 1차 지표**다. 원고 0건 = 지금 바꾸면 본문 재작업 없음(캐논만 수선). 원고 다수 = 이미 집필된 본문 전량 재작업.

## 2. 클러스터별 상세

### A. 루미에 / 루미에르 / 페르디타 — 그리고 ⑤ 서사 연관 검토

**정체**
- **루미에(Λumie)** — 에반 어깨 위 아카식 번역가, 마력 잔량 실시간 계측, **상시 화자**(원고 1290회).
- **루미에르 아르카나** — 히로인, 아크브릿지 공방주, 촉매 호환성·공명 측정. Act1 16~40화에 **실제 등장**(원고 572회).
- **페르디타(Perdita)** — 에반의 팻, 성국 기원 신성-사령 하이브리드 진홍 파랑새, 세레나와 성력 콤보. **원고 미등장**.
- 추가 근접음: **세레나 루미나스(Luminas)**, 영웅 姓 **루미너스(Luminous)** → 실질 5중.

**① 위험도 = 최고.** 근거: 감사 §12 "둘 다 소환 계측 역할이라 '누가 게이지를 읽나' 혼선" — 루미에·루미에르는 음가뿐 아니라 **기능(소환 계측)까지 유사**하다. 팻 페르디타까지 한 챕터에 들어오면 라이트 독자는 화자·연애서사 대상·펫을 즉시 분간 못 한다(감사 접근성 §207).

**② 회피 규칙만으로?** 부분적. 02.시점-문체 §A4 화자 클러스터 특칙(루미에=대사 태그 없이 문맥 식별, 루미에르·페르디타는 소속·직함 병기)이 이미 존재한다. 다만 3중 동시 노출은 각주 미열람 본문 독자에게 과부하 — 물리적 이름 수를 3→2로 줄이는 편이 안전.

**③ 최소파급.** 루미에(원고 1290)·루미에르(원고 572)는 **개명 불가 영역**(집필된 본문 전면 재작업). 페르디타만 원고 0·캐논 62/10 → 개명 비용이 압도적으로 낮다.

**⑤ 루미에 ↔ 루미에르 서사 연관(파편·아르카나) 검토 — 결론: 근접음은 부분적으로 의도된 설계이므로 유지가 옳다.**
- 루미에.md L57 + 루미에르 아르카나 위키 §11: 루미에르 아르카나는 **"Λumie와 데이터상으로 교신 가능한 유일한 화이트 해커"**로 정합돼 있다. 루미에의 목소리가 매개체 없이 처음 외부에 닿는 대상이 루미에르다(스토리-상태 §172~173, 023화 등).
- 즉 두 이름의 음가 겹침은 우연이 아니라 **교신 정합 + 헌법 §1 인과율 메아리**로 읽을 여지가 있다(루미에.md L14 동명이인 박스도 "음가의 겹침은 헌법 §1 인과율 메아리이거나 단순 동명"이라 명시).
- 그러나 관계도-허브 L95는 둘을 **명백한 별개 존재**로 못박는다. 서사적 링크는 있으나 '동일 존재 암시'는 아니다.
- **판단**: 루미에·루미에르의 근접음은 (a) 원고 파급이 감당 불가하고 (b) 교신 서사와 공명하는 설계 여지가 있으므로 **개명하지 않는다**. 대신 **의도 없는 제3바퀴 페르디타(팻)를 개명**해 트리오를 듀오로 정리하고, 남은 듀오는 §A4 특칙 + 기능 차별화(Λumie는 대사 태그, 루미에르는 姓/직함 '아르카나')로 방어한다.
- ⚠️ 만약 작가가 루미에↔루미에르를 **서사적으로 하나로 수렴**시킬 의도라면(파편이 곧 루미에르?), 근접음은 복선으로 승격되고 개명 자체가 불필요해진다 — 이 방향이면 A안 전체를 보류.

**④ 페르디타(팻) 후보** — 진홍의 신성-사령 하이브리드 새. 기존 팻 명명 결(이그니스·크리서스·오메가 = 단어형 고유명)에 맞춘 라틴계, '루미' 어두 완전 회피:
- **상귀스(Sanguis)** — '피'. 진홍·성수 폭발의 살상성과 직결.
- **페르디타(Perdita)** — '잃어버린 자'. 화형당한 수녀 원혼 모티프.
- **카르나(Carna)** — '육(肉)의'. 치유 아닌 육체 융해 상징.

### B·C. 세이렌 / 세레나 · 칼리스 / 카엘리스 (통합 검토)

**정체**
- **세이렌 칼리스(Seiren Calis)** — 히로인, 신앙적 봉인 vs 자유의지, 성녀→흑화. **given name 세이렌 + 姓 칼리스 둘 다 클러스터에 걸림.**
- **세레나 루미나스(Serena Luminas)** — 히로인, 예언. (given name 세레나는 루미나스 姓과 A군에도 간접 연루.)
- **오로라 카엘리스(Aurora Caelis)** — 히로인, 감시자·결정론.
- 배경 근접음: **칼리스타·칼리스트·칼리스토**(별개 인물/지명, 캐논 100+), **미카엘리스(Michaelis, 78)** — 카엘리스를 어미로 포함.

**① 위험도.** 세이렌↔세레나(given name 2인)와 칼리스↔카엘리스(姓 2개)가 히로인 명부에 동시 존재. 세이렌 칼리스는 **두 클러스터의 교차점**이라 단독으로 B·C를 동시에 오염시킨다.

**② 회피 규칙만으로?** 미흡. given name은 산문에서 가장 자주 노출되는 호칭이라 '세이렌/세레나'를 회피어로만 다루기 어렵다. 姓 병렬은 §A4로 완화 가능하나 given name 병렬은 규칙 방어의 사각.

**③ 최소파급 + 통합안.** 전원 원고 0건. 캐논 파급: 세이렌 227/45, 세레나 284/48(세레나가 근소 우위 = 세이렌 개명이 파급 적음). 카엘리스(순수 姓 ~75) vs 칼리스(순수 姓 ~127) — 카엘리스 개명이 파급 적음.
- **접근성반 권고(1수 2해소)**: **세이렌 칼리스를 given+姓 전면 개명**하면, given에서 세이렌↔세레나가 풀리고 姓에서 칼리스가 히로인 명부를 떠나 카엘리스가 자동으로 유일해진다(B·C 동시 종결). 배경의 칼리스타/칼리스트는 히로인이 아니라 산문 최전선 충돌 위험이 낮다.
- **대안(분리 개명)**: 세이렌 칼리스 given만 교체(세이렌→후보) + 오로라 카엘리스 姓만 교체(카엘리스→후보). 두 문서를 건드리므로 통합안보다 파급 큼.

**④ 후보**
- **세이렌(given) →** 성녀·치유·성국 결의 라틴/서구계, '세-' 어두 회피: **아우렐리아(Aurelia)** · **미리엘(Miriel)** · **비비안(Vivian)**.
- **(통합안 시 姓 칼리스는 유지 불필요 — 세이렌 칼리스 자체를 위 given + 새 姓으로.** 새 姓 후보(카엘리스와도 이격): **소랑주(Solange)** · **아르덴(Arden)** · **벨포르(Belfort)**. 단 '벨포르'는 D군 벨라와 어두 유사하니 제외 권장 → 소랑주/아르덴 우선.)
- **(분리안 시) 카엘리스(姓) →** 감시자·천문·결정론 결: **시더리스(Sideris, '별의')** · **아스트라이아(Astraea)** · **옵스쿠라(Obscura)**.

### D. 벨라 3연 — 벨라마르 / 벨라시아 / 벨라돈나

**정체** — 히로인 3인이 '벨라' 어근 공유: 마리나 **벨라마르**(항해)·**벨라시아** 문(오컬트 정보원)·릴리스 **벨라돈나**(네크로맨시). 배경 근접음: 벨라이트(보석상단 348)·벨라트릭스·벨라나 등.

**① 위험도 = 중~높음.** 다만 산문 호명 습관이 완충한다: 마리나·릴리스는 given name으로 불려 '벨라' 어두가 안 뜬다. **벨라시아만 given name 자체가 '벨라' 어두** → 세 히로인이 동석하고 벨라마르/벨라돈나 姓이 언급되는 순간 벨라시아와 3중 병렬.

**② 회피 규칙만으로?** 부분적. 姓(벨라마르·벨라돈나)은 §A4 병기 규칙으로 완화 가능. 문제는 **벨라시아의 given name 노출**은 회피 불가(주 호칭).

**③ 최소파급.** 전원 원고 0건. **벨라시아 given name 교체가 핵심 1수** — 산문 최전선의 '벨라' 어두를 제거한다. 완전 분리를 원하면 벨라마르(51) 또는 벨라돈나(48) 중 의미결속이 약한 쪽을 추가 교체. (릴리스 벨라돈나=Belladonna 독초=네크로맨시·독 정체성과 강결속이라 **벨라돈나는 보존 권장**; 추가 교체가 필요하면 벨라마르 쪽.)

**④ 벨라시아(given) 후보** — 오컬트 정보원·운명 결, '벨라' 어두 완전 회피: **카산드라(Cassandra, 예언)** · **모르가나(Morgana, 오컬트)** · **헤카테(Hecate, 마녀 여신)**.
> ⚠️ 확인 필요: '벨라시아'가 인물 given name 외에 지명/기원(벨라시아 문 = 세력·달)으로도 쓰이는지(캐논 벨라시아문 30건) 작가 확인 후 개명 범위 확정.

### E. 솔라리스 / 솔라리스 프라임 / 솔라리안

**정체**
- **솔라리스** — **성국 수도**(에테르 대륙 중앙, '신성한 분지'). 성국 심장, 캐논 929/318.
- **솔라리스 프라임** — **솔라리안 제국 수도**(크림슨 대륙 중앙, 태양의 평원). 구명칭 **헬리안도르(Heliandor)** 잔재 존재(공간정합 §4).
- **솔라리안** — 크림슨 대륙 제국명/형용(2369/667, 크림슨 문서 전역).

**① 위험도 = 높음.** '솔라리스'라는 **동일 어근 수도가 두 대륙에 각 1개**(성국 수도 = 솔라리스, 제국 수도 = 솔라리스 프라임) + 제국명 솔라리안 3중. 크림슨-에테르 정치가 교차 서술되는 후반(대균열 수렴)에서 "어느 솔라리스인가" 혼선.

**② 회피 규칙만으로?** 부분적. §A1이 이 군을 인지 등재했고, 두 대륙 분리로 초·중반은 자연 격리된다. 그러나 **두 수도의 동어근**은 후반 교차에서 회피 규칙의 한계 — 도시 vs 도시라 직함 병기로도 잘 안 갈린다.

**③ 최소파급.** 전원 원고 0건. 성국 수도 솔라리스(318파일)는 성국 서사의 중핵이라 개명 파급 최대. **솔라리스 프라임(49파일)이 최소파급**이고, 결정적으로 **대체명 헬리안도르가 이미 캐논에 존재**(공간정합 §4가 별칭/삭제 권고) → 무(無)발명 승격 가능.

**④ 솔라리스 프라임 후보**
- **헬리안도르(Heliandor)** — 기존 잔재명 정본 승격. 발명 0, '솔라리스' 어근 완전 이탈, 대지맥 종착지 서사와 결속.
- **아우레아(Aurea, '황금')** — 태양의 평원·황금 곡창 결.
- **임페리온(Imperion)** — 제국 수도 위엄.
> 성국 수도 **솔라리스**와 제국명 **솔라리안**은 유지 권장(다른 대륙·다른 범주=도시 vs 제국, §A1 회피로 충분). 사안의 급소는 '수도 vs 수도' 동어근뿐.
> ⚠️ 공간정합 §4 미결(수도 위치 서부 vs 중앙, 헬리안도르 처리)과 연동 — 개명 결정 시 그 감사건과 합산 처리.

### F. 마르코 / 마테오 블랙잭 — (근접음 아님 · 표기 정합)

**정체**
- **마르코 블랙잭(Marco Blackjack)** — 카르델라 대부, 조직 '블랙 수트', **Act1 캐논 실명 확정**(콘티 §226: 설계안의 '마테오'는 표기 오류, 마르코가 실명). 원고 다수 등장.
- **마테오 블랙잭(Mateo Blackjack)** — 루시엔 아르카나(성장영웅) 문서의 카르텔 대부. 원고 0.
- 참고: **모건 블랙잭(황금손아귀)** — 별개 인물. '블랙잭'은 영웅 명명체계처럼 **姓 공유 설계**가 이미 존재.

**성격.** 이는 근접 '음가'가 아니라 **동일 역할(카르텔/카르델라 대부)에 두 given name이 붙은 표기 불일치**다. 동명판정 대장 L63에 '작가 판단 보류(동일인 여부)'로 등재, 감사 §12·§74가 "루시엔 합류 전 선(先)해소 필수"로 지정.

**③ 최소파급.** 마테오 54/11(원고 0) vs 마르코 68/29+원고 59. 마테오 개명/통합이 파급 최소.

**옵션**
- **옵션 A(권고) — 동일인 통합**: 마테오 블랙잭 → **마르코 블랙잭**으로 통일. 콘티 §226이 이미 마르코를 실명으로 확정했으므로 발명 0, 루시엔 문서만 11파일 수선. 세력 모순 소멸.
- **옵션 B — 별개 형제/일족**: 마르코·마테오를 블랙 수트 vs 다른 카르텔의 별개 인물로 확정(모건 블랙잭처럼 姓 공유 일족). 이 경우 개명 불요, 대신 두 인물이 한 씬 공존 시 조직·직함 병기 규칙만 적용.
- **④(옵션 B에서 given name까지 이격하려면)** 마테오 → 마피아/카르텔 톤: **살바토레(Salvatore)** · **도미닉(Dominic)** · **비토(Vito)**.

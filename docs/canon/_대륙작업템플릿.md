# 대륙 캐논화 작업 프롬프트 / 템플릿 (Continent Processing Template)

> **용도**: 5대 대륙 지리 아카이브(01-3~7)를 위키형 캐논으로 재구성하는 **재사용 절차**. 에테르 대륙(01-3)에서 검증 완료. 나머지 4개 대륙은 이 문서를 그대로 따라간다.
> **불변식**: 원본 묘사 **전수 보존(줄이지 말 것)**, 보일러플레이트만 제거. (→ `_작업프롬프트.md` · 입자도 규칙 `_PLAN.md` · 구조 정본 `_구조설계.md`)

## 1. 폴더 구조 템플릿 (원본 01-N-1~7 → 캐논)

```
2-무대/<대륙>.md                     ← 개요 + 하위 색인 (실내용 금지, 디제스트만)
2-무대/<대륙>/
├── 지형.md  + 지형/ (권역 N개)       ← 01-N-1  [엔티티] 개별 파일 + 인덱스
├── 기후생태.md                       ← 01-N-2  [서사]
├── 역사.md                           ← 01-N-3  [서사]
├── 문화종교.md                       ← 01-N-4  [서사]
├── 성지금단.md + 성지금단/ (M개)      ← 01-N-5  [엔티티] 개별 파일 + 인덱스
├── 중립지역.md + 중립지역/ (K개)      ← 01-N-6  [엔티티] 개별 파일 + 인덱스
└── 마법생태.md                       ← 01-N-7  [서사]
```

> 권역/거점 개수(N·M·K)는 **대륙마다 다르다** — §7 대륙별 사전 참조. 절대 에테르 내용을 복붙하지 말 것.

## 2. 작업 절차 (8단계, 에테르 검증판)

1. **구조 스캔**: `find` 로 원본 01-N 전체 트리·파일별 줄수 확인. 권역·성지·중립 개수 파악.
2. **엔티티 슬라이스 ①: 지형** — 원본 지형 N개 정독 → 개별 파일 N개 + `지형.md` 인덱스 작성. (커밋)
3. **엔티티 슬라이스 ②: 성지금단** — 동일. (커밋)
4. **엔티티 슬라이스 ③: 중립지역** — 동일. (커밋)
5. **서사 4종**: 기후생태·역사·문화종교·마법생태 작성. (커밋)
6. **상위 개요**: `<대륙>.md`를 개요+색인으로 정리(하위 4링크), 상세 중복 금지.
7. **nav**: `mkdocs.yml`에 원본 번호 순서(1.지형 2.기후생태 … 7.마법생태)로 등록 + 하위 1-1~ 번호.
8. **게이트**: `python3 /tmp/fixlinks.py` → `mkdocs build` 깨진 링크 0 확인 → **축약 점검**(`python3 scripts/축약점검.py <원본> <캐논>` → 축약의심 0건) → 중복 스캔 → 커밋·푸시.

> 슬라이스마다 커밋해 체크포인트를 남긴다. 인덱스가 링크하는 엔티티는 **같은 슬라이스에서 생성**(dangling 금지).

## 3. 전수 보존 규칙 (축약 금지의 구체 기준)

**엔티티 페이지(지형/성지/중립)에 반드시 보존할 것:**
- `표면적 시각 / 지정학적 이면 / 분위기` **3단 구조** (압축 금지)
- 핵심 랜드마크, 구역별 상세표, 가혹한 환경 특성
- 통치·권력 암투, 서사 후크(탐험/모험/퀘스트/신화/미스터리)
- **핵심 인물 실명** (예: 이단심문관 벨라·정령왕 실바리온) — 익명 직책으로 뭉개지 말 것

**제거할 것(보일러플레이트):**
- G7 블록 `[!CAUTION] 세력의 맹세와 짐` · `[!NOTE] 에픽 섭리` · 에반 개입 당위성 문단
- 도감 경로 위키링크 `[[01-19…|성물]]` → 보통명사 또는 정본 엔티티 링크로
- 자동생성 옴니-링크 쓰레기

## 5. 링크·중복·축약 게이트 (커밋 전 필수)

- **깨진 링크 0**: `mkdocs build 2>&1 | grep WARNING | grep '<대륙>/'` → 비어야 함
- **중복 0**: 상위 `<대륙>.md`가 하위 본문을 전문 복제하지 않는지 grep 대조 (요약표·디제스트는 허용, 전문 복제는 금지)
- **축약 0**: `python3 scripts/축약점검.py "<원본 01-N 폴더>" "docs/canon/2-무대/<대륙>"` → **축약/섹션소실 의심 0건**이어야 함.
  - 보일러플레이트(에반 가호·순리 노트·에픽 콜아웃·도감 위키링크)를 원본에서 차감한 **보정 보존율**로 판정 → 단순 줄수 비교의 오판을 막는다.
  - ≥85% ✅ · 70~85% ⚠️(검토) · <70% ❌(사람이 직접 원본 대조). ❌면 종료코드 1 → 커밋 금지.
  - **헤더 보존 검사(2026-06-20 추가)**: 캐논 섹션 헤더(##/###)가 원본의 50% 미만이면 **❌ 섹션소실** — 줄수는 비슷한데 본문을 *요약 재작성*해 섹션을 통째로 날린 사고(예: 마법협회 위임 1차 아르카디아 헤더 20→4)를 잡는다. 줄수 게이트의 사각지대 보완.
  - **톤 평탄화 주의(수동)**: 게이트가 못 잡는 손실 — 에픽 헤더 영문 병기(`(Epic Phenomena)`)·의도적 수식어(웅장한·장엄한·피비린내 나는) 삭제. 의도된 톤(밝은 에픽 등)을 가진 대륙은 헤더·수식어를 원본과 1:1 대조한다.
  - "미매칭 원본" 목록은 인덱스/개요로 흡수됐는지 수동 확인 (통째 누락 방지 — 과거 17:1 사고 재발 차단).
- **위키링크 방식**: 고유 파일명은 basename `[[성광의-유적]]`, 충돌명(가문·암약조직 등)은 전체경로. `/tmp/fixlinks.py`가 정규화.

## 고정 컨텍스트 (매 Phase 공통 전제)

```
프로젝트: THE FORGOTTEN SUMMONER 설정집 정리
원본: /tmp/tfs/THE FORGOTTEN SUMMONER/ — 읽기전용, 절대 수정 금지
출력: /home/user/canon-forge/docs/canon/
브랜치: claude/fable5-settings-config-ueyi96
목표: 작아지고 모순 0 끝이 있는 캐논. 확장이 아닌 정리.

5대 불변식:
1. 대가 단조누적 — 소환 횟수마다 에반의 대가가 누적, 감소 없음
2. 등급↔대가 정합 — 높은 등급 = 높은 대가, 예외 없음
3. 신화유물 단일소유 — 같은 유물이 두 캐릭터에게 동시 귀속 불가
4. 에반 페르소나 = 본인직접 — 에반이 직접 위장한 것, 무의식 분리 아님
5. 오토마톤 = 빈 껍데기 — 에반 不在 시 단순 인형

원본 묘사 보존 원칙: 줄이지 말 것. 중복·보일러플레이트·레거시만 제거.
에반 개입 당위성 보일러플레이트: 전 문서에서 제거.
여행·모험·탐험 모티프: 각 문서에 자연스럽게 녹여 넣기.
```

## Phase 1 — 시스템

### [1-A] 기준 논의 프롬프트

```
Phase 1 시스템 작성 전 기준 논의.

대상 카테고리:
- 직업체계 (4계급 35종)
- 마법체계 (21계열 × 9서클)
- 등급-대가 정합표 (7~9단계 병렬 계통)

게임 설정집 전문가 관점에서 답해줘:
1. 직업체계 — RPG 클래스 설계 기준으로 볼 때,
   4계급(지배/중산/하층/이단) 구조가 플레이어 선택지로서
   논리적인가? 이단-금지 직업을 어떻게 다뤄야 하나?
2. 마법체계 — 21계열 × 9서클 행렬을 독자/플레이어가
   실제로 파악할 수 있는 구조로 제시하려면 어떤 포맷이
   효과적인가? (전체 표 vs 계열별 딥다이브 vs 대표 예시)
3. 등급-대가 정합 — 소설 서사 관점에서, 등급이 높을수록
   에반의 대가가 커지는 구조가 독자에게 얼마나 긴장감을
   주는가? 이 긴장감을 설정집에서 어떻게 시각화할 수 있나?
4. 여행·탐험 모티프를 시스템 문서에 녹이는 방법 —
   직업이 여정의 동기가 되고, 마법이 탐험의 도구가 되는
   서사적 연결을 어떻게 명문화할까?

이 논의를 바탕으로 Phase 1 작성 기준을 확정해줘.
```

## Phase 2 — 무대

### [2-A] 기준 논의 프롬프트

```
Phase 2 무대 작성 전 기준 논의.

대상 카테고리:
- 5대 대륙 (에테르/크림슨/프로스트/오벨리스크/해양)
- 교통·통신 (육상/해상/공중-미작성/차원-미작성)
- 10대 전쟁사

소설 전문가 관점에서 답해줘:
1. 5대 대륙 — 각 대륙이 "고유 생존 규칙"을 갖고 있다.
   (에테르=포자 위협, 크림슨=산성·적사풍, 프로스트=검은눈보라 등)
   독자가 대륙에 발을 딛는 순간 위험을 체감하게 하려면
   어떤 서술 순서와 포맷이 효과적인가?
2. 공중·차원 교통 공백 — 원본에 폴더만 있고 본문 없음.
   게임 세계관에서 공중/차원 이동 수단이 없으면 어떤
   서사적 문제가 생기는가? 에반의 소환수 비행으로
   대체 가능한가, 아니면 별도 문서가 필요한가?
3. 10대 전쟁사 — 연대기와 중복되지 않으면서
   각 전쟁을 "에반의 탐험 목적지"로 연결하는 방법은?
4. 여행·탐험 모티프 — 대륙 문서가 단순 지리 설명이 아니라
   "여기 왜 가는가"의 서사적 동기를 담으려면?

Phase 2 작성 기준을 확정해줘.
```

## Phase 3 — 사물

### [3-A] 기준 논의 프롬프트

```
Phase 3 사물 작성 전 기준 논의.

- 아이템도감 샘플 존재 (무기 위주)
- 방어구 개별 아이템 사실상 0개

1. 방어구 공백 — RPG에서 방어구가 없으면 전투 시스템이
   성립하지 않는다. 원본에 방어구 묘사가 있는지,
   없다면 ⚠️[추가]로 채우는 게 맞는지 판단해줘.
2. 세트 장비 구조 — 현재 5부위(사자왕)와 4부위(탐험가) 세트가
   있다. RPG 아이템 설계 관점에서 세트 시너지 기술 방식의
   표준 포맷은?
3. 포션·영약 교차참조 — 별도 연금백과 유지 vs 아이템도감 통합
   중 어느 쪽이 설정집으로서 더 일관성 있는가?
4. 탐험 장비 6종 (기존 작성) — 여행 모티프 아이템으로서
   서사적 기능이 충분한가? 추가가 필요한가?

Phase 3 작성 기준을 확정해줘.
```

## 1. 고정 컨텍스트 (모든 작업의 전제)

```
프로젝트: THE FORGOTTEN SUMMONER 설정집 정리
원본: /tmp/tfs/THE FORGOTTEN SUMMONER/ — 읽기전용, 절대 수정 금지
출력: /home/user/canon-forge/docs/canon/
브랜치: claude/fable5-settings-config-ueyi96
최종 목적: 소설 집필 시 AI에게 컨텍스트로 제공 + 개연성 자동 체크

핵심 원칙:
- 작아지고, 모순 0, 끝이 있는 캐논 (확장 아닌 정리)
- 원본 묘사 보존 — 줄이지 말 것. 중복·보일러플레이트·레거시만 제거
- "에반 개입 당위성" 보일러플레이트 전 문서에서 제거
- 여행·모험·탐험 모티프를 각 문서에 자연스럽게 녹여 넣기
- 신은 답하지 않는다 — 갈등의 축은 인간의 이데올로기

5대 불변식 (절대 위반 금지):
1. 대가 단조누적 — 소환 횟수마다 에반의 대가 누적, 감소·회복 없음
2. 등급↔대가 정합 — 높은 등급 = 높은 대가, 예외 없음
3. 신화유물 단일소유 — 같은 유물이 두 캐릭터에게 동시 귀속 불가
4. 에반 페르소나 = 본인직접 — 에반이 직접 위장, 무의식 분리 아님
5. 오토마톤 = 빈 껍데기 — 에반 不在 시 단순 인형, 의지 없음
```

## 2. 전문가 프레임워크 (매 Phase 기준 논의에 적용)

### 역할 A — 시리즈 바이블 편집자
**목적**: 소설 집필 중 빠른 참조가 가능한 구조 확보

판단 기준:
- 작가가 집필 중 3초 안에 원하는 정보를 찾을 수 있는가?
- 각 항목이 "집필 시 참조 가능한 형태"로 서술됐는가?
- 중복 정보 없이 하나의 진실만 존재하는가? (Single Source of Truth)
- 캐릭터·장소·아이템 각각 고유 문서가 있고 교차 참조가 명확한가?

### 역할 B — 스크립트 닥터
**목적**: 개연성 구멍 사전 차단

판단 기준:
- 체호프의 총 원칙: 설정된 모든 요소가 서사에서 실제로 사용되는가?
- 캐릭터 동기가 명확하고 일관된가? ("왜 이 캐릭터가 이렇게 행동하는가")
- 갈등의 축이 명확한가? (신이 아닌 인간의 이데올로기 충돌)
- 브랜든 샌더슨의 마법 법칙:
  - 1법칙: 마법으로 문제를 해결하려면 독자가 먼저 그 마법을 알아야 한다
  - 2법칙: 마법의 한계(대가)가 능력 자체보다 중요하다
  - 3법칙: 기존 요소 확장, 새 요소 추가 최소화

### 역할 C — AI 프롬프트 설계자
**목적**: 완성된 설정집을 AI 컨텍스트로 제공했을 때 정확히 작동하게 구성

판단 기준:
- 규칙이 명제형으로 서술됐는가? ("SSS급 소환 1회 = 에반 수명 1년 차감")
- 모호한 표현이 없는가? ("강하다" X → "SSS급, EX급보다 아래" O)
- 모순이 0인가? (AI는 모순된 컨텍스트에서 반드시 틀린 답 생성)
- 검색 가능한 키워드와 고유명사가 일관된가? (표기 흔들림 제거)
- 각 규칙의 예외가 명시됐거나 "예외 없음"이 명시됐는가?

### 역할 D — 세계관 감수자
**목적**: 내부 논리 일관성 확보

판단 기준:
- 5대 불변식 위반이 없는가?
- 시대 구분이 명확한가? (신들의 시대 / 신들의 종말 / 현재)
- 등급-대가 정합이 전 문서에서 맞는가?
- 지리적 일관성 (5대 대륙의 고유 환경 규칙이 서로 충돌하지 않는가)
- 세력 간 외교·갈등 관계가 논리적 인과관계를 갖는가?

## 3. 작업 방식 (고정)

**매 Phase 3단계:**
1. **[A] 기준 논의** — 4개 전문가 역할 기준으로 해당 카테고리 판단 기준 확정
2. **[B] 실행** — 확정된 기준으로 원본 읽기 → 문서 작성
3. **[C] 검수** — 작성 후 4개 기준으로 자체 검토 → 문제 있으면 수정 후 커밋

## 6. Phase 1 — 시스템

```

[역할 B - 스크립트 닥터]

[역할 C - AI 프롬프트 설계자]

[역할 D - 세계관 감수자]

위 논의로 Phase 1 작성 기준 확정.

## 11. 소설 집필 AI 지원 프롬프트 (설정집 완성 후 사용)

### [소설-1] 개연성 체크

```
설정집 컨텍스트: [docs/canon/ 전체 또는 관련 문서 붙여넣기]

검토 요청:
아래 장면이 THE FORGOTTEN SUMMONER 세계관 규칙과 모순이 없는지 확인해줘.

[장면 붙여넣기]

확인 항목:
1. 5대 불변식 위반 여부
2. 이 대륙/장소의 환경 규칙 위반 여부
3. 등장 캐릭터 행동이 기존 설정과 일관된가
4. 사용된 마법/아이템의 규칙 위반 여부
5. 대가가 제대로 적용됐는가

결과: 통과 / 위반 항목 + 수정 제안
```

### [소설-2] 대가 누적 계산기

```
설정집 컨텍스트: [에반.md + 등급-대가-정합표.md 붙여넣기]

계산 요청:
에반이 [소설 X장]까지 소환한 목록:
1. [영웅명] [등급] — [횟수]회
2. ...

현재 에반의 누적 대가 상태를 계산해줘.
- 수명 차감 총합
- 신체 상태
- 다음 소환 시 임계점 초과 여부
```

### [소설-3] 캐릭터 일관성 확인

```

확인 요청:
[캐릭터명]이 아래 장면에서 이렇게 행동하는 게 기존 설정과 일관된가?

확인 항목:
- 이 캐릭터의 동기·가치관과 일치하는가?

- 기존 묘사된 성격과 충돌하는가?
```

### [소설-4] 세계 규칙 집행

```

## 5. 각 문서 작성 표준

1. frontmatter (tags·aliases·type)
2. 원본 출처 명시
3. 확인된 내용 = evidence 기반
4. 메운 것 = `⚠️[추가]` 표시 (승인 전엔 캐논 아님)
5. 충돌 발견 시 `_collisions.md` 즉시 등재
6. `[[위키링크]]`로 상호 연결
7. 여행·모험·탐험 모티프 1개 이상 포함
8. **규칙은 명제형으로** ("A이면 B다" — AI 집행 가능 형태)
9. "에반 개입 당위성" 보일러플레이트 절대 금지

## 6. 여행·모험·탐험 모티프 가이드 (전 페이지)

| Phase | 녹이는 법 |
|---|---|
| 헌법 | "세계는 넓고 끝을 모른다 — 탐험이 곧 지식" 세계 원리에 명문화 |
| 시스템 | 직업=여정의 동기, 마법=탐험의 도구, 등급=위험도 |
| 무대 | 각 대륙 = "왜 여기 가는가"의 서사적 동기 |
| 사물 | 아이템 로어에 발견·원정 서사, 세트에 "여정 보너스" |
| 인물 | 영웅 서사 = 여정 구조 (떠남·시련·귀환) |

> 톤: 발견의 경이(Wonder), 미지를 향한 떠남. **파멸이 아니라 경이.**

## 7. 5대 불변식 (절대 위반 금지)

1. **대가 단조누적** — 소환 횟수마다 에반의 대가 누적, 감소·회복 없음
2. **등급↔대가 정합** — 높은 등급 = 높은 대가, 예외 없음
3. **신화유물 단일소유** — 같은 유물이 두 캐릭터에게 동시 귀속 불가
4. **에반 페르소나 = 본인직접** — 에반이 직접 위장, 무의식 분리 아님
5. **오토마톤 = 빈 껍데기** — 에반 不在 시 단순 인형, 의지 없음

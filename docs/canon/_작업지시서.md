## 3. 여행·모험·탐험 모티프 가이드 (전 페이지 적용 규칙)

별도 챕터가 아니다. 아래 방식으로 **모든** 캐논 문서에 스며들게 한다.

| 페이즈 | 여행·모험·탐험을 녹이는 법 |
|---|---|
| 사물 | 탐험 도구(나침반·지도석·텐트), 세트 장비에 "여정 보너스", 아이템 로어에 발견·원정 서사 |
| 무대 | 각 지역에 "가는 길/위험/숨겨진 것", 교통=모험 경로, 미답지·유적 탐험 포인트 |
| 인물 | 영웅 서사를 "여정(journey)" 구조로 — 떠남·시련·귀환 |
| 시스템 | 직업에 탐험가/추적자/항해사 계열, 소환·정령을 미지 영역 조사에 연결 |
| 헌법 | "세계는 넓고 끝을 모른다 — 탐험이 곧 지식" 을 세계 작동 원리에 명문화 |

> 톤: 발견의 경이(Wonder), 미지를 향한 떠남, 길 위의 동료. **파멸이 아니라 경이.**

## 4. 각 문서 작성 표준 (고정 템플릿)

1. frontmatter (tags·aliases·type)
2. 원본 출처 명시 (`원본: 경로`)
3. 확인된 내용 = evidence 기반
4. 메운 것 = `⚠️[추가]` 또는 `⚠️[추가 제안]` 표시 (승인 전엔 캐논 아님)
5. 충돌 발견 시 `_collisions.md`에 즉시 등재
6. `[[위키링크]]`로 상호 연결
7. 여행·모험·탐험 모티프 1개 이상 포함

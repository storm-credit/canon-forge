## 0. 왜 이 계획서가 필요한가 (문제 진단)

**범대륙은 끝났다 — 그러나 그건 `01-8 세력 아카이브`(3,903파일)뿐이다.** `_커버리지.md`는 세력 아카이브와 5대륙만 원본-폴더 단위로 추적했고, **`01-31~42`의 백과 12종은 Phase 1·3 "체계 문서"로 흡수만 된 채 전수 소비 여부가 한 번도 검증되지 않았다.**

## 2. 4기둥 (모든 작업 단위에 고정 적용)

범대륙에서 확립한 모델을 그대로 승계한다.

1. **전수 보존** — 원본 묘사 줄이지 않음. 보일러플레이트·레거시만 제거. (입자도 규칙: 시스템=통합 체계 문서+인덱스 / 엔티티=개별 파일+인덱스 / 서사=카테고리 전용 파일)
2. **개연성** — 최종 목적은 "소설 집필 AI 개연성 체크". 개연성 구멍 0. ⚠️ 단 **"Grim Plausibility(잔혹한 개연성)" 보일러 문구는 제거**하고 항목 고유의 에픽 모티프(연성식이면 "진리의 대가·등가의 율법" 등)로 자연화 — 범대륙 무역상단 모델(보일러 130건 제거) 승계.
3. **에픽 판타지 톤** — 드래곤라자·룬의아이들 밀도. 여행·탐험·발견의 경이 모티프 1개+. 신은 침묵(신 중심 구원 서사 금지).
4. **5대 불변식** — ③신화유물 단일소유 ④에반 페르소나=본인직접 ⑤오토마톤=빈껍데기 (①②등가교환 폐기).

**4단계 실행 (매 작업 단위):** `[A] 기준 → [B] 변환(전수 보존 스크립트) → [C] 전수 대조(원본 1:1, 3중 게이트) → [D] 4전문가 검수 → 커밋·푸시`
**4전문가:** 시리즈 바이블 편집자 · 스크립트 닥터 · AI 프롬프트 설계자 · 세계관 감수자.

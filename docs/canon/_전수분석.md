> [!NOTE]

### 품질·정리
- 실내용 79.7% / 인덱스 20.3% / 빈 파일 0

## 2. 마법 백과 — 1,866파일

**21계열 × 9서클(클래스) 완벽 행렬.** 실내용 99%, 중복 0.16%.

### 구조
- **원소 마법계 10계열**: 화염·수계·빙결·대지·기류·뇌전·생태·금속·빛·어둠 (각 90개)

- 기초 이론 8문서 (마력 흐름·마법과 신·마법사 등급 1~9클래스 등)

### 대가 규칙
**클래스 문서에 체계적으로 명시.** 1클래스 "영혼의 소외감"부터 9클래스까지 단조누적. 4대 근원 기둥(크로노스=영혼 마모, 오르도스=환통, 아나테아=명줄, 네메시스=고독)이 대가의 신화적 근거.

### 정리
- **1,850 → ~318개로 압축 가능 (81% 감축)**: 기초이론 8 + 계열별 대표마법 ~300 + 대가 메타규칙 10
- 통째 수록 ❌ → 계열별 대표 추출 + 전체는 부록 인덱스

## ★ [D] 개연성 심층검수 + 정본화 전환 (2026-06-21)

### 적발 — [D]를 표면 체크로 갈음한 문제

그간 [D] 4전문가 검수를 grep 키워드 확인(갈등축·에피그래프 존재 여부)으로 갈음했고,
실제 **본문 정독 + 모순 해소**는 하지 않았다. 오벨리스크 4세력을 검수 에이전트 4개로
**전문 정독**시킨 결과, 4세력 모두 **치명적 구조 모순**이 드러났다.

**공통 패턴 — 각 세력이 "양립 불가능한 두 정체성"으로 분열:**
| 세력 | 정체성 A (개요·장소) | 정체성 B (서사 문서) |
|---|---|---|
| 심연군단 | 12군주·심연심·다크니스 하트 | 아자토르 신앙·검은우물 교단·암흑의 심장 |
| 망자의왕국 | 기억 경제·칠관의 궁정·네크로폴리스 프라임 | 사후 노예제·시체매매·사자의 수도 |
| 잊힌자연합 | 망명자 네트워크 | 혈마법 고대 도굴 학자 부족 |
| 봉인수호단 | 비극적 버려진 수호자 | 광신적 학살·갈취 컬트 |

**4세력 전부** `주요인물.md`가 개요 핵심 인물과 **교집합 0**인 딴 명단을 담음.

### 근본 원인 — 원본 유래 모순 (변환 탓 아님)

원본 직접 대조로 확인: 네크론/네크로스(역사 원본), 수도명(원본 '사자의 수도' 9파일
vs '네크로폴리스 프라임' 4파일), 48h/72h(원본에 둘 다) 등 **모순이 원본에 이미 존재**.
원본이 여러 출처에서 독립 생성된 문서를 한 폴더에 모은 것이라 자기들끼리 안 맞고,
**"전수 보존(원문 그대로)" 정책이 그 모순까지 충실히 보존**한 결과.

### 정책 전환 — 원문 보존 → 정본화 (사용자 결정 2026-06-21)

원본 모순 vs "모순 0"(AI 개연성 체크 도구라는 최종 목적) 충돌. 사용자가 **정본화(모순
해소) + 전 18세력 재검수** 결정. 톤은 **세력마다 판단**(심연군단=어두운 원문 톤 유지).

**정본화 원칙:**
1. 개요(`세력명.md`)가 상위 정본 — 하위 문서가 충돌하면 개요에 맞춤
2. 톤은 해당 세력에 맞는 것을 살림(순화 금지). 구조적 사실만 통일
3. 개요에 없는 정보는 보존(보강), 충돌만 수정. 최소 수정
4. 창작적으로 갈리는 중대 결정은 사용자 확인

> **검증 방법 변경**: 정본화하면 `원본대조.py`가 의도적 차이(원본 '암흑의 심장' →
> 정본 '다크니스 하트')를 "누락"으로 표시한다. 정본화 후 항목대조는 "누락 0"이 아니라
> **"남은 누락이 전부 의도된 정본화 차이인가"**로 판정한다.

# 원본 전수 분석 (Full-Vault Analysis)

> 2026-06-15 · 읽기 에이전트 5개 병렬 분석 · 원본 **8,953파일 100% 커버**  
> 목적: "내가 말한 것만"이 아니라 **설정집 전체**가 뭔지 파악. 정리하면 몇 개로 주는지 산출.

> [!NOTE]
> **역사적 기록**: 이 문서는 원본을 *있는 그대로* 분석한 스냅샷이다. 따라서 등가교환·대가 관련 서술은 원본(Hanesis Protocol 주입 상태)을 반영한다. 2026-06-16에 등가교환 레이어가 캐논에서 제거됐으므로, 이 분석의 대가 관련 항목은 *원본의 기록*일 뿐 현행 캐논 규칙이 아니다.

## 0. 최상위 발견 — 구조의 진실

| 발견 | 내용 | 의미 |
|---|---|---|
| **크로니클은 1개뿐** | "6대 크로니클"은 `00. 세계의 틀`의 *구상*. 실제 콘텐츠는 전부 **아스트라리스 1개**에 집중 | 48,000화는 계획, 설정집 대상은 아스트라리스 |
| **3대 거인 = 전체의 81%** | 세력 3,903 + 마법 1,866 + 영웅 1,491 = 7,260파일 | 이 셋의 압축이 정리의 핵심 |
| **품질은 전반적으로 높다** | 빈 껍데기 비율이 낮음(마법 0%, 세력 20%, 영웅 2%). 수작성 다수 | "내용은 보존, 중복·껍데기·레거시만 제거"가 맞는 전략 |
| **파일 팽창의 정체** | 일관된 템플릿(세력 14항목, 마법 21계열×9서클)을 기계적으로 적용한 결과 | 실제 창작 단위는 훨씬 적음 |

## 1. 세력 아카이브 — 3,903파일

**실제 고유 세력 ≈ 73~80개.** 각 세력이 14항목 템플릿(장소·군사·정치·경제·종교·문화·인물·역사·외교 등)으로 40~65개 문서를 갖는다.

### 구조
| 권역 | 세력 수 | 대표 세력 |
|---|---|---|
| 에테르 대륙 | 5 | 성국 · 왕국연합 · 자유도시연합 · 은반의 마탑 · 정령연합 |
| 크림슨 대륙 | 5 | 솔라리안 제국 · 아르카나 잔재 · 용의 후예 · 붉은 사막 부족 · 잊혀진 서고단 |
| 프로스트 대륙 | 4 | 딥 포지 왕국(드워프) · 프로스트본 연합 · 빙하의 신전 · 그림자 항구 |
| 오벨리스크 대륙 | 4 | 심연 군단 · 망자의 왕국 · 섀도우 워커 · 블러드 문 |
| 해양 대륙 | 3 | 황금 함대 · 자유 해방 연맹 · 바다의 교단 |
| **범대륙 초국가** | **52** | 길드6·상단10·용병13·마공연맹9·금융9·학술6·첩보6 (7유형) |

### 품질·정리
- 실내용 79.7% / 인덱스 20.3% / 빈 파일 0
- **정리 대상**: `_Legacy_중립세력` 백업 **666파일** (현재 구조와 ~50% 중복) → 격리. 오류 폴더 1개.
- **공백**: 해양 대륙이 가장 부실(3세력 190파일). 대륙 간 외교 관계 문서 부족.
- **흥미**: 거의 모든 세력이 "우주적 십자가" 보유 — 겉은 번영, 속은 하층민 생명력을 연료로 굴러가는 구조적 저주. ("신은 답하지 않는다"가 세력 단위로 구현)

## 2. 마법 백과 — 1,866파일

**21계열 × 9서클(클래스) 완벽 행렬.** 실내용 99%, 중복 0.16%.

### 구조
- **원소 마법계 10계열**: 화염·수계·빙결·대지·기류·뇌전·생태·금속·빛·어둠 (각 90개)
- **특질/초상 마법계 11계열**: 고대·심연·시공간·그림자·꿈·텔레파시·환각·맹독·강령·혈액·차원 (각 88~91개)
- 기초 이론 8문서 (마력 흐름·마법과 신·마법사 등급 1~9클래스 등)

### 대가 규칙
**클래스 문서에 체계적으로 명시.** 1클래스 "영혼의 소외감"부터 9클래스까지 단조누적. 4대 근원 기둥(크로노스=영혼 마모, 오르도스=환통, 아나테아=명줄, 네메시스=고독)이 대가의 신화적 근거.

### 정리
- **1,850 → ~318개로 압축 가능 (81% 감축)**: 기초이론 8 + 계열별 대표마법 ~300 + 대가 메타규칙 10
- 통째 수록 ❌ → 계열별 대표 추출 + 전체는 부록 인덱스

## 3. 영웅 백과 — 1,491파일

**3분류:** 성장영웅 115(과거/멸망) · 현존영웅 1,211(현재) · 소환영웅 165(등급별). 실내용 98%.

### 핵심: 대가는 영웅이 아니라 에반이 짊어진다
| 영웅 | 등급 | 대가(에반에게) |
|---|---|---|
| 가렌 발렌타인 | SSS소환 | 받는 피해 절반이 에반에게 환상통 |
| 레온하르트 에르드반 | SSS소환 | 시간 1초 조작 = 에반 수명 1년 |
| 루시엔 | 기원급 | 소환 후 에반 7일 실명 |

### 에반 페르소나 중복 (O1 부분 해소)
- 9페르소나가 영웅 백과에 **광범위 중복되진 않음.** 1,200 현존 영웅은 진짜 NPC.
- 단 **레온하르트만 모호**: 카멜로스(현존 왕) vs 에르드반(소환·과거) — 이름·맥락 달라 별개일 가능성 높음.
- **해결책**: 프론트매터 `evan_persona: true/false` 태그.

### 소환 등급 체계
EX(초월) · SSS(신화) · SS(전설) · S(영웅) · A(정예) · B · C · D · 기원 · 망각 · 이계 · 특수 · 군단급

### 정리
- `_Legacy` 백업 **317파일** 격리. 로마자 표기 흔들림(카멜로스/에르드반 등) 정규화.

## 4. 시스템 백과 — 약 640파일 (10종)

**직업 체계가 모든 시스템의 기준축.**

### 직업 (40파일)
4계급: ① 지배(마법사·성기사·사제·길드마스터) ② 중산(베테랑 용병·연금술사·무역상) ③ 하층(도적·하급 사냥꾼·미등록 마법사) ④ 이단-금지(심연 사도·강령술사 — 화형 대상). 직업 35종. **직업↔신앙 불가분.**

### 모든 마법계가 평행 등급 (정합성 매우 높음)
| 계통 | 등급 | 파일 |
|---|---|---:|
| 마법사 | 9서클 | 1866 |
| 연금 | 7대 진리 | 212 |
| 정령 | 공명도(20정령왕) | 212 |
| 소환 | **7조약** | 27 |
| 주술 | 7주계 | 27 |
| 마공학 | 7기어 | 24 |
| 신성 | 7위계 | 21 |
| 차원·공간 | 4층위 | 5 |

### 핵심 발견
- **소환 백과가 최상급 품질** — 7조약마다 대가 누적이 극상세(7단계 = 영혼 코어 소멸 + 세계관에서 영구 삭제). 에반 전용 소환 메커니즘 별도 문서.
- **동반수만 유일하게 대가 없음** — 계약 아닌 생물학적 유대. → 소환수(이계·계약) / 동반수(현세·무계약) / 몬스터(야생) 셋이 명확 구분, 중복 0.
- 연금·정령 212개씩은 자동생성 아닌 **수작성** (고유 부제명·일관된 문체).

## 5. 무대 — 약 210파일 (대륙·교통·전쟁사·인물)

### 5대 대륙 (각 고유 생존 규칙)
| 대륙 | 환경 위협 | 경제 | 완성도 |

| 크림슨(붉은 황무지) | 적사풍·산성 호수·마나 무덤 | 타락 마석·목숨값 | 7/10 |
| 프로스트(얼음의 땅) | 검은 눈보라·서리 독기 | 화로 경제 | 6.5/10 |

### 교통·통신
- **통신**: 대마탑 마석 광학 중계망(공식 고속) + 6종(세계수 지맥·심해 텔레파시·성국 뇌파 등). 에반은 비행 소환수로 광선 가로채기.
- **교통**: 육상(마력 열차·골렘 마차·4대 도로) / 해상(4대 항로·5종 선박·4대 항구) / **공중·차원·교통정치 = 폴더만 있고 본문 미작성(공백)**

### 전쟁사 (10대 전쟁)
창세 대전 → 고대문명 대분열 → 운명의 전쟁 → 엘프 전쟁 → 바다 성전 → 용의 전쟁 → 왕국연합 통합 → 마도제국 멸망 → 그림자 첩보 대전 → 황금 전쟁. 각 전쟁에 에반 연계 탐험 퀘스트 존재.

### 인물 백과 (01-15, 영웅 백과와 별개)
- 주인공 일행: 에반 · 루미에(빛의 요정) · 히로인 · 펫
- 에반의 9대 위장 페르소나 (별도 폴더)
- 주요 인물(Key Figures)
- **차이**: 01-15=주인공 중심 인물지 / 01-14=세력 리더십. ⚠️ 경계 일부 모호.

### 여행·탐험 (이미 풍부함 — 신규 창작 아님)
성지/유적 15+ · 중립 안전거점 10+ · 차원 결함(크림슨)·기억의 우물(오벨리스크) 같은 특수 경험 메카닉. → 내가 따로 "탐험 장비"를 추가했지만, **원본 자체가 이미 탐험 서사로 가득**. 정리 시 이걸 전면에 끌어내면 됨.

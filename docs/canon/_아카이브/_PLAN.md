### 귀속 메모 (경계가 헷갈리는 것)

- **신앙/종교** = 2층: *체계*(신앙 체계·창세 신화)는 **Phase 0**, *마법으로서의 신성력*(신성 백과)은 **Phase 1**, *세력별 종교 운용*은 **Phase 2** 세력 아카이브.
- **교통/통신** = 마도 교통·통신 → **Phase 2** 무대(인프라).
- **정령/연금** = 마법 체계 일부 → **Phase 1**. (정령 212개 개별 엔티티는 인덱스식 처리)
- **몬스터/동반수** = **Phase 3** 사물로 묶되, 서사 비중 큰 개체는 Phase 4 교차.
- **전쟁사** = **Phase 2** 무대(세력 간 역사).

## 산출물 입자도 규칙 — 폴더링 vs 통합 (2026-06-19 신설)

> **배경**: 초기 "아카이브 1개 → 요약 1문서" 모델이 `_작업프롬프트.md`의 **"줄이지 말 것"** 불변식을 어기고 엔티티·서사를 축약했음(세력 서사·대륙 지리·창세신화 신 목록 등). 아래 3분류로 입자도를 확정한다.

| 분류 | 판정 기준 | 처리 | 예시 |
|---|---|---|---|
| **엔티티** | 원본에 **고유명사로 명명된 객체가 3개 이상** 나열 | 엔티티당 **개별 파일** + 인덱스(.md) | 도시·성지·요새·가문·기사단·암약조직·상인협회 · **신(神)** · **대륙 지형 지역**·성지금단·중립지역 |
| **시스템·규칙** | **수십~수천 개 동종 항목**으로 이루어진 규칙 체계 | **통합 체계 문서** + (필요시 인덱스). 개별 파일 **금지** | 마법 1,866 · 연금 212 · 정령 212 · 직업 35 |
| **서사·설정** | **1~2개 산문** 문서 (관계·연표·제도) | **카테고리 전용 파일**로 풀 보존(축약 금지) | 외교·역사·법률·사회정치·종교문화·예술건축·의복·생활양식 |

**상위 문서 역할 축소**: `<세력>.md`·`<대륙>.md`·`창세신화.md` 등은 **순수 개요 + 색인**으로만. 실내용은 하위 엔티티/카테고리 파일이 보유.

**적용 순서**: 의존성 원칙대로 **Phase 0 → 1 → 2** 순으로 재감사. 각 Phase에서 위 표로 분류 재판정 후, 축약된 엔티티/서사를 복원.

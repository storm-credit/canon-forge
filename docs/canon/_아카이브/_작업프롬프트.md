## 1-B. 서사 구조 & 톤 레퍼런스 (창작 방향 — 고정)

> 사용자 확정 (2026-06-17). 모든 캐논 문서 작성·검수 시 이 톤과 구조를 기준으로 한다.

### 톤: 에픽 판타지

레퍼런스 작품 (분위기·문체·세계 밀도 참조):
- **드래곤 라자** (이영도) — 서사시적 세계관, 서술자의 관조적 유머
- **룬의 아이들** (전민희) — 촘촘한 복선·캐릭터 군상극, 감정선의 깊이
- **태양의 탑** (전민희) — 문명 간 충돌, 권력과 이상의 대립
- **비상하는 매** — 성장영웅의 고독한 여정
- **하얀 늑대들** — 전쟁 서사, 군상극, 회색 도덕

### 서사 구조: 이스이터널식 Act 완결 — 총 39 Act / 48,000화

```
크로니클 1개 = 독립 시리즈 = 6~7개의 Act로 구성
각 Act = 독립 완결 (이스 이터널식)
→ 어느 Act에서 유입해도 독자가 진입 가능

6대 크로니클 전부에 Act가 있다:
  아스트라리스  Act 1~6  (6)
  아포크립토스  Act 7~12 (6)
  천극          Act 13~18 (6)
  크로노스 강철 Act 19~25 (7)
  에테리온      Act 26~32 (7)
  베리타스      Act 33~39 (7)
  ──────────────────────────
  합계          Act 1~39  / 총 48,000화
```

**핵심 원칙**:
- 6대 크로니클이 6개 Act가 아니다. **각 크로니클 내부에 6~7개 Act**가 있다 (총 39 Act).
- 아스트라리스만이 아니라 **모든 크로니클**이 다수 Act를 가진다 — 각 Act가 독립 완결되면서 전체 아크를 관통한다.
- **영웅전설·이스이터널** 참조: Act별 결말이 있어 언제 어느 이야기에 들어와도 독자 유입 가능
- 전체 구성표: [[0-헌법/크로니클-로드맵]]

### 캐릭터 구조: 성장영웅 스핀오프

- **한백무림서** 참조: 하나의 세계관에서 다른 주인공의 이야기가 별도 편으로 파생 가능
- 에반이 중심이되 다른 캐릭터의 성장 서사도 독립 스핀오프로 전개 가능한 구조

### 에반 분신체 운용: 약먹는 천재 마법사 참조

- **약먹는 천재 마법사** 참조: 분신체를 활용한 다중 활동·정보 수집·동시 전개
- 에반의 오토마톤(빈 껍데기)·페르소나(본인 위장)가 이 구조를 세계관에 맞게 변주

### 서사 금기: 신 대립 회피

- 다른 소설들이 마지막에 매번 신과 대립하면서 루즈해지는 패턴 거부
- **갈등의 축 = 인간의 이데올로기 충돌** — 신은 배경이지 최종 보스가 아니다
- 신들의 종말 이후 신은 부재/침묵 → 모든 선택과 결말은 인간의 것

## 2. 전문가 프레임워크 (매 Phase 기준 논의에 적용)

### 역할 A — 시리즈 바이블 편집자
**목적**: 소설 집필 중 빠른 참조가 가능한 구조 확보

판단 기준:
- 작가가 집필 중 3초 안에 원하는 정보를 찾을 수 있는가?
- 각 항목이 "집필 시 참조 가능한 형태"로 서술됐는가?
- 중복 정보 없이 하나의 진실만 존재하는가? (Single Source of Truth)
- 캐릭터·장소·아이템 각각 고유 문서가 있고 교차 참조가 명확한가?

### 역할 B — 스크립트 닥터
**목적**: 개연성 구멍 사전 차단

판단 기준:
- 체호프의 총 원칙: 설정된 모든 요소가 서사에서 실제로 사용되는가?
- 캐릭터 동기가 명확하고 일관된가? ("왜 이 캐릭터가 이렇게 행동하는가")
- 갈등의 축이 명확한가? (신이 아닌 인간의 이데올로기 충돌)
- 브랜든 샌더슨의 마법 법칙:
  - 1법칙: 마법으로 문제를 해결하려면 독자가 먼저 그 마법을 알아야 한다
  - 2법칙: 마법의 한계(대가)가 능력 자체보다 중요하다
  - 3법칙: 기존 요소 확장, 새 요소 추가 최소화

### 역할 C — AI 프롬프트 설계자
**목적**: 완성된 설정집을 AI 컨텍스트로 제공했을 때 정확히 작동하게 구성

판단 기준:
- 규칙이 명제형으로 서술됐는가? ("SSS급 소환 1회 = 에반 수명 1년 차감")
- 모호한 표현이 없는가? ("강하다" X → "SSS급, EX급보다 아래" O)
- 모순이 0인가? (AI는 모순된 컨텍스트에서 반드시 틀린 답 생성)
- 검색 가능한 키워드와 고유명사가 일관된가? (표기 흔들림 제거)
- 각 규칙의 예외가 명시됐거나 "예외 없음"이 명시됐는가?

### 역할 D — 세계관 감수자
**목적**: 내부 논리 일관성 확보

판단 기준:
- 5대 불변식 위반이 없는가?
- 시대 구분이 명확한가? (신들의 시대 / 신들의 종말 / 현재)
- 등급-대가 정합이 전 문서에서 맞는가?
- 지리적 일관성 (5대 대륙의 고유 환경 규칙이 서로 충돌하지 않는가)
- 세력 간 외교·갈등 관계가 논리적 인과관계를 갖는가?

## 3. 작업 방식 (고정) — 4단계 필수

> **원칙**: 구조(뼈대)만 맞추는 게 아니라, 원본의 **모든 고유명사·표·메커니즘·수치·인물**이 캐논에 반영됐는지 전수 대조한다. 이 단계를 건너뛰면 안 된다.

**매 Phase / 매 문서 4단계:**

1. **[A] 기준 논의** — 4개 전문가 역할 기준으로 해당 카테고리 판단 기준 확정
2. **[B] 실행** — 확정된 기준으로 원본 읽기 → 문서 작성
3. **[C] 전수 대조 (Hanesis 검수)** ★신규
   - 캐논 문서의 `출처:` 프론트매터에 적힌 원본 파일을 **전부** 연다
   - 원본 파일 각각에서 **고유명사·표·메커니즘·의식·인물·수치·장소·유물**을 추출한다
   - 캐논 문서에 해당 항목이 **있는지/없는지/틀렸는지** 1:1 대조한다
   - 결과를 **누락 대장(ledger)**으로 기록:
     ```
     | 원본 파일 | 항목 | 캐논 상태 | 조치 |
     |---|---|---|---|
     | 01-10/교단명.md | 태양 교단 세례 의식 3종 | ❌ 누락 | 보강 |
     | 01-1/창세전쟁.md | 카이론의 사냥 | ❌ 누락 | 보강 |
     | 01-20/직업.md | 환술사 직업 조건 | ✅ 있음 | — |
     ```
   - 누락이 **의도적 생략**(보일러플레이트·Hanesis 주입물·레거시)이면 사유를 명시
   - 누락이 **실수**면 즉시 보강
4. **[D] 4전문가 검수** — 보강 후 A·B·C·D 4개 전문가 관점으로 최종 검토 → 문제 있으면 수정 후 커밋

> **[C] 전수 대조는 기존 완료 문서에도 소급 적용한다.** Phase 0~2 기완성 문서도 이 절차를 거쳐야 "검수 완료"다.

### 전수 대조 산출물 위치

- 문서별 누락 대장: `docs/canon/_전수분석.md` 에 누적
- 보강 후 `_progress.md`에 "✅ 전수 대조 완료" 표시

## 6. Phase 1 — 시스템

### [1-A] 기준 논의 프롬프트

```
Phase 1 시스템 작성 전 기준 논의.

원본 규모:
- 직업 백과: 40파일 (4계급 35종 직업)
- 마법 백과: 1,866파일 (21계열 × 9서클 완벽 행렬)

[역할 A - 시리즈 바이블 편집자]

- 4계급(지배/중산/하층/이단) 구조에서 이단-금지 직업의 서사적 역할은?

[역할 C - AI 프롬프트 설계자]
- 21계열 × 9서클 행렬을 AI가 참조 가능한 형태로 구조화하면?

- 직업-마법-등급 간 관계를 명제형으로 서술하는 방법은?

[역할 D - 세계관 감수자]
- 전 마법계 병렬 등급 정합 확인:
```

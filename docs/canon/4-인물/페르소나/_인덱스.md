# 에반의 9대 위장 페르소나

> 에반이 세계 각 세력에 침투하고 흑막으로 활동하기 위해 유지하는 **9개의 독립된 위장 신분**. 각 페르소나는 에반이 직접 변장·조종하는 본인의 자아다 — 무의식 껍데기가 아니다.

---

## 구조 원칙

- **페르소나 = 에반 본인**: 에반이 직접 위장 매개체(유물)를 사용해 외모·목소리·마력 파동을 전환. 자아와 판단은 에반의 것.
- **오토마톤(더미) = 빈 껍데기**: 페르소나로 외출한 동안 본진(숙소·사교계)에 남아 에반 일상을 연기하는 의식 없는 골렘/호문쿨루스.
- **동시 등장 가능**: 진짜 에반(페르소나)과 가짜 에반(오토마톤)이 같은 공간에서 교차하거나 대화하는 연출은 모순이 아님.
- **멀티태스킹 부담**: 오토마톤을 원격 동기화하면서 페르소나 전투를 수행해야 하므로, 에반은 극심한 마력 과부하(코피·편두통)를 감당한다.

## 페르소나 간 관계망

```
레온하르트(01) ←[암시장 앙숙]→ 카일락(02)
카일락(02) ←[시체 정보 제공]→ 세드릭(03)
세드릭(03) ←[정보 공유]→ 에리히(07)
단테스(08) ←[군수 지원]→ 렉시온(06)
루시우스(05) ←[신성 권위]→ 빈센트(04) [상호 묵인]
파우스트(09) ←[장비 수리 의존]→ 렉시온(06)
```

> 이 관계망은 에반이 의도적으로 설계한 것 — 각 페르소나가 별개 인물처럼 보이면서도 서로 기능적으로 연결되어 대륙 전체 네트워크를 에반 1인이 장악하는 구조.

## ⚠️ 미해결 (O1)

영웅 백과(01-14) 현존 영웅 1,211명 중 위 9명과 **이름이 같은 독립 인물이 등재되어 있는지** 교차 확인 필요. 전수분석 결과 광범위한 중복은 없음. 단 레온하르트의 경우:

- **레온하르트 카스테로** (페르소나 01) = 에반 본인
- **레온하르트 에르드반** (영웅 백과 01-14, SSS급 소환 영웅) = **별개 인물**

두 레온하르트는 이름만 비슷할 뿐 무관. → [O5]

# 이름 없는 소환사

> **전제(엄수)**: 아래는 원고가 실제로 보여 준 사실만 등재한다. 이 인물은 **에반과 같은 방식(옆자리 허공에 아무도 못 볼 만큼 얇게 형체를 세우는 소환)을 쓰는 정체·소속 미공개 소환사**이며, 에반의 코트 속 조각들(메인 플롯 직결)을 정면으로 겨냥한다. 정체·소속·목적은 **미공개**이고, 아래 §봉인 정합 주의를 반드시 함께 읽는다.

> [!warning] 봉인 정합 주의 (엄수)
> ① **정체·이름·소속은 전면 봉인**이다 — 원고 053~059화 내에서 그의 이름은 끝내 밝혀지지 않으며("이름은 아직 못 얻었습니다만", 055·056화), 소속은 "성국은 아니"라는 배제 정보(056화)만 존재한다.
> ② 상위 지시의 인물 규정 표현 **"손목 문양 파편 보유자"**는 053~059화 원고에 **문자 그대로는 나오지 않는다.** 원고 실측으로 확인되는 것은 (a) 그가 에반과 동형의 얇은 형체 소환을 쓴다는 점, (b) 062화에서 그의 표식으로 보이는 문양 — **"가느다란 원 안에 선 하나를 그은 문양"**(저울 문양과 별개)이 봉투 안감에서 발견된다는 점, (c) 059화에서 그가 에반의 **여덟 조각(파편)**을 "전부 파악을 끝냈"다고 통고한다는 점뿐이다. '손목 문양'이라는 신체 표식·'파편 보유' 여부는 **원고 미확정 → 작가 판단**으로 남기고, 이 문서에서 신체 표식을 단정하지 않는다.

## 1. 원고가 보여 준 사실

- **첫 조우(055화 말미~056화)**: 에반 일행 여관 아래층 복도. 스물 후반, 등불 아래 푸르스름할 만큼 창백한 젊은 남자. 옆 허공에 얇게 걸친 형체를 두고 있다가 "들켰네요" 하며 흩어 버림 — 루미에 평 "너랑 똑같아. 부르지도 않았는데 옆에 있다가, 부르지도 않았는데 사라져"(056화).
- **거울상 화법**: 정보를 반쪽만 내주고, 그 반쪽조차 상대가 이미 아는 만큼만 확인시켜 주는 방식 — 에반과 정확히 같은 수. "마주 앉은 게 아니라 거울을 보는 기분"(056화). 자기도 "대부님 밑에서 이름 없이 굴러다니는 처지"라 밝힘.
- **넷째 깃발(056화)**: 자신이 앉을 판돈이 "성국은 아니"라고만 밝힘 → 에반의 셈판에 성국·유물 상단·에반 자신 외 **넷째 깃발(이름도 문양도 모르는)**이 추가됨.
- **판의 촉매(057화)**: 대국에 참관인으로 있다가 베로니카에게 귓속말 → 가르나크 급전·판 중단을 촉발. 나가며 에반에게 고개를 까딱 — "너도 이 판 안에 있다".
- **이름을 판돈으로(058화)**: 마르코를 통해 **라인하르트(에반)의 이름을 다음 판 보증으로 걸고 싶어 함.** 마르코는 그를 "저처럼, 어느 깃발도 안 든 사람"으로 규정.
- **진짜 노림수(059화)**: 에반의 창을 골목에서 올려다본 뒤, 문틈으로 쪽지 — 위원 억류로 대국이 열리지 않게 되었는데도 "이름은 그대로 걸려 있음". 말미 필압 깊은 한 줄: **"당신 코트 속의 그것들, 이제 전부 파악을 끝냈습니다."** 에반의 여덟 조각을 정확히 겨냥한 울림이 조각들을 요동치게 함 → 대국은 미끼, 목적은 에반을 도시에·이름에 묶어 두는 것 자체였음이 드러남.
- **표식(062화)**: 베로니카의 저울 문양 봉투 안감 접히는 자리에 얕게 눌린 다른 문양 — "가느다란 원 안에 선 하나를 그은 문양". 저울 문양 사내에게 들키지도 않고 봉투 배달 중간에 개입.

## 2. 서사 위치 · 관계

- **에반(메인플롯 직결)**: 에반과 **동형의 소환 방식**을 쓰는 유일 확인 인물이자, 에반의 파편을 직접 겨냥한 최초의 외부 세력. 소환 원리·파편의 정체(FS 봉인 축)와 직접 얽히는 메인 플롯 매듭.
- **[[마르코블랙잭|마르코 블랙잭]]**: 그를 판에 끌어들인/그의 판돈을 중개한 대부. "어느 깃발도 안 든 사람"이라는 규정을 공유.
- **[[베로니카|베로니카]]**: 057화 그의 귓속말이 판 중단을 촉발. 062화 그의 표식이 그녀의 봉투에서 발견됨.
- **[[낯선사내|낯선 사내]](12~26화)와의 연결 — 미확정**: 두 인물 모두 **"어느 깃발도 안 든다"**로 규정된다(낯선 사내: 013·015화 / 이 인물: 058화 마르코의 규정). 동일 인물인지, 같은 근원인지는 **원고 미확정 → 작가 판단**. 단정하지 않는다.

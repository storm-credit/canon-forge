# 종족·혈통 대장 (Race & Bloodline Harvest Ledger)

> **역할**: 종족·혈통 정본 신설을 위한 **모으기(harvest) 결과 대장**. 캐논 전체(204+파일)에 흩어진 종족·혈통 사실을 출처째 긁어모은 1차 수확물이다. **아직 정본 아님 — 검토용.**
> **방법**: 4영역 병렬 정독(① 헌법/기원 ② 대륙/분포 ③ 혈통 시스템 ④ 인물/가계). 읽기 전용, 원본 무수정.
> **다음 단계**: 이 대장 검토 → 모순 판정 + 공백 정책 확정 → 종족별 정본 문서 합성([축약 없이·개연성·우리 톤]).
> 작성: 2026-06-22

## B. 종족 ↔ 대륙 ↔ 세력 매핑

| 대륙 | 종족 구성 | 갈등 축 | 연결 세력(기존 캐논) |
|---|---|---|---|
| **에테르** | 인간 + 엘프/하이엘프 + 정령 | 인간 ↔ 정령·엘프 적대 | 성국·왕국연합·자유도시·마법협회 / 정령연합 |
| **크림슨** | 인간 + 용의 후예(반인반룡) + 붉은사막부족 | 용혈 순혈주의 / 사막 약육강식 | 솔라리안제국·아르카나잔재·잊혀진서고단 / 용의후예 / 붉은사막부족 |
| **프로스트** | 드워프 + 프로스트본 + 설인 + 빙룡 + 거인족(고대) | 드워프 전통파↔개혁파 | 프로스트본 연합 / 딥포지왕국 |
| **오벨리스크** | 언데드 + 인간(하층) + 잊힌 자들 | 마력=신분 사령 신분제 | 망자의 왕국 |
| **해양** | 인간 + 인어/해인/사이렌/수인 + 엘프 소수 | (해양 4종 경계 모호) | 해적연합·황금함대·바다의교단 |

> **핵심**: 종족 정본 문서는 위 세력 캐논(이미 완료)을 **역참조**해야 중복·모순이 안 생긴다. 종족=세력 정체성과 직결(용의 후예 세력=반인반룡 종족, 프로스트본 연합=혼성 종족).

## D. 혈통(Bloodline) 제도

### D-1. 4계급 + 세습 권력
- **4계급**: ①귀족·지배층(신성력·마력 독점) ②중산층 ③하층민·아웃로 ④이단·금지 (직업체계.md:30-37)
- **세습 = 절대 권력 직업 통로**: "마력 자원 + 신성한 핏줄"이 직계 가문 내 대물림 (직업체계.md:250-252)
- **혈통→직업 게이트**: 순수 혈통 귀족 자제만 왕실 기사·성기사 입교(:256); 드워프 명가만 룬 대장장이 세습(:257)

### D-2. 혈통각인 (철의금융 — 혈통을 금융 권력으로)
- **혈통 각인 = 자본 접근권·마나 채권 발행 자격 암호 키** (단순 신분증 아님). 룬 기록소가 원본 규격 보유 (파벌.md:38-44)
- **3대 집행기구**: 장미 십자단(혈통 판독·이단자 색출, [혈통 봉인]) / 성좌의 파수꾼(혈통 계승 예측, [성좌 혈도]) / 잿빛 그림자(이단자 암살·각인 소각 [재화 소각])
- **3대 파벌**: 잿빛 의회(혈통 가중치 표결) / 진리 탐구회(족보 위조 추적) / 룬 기록소(각인 원장)

### D-3. 후천 혈통 변이 (용혈 — 선천 세습과 별개 축)
- **용혈의 사도**: 고룡 피 마시고 비늘 변이 (신앙체계.md:205)
- **용석 이식 시술**: 죽은 용 마력핵·골수를 전사 혈맥에 생주입 → 십중팔구 사망, 생존 시 반인반룡 (용의후예/마법체계.md:18,107)
- **부작용**: 비늘갈이 증후군·수혈 강박·화석화, 숨결술사 30세 미만 사망. **대량 실패 사례**: 스칼렛 몰트(용혈 식수 투입 6천명 10분 전멸)
- **입단**: "용혈을 이어받았거나, 용의 시험으로 부여받은 자" → 선천+후천 양립 (용혈전사단.md:57)

### D-4. 혈통↔직업↔마법 삼각
- 혈통→직업: 순혈만 기사단/룬 대장장이 입문
- 혈통→마법: 순혈 마법사 vs 기공사 "마도 반란"(연대기.md:108) / 천극 마나=혈맥 역학(세계의-헌법.md:55)
- 후천 변이축: 용혈 농도가 혈통=마법=직업=계급을 단일 척도로 통합("가장 진한 용혈의 뜻이 곧 헌법")

## G. 톤·불변식 체크포인트 (합성 시 적용)

- **신=침묵**: 종족 기원을 "신의 창조 행위"가 아니라 **"신앙·전승에 따르면"**으로 프레이밍. 갈등축은 종족 간 정치·이데올로기(인간↔엘프, 순혈↔혼혈)
- **샌더슨 법칙**: 용혈 변이 = 대가/제약 명시(스칼렛 몰트·30세 사망)
- **체호프의 총**: 종족 특성이 후크/서사에서 실제 쓰이는가
- **5대 불변식**: 에반 페르소나의 가짜 가문(카스테로·무라드 등) ≠ 에반 진짜(미상) 혈통 구분
- **명제형 + 표기 일관**: "수인족 = 해양 거주, X능력, Y제약" / 엘프·인어 표기 1종

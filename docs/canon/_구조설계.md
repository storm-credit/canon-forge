# 설정집 정보구조 설계 (Canon Information Architecture)

> **이 문서의 역할**: THE FORGOTTEN SUMMONER 설정집의 **전체 구조 정본**. 원본 29개 아카이브(8,954파일)를 어느 계층·어떤 입자도로 배치하는지의 마스터 청사진. 작업은 이 표를 따라가고, 매 단계 "원본 N개 → 캐논 N페이지" 대조로 검증한다.
>
> 진행 추적 = `_커버리지.md` · Phase 서술 = `_progress.md` · 입자도 규칙 = `_PLAN.md` · **구조 정본 = 이 문서**

---

## 0. 대원칙 — 위키형 설정집의 3원칙

1. **엔티티 1개 = 페이지 1개.** 고유명사로 명명된 개체(도시·인물·신·가문·조직·지역)는 절대 한 문서에 뭉치지 않는다. 각자 페이지 + 상위 인덱스.
2. **시스템은 통합.** 수십~수천 개 동종 규칙(마법 1,866 등)은 체계 문서 1개 + (대량이면) 인덱스. 개별 파일 금지.
3. **서사는 주제별 보존.** 관계·연표·제도 산문(외교·역사·법률)은 주제당 1파일, 축약 금지.

> 상위 문서(`<세력>.md`·`<대륙>.md`·`창세신화.md`)는 **개요 + 색인** 역할로만. 실내용은 하위 엔티티/서사 파일이 보유.

**입자도 판정 플로차트**
```
원본 카테고리를 만나면 →
 ├ 고유명사 객체가 3개 이상 나열? ── 예 → [엔티티] 개체당 파일 + 인덱스
 ├ 수십~수천 동종 항목 규칙체계? ── 예 → [시스템] 통합 문서(+인덱스)
 └ 1~2개 산문 문서?            ── 예 → [서사] 주제당 파일(풀 보존)
```

```
0-헌법    토대   세계규칙·창세신화·신앙·연대기·고대문명

2-무대    장소   5대륙 → 28세력 → 도시/요새/지역  + 교통통신·전쟁사
3-사물    물건   아이템·몬스터·동반수
4-인물    사람   영웅·인물·에반·9페르소나
```
검증·작업은 **0 → 1 → 2 → 3 → 4** 순서로 올라간다.

## 3. 실행 순서 (의존성 준수)

1. **Phase 0 입자도 결정** — 18신·9교단을 폴더 개별화(선택지 A)할지 사용자 확정. (내용은 이미 보존; 입자도만)
2. **Phase 1 인덱스 점검** — 각 체계 문서에 전체 목록 인덱스 유무 확인.
3. **Phase 2 (가) 대륙 복원** — 🔴 최우선. 에테르부터 5대륙 지리 엔티티 전개.
4. **Phase 2 (나) 세력 서사 복원** — 🟡 3세력 서사 카테고리 마저 + 상인협회 / 이후 미착수 25세력.
5. **Phase 3·4 입자도 점검** — 아이템·영웅 인덱스+개별 입자도 확인.
6. **Phase 5 수렴** — 위키링크 정합·고아 제거·모순 0.

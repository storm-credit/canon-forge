# 세계 종교 및 신앙 체계

> **[공통 프레임]** 신들은 황혼 이후 **부재·침묵** 상태다. 각 교단이 '신의 힘'이라 부르는 것은 신화시대 잔향(殘響)과 성물의 메아리이며, 현재 시점에 신이 직접 기적·강림·개입하는 것은 없다.  
> 신앙 대상의 **진명(眞名)**: 창세신화 기준을 정본으로 하며, 교단의 통속명(가이아·포세이돈 등)은 민간 호칭이다.

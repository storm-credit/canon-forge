# 🏗️ 1장 리모델링 설계안 — 하이브리드 채택안의 뼈대

> **집필 금지 — 이 뼈대 전 층위 승인 전.** 아래 화별 스파인은 사건·보상·훅·사용 자재·이식되는 기존 장면을 좌표로 명시한다. 고유명 신설은 발명 0 게이트로 전부 작가 확인 표기(※).

---

## 0. 채택안 엔진 한 눈에

- **가로축(반복 재미 단위)**: 화당~2화당 **감정 케이스 1건 완결** — ①의뢰 접수 → ②유물 기억 읽기(능력을 '목격'으로·목소리 번역=순정보) → ③소환/기지 제압(도망 아닌 제압) → ④보상(수임료·사이다·세계 한 조각·시그니처) + 대가 미세 동반. **5화 내 '이 소설은 이런 재미' 공식 학습 완료.** 케이스=세계관 튜토리얼(설정 덤프 0).
- **세로축(장기 궁금증)**: 에반이 유일하게 값 못 매기는 **부러진 나침반**이 매 케이스 말미 지하 방향으로 진동을 키운다. 근거리 의문(케이스 내 회수)+장거리 떡밥(FS-0001·FS-0003) 겹쳐심기 → **10화에서 '이름 없는 유물 회수'** 손에 잡히는 미션으로 결정화.
- **하류 규율(C)**: 루미에 순정보 불변식(net-gain ≥1) + 중형 보상 11~25화 실삽입 + 발명 0.

## 1. 1~10화 화별 뼈대 (스파인·보상·훅·자재·이식)

> **훅 표기**: 클리프행어 7종 = (1)위기 노출 (2)정체 균열 (3)방향 지시 (4)새 변수 (5)파손·습격 (6)그림자 접근 (7)세계규칙 붕괴. 같은 유형 3연속 금지.

### 1화 — 케이스1 완결 (우는 유물) · **[전면 재집필]**

- **스파인**: 시그니처 라인 콜드오픈으로 **want 자백**('남의 인생은 또렷한데 내 인생은 흐릿하다 — 그래서 나는 지워진 걸 되살려 세우는 값으로 산다') → 만짐=기억 읽기로 능력 **'목격'** → 유물 임계 폭주 → **첫 소환 현현 + 에반 기지로 제압**(첫 승리, 도망 아님).
- **보상**: 두둑한 수임료 · '1인 용병단' 이명 씨앗 · 시그니처 라인 각인. **대가 미세**(코피).
- **훅**: (5→개시) 다른 의뢰인이 맡긴 **'부러진 나침반' — 에반이 처음으로 값을 못 매긴다.** 손끝이 지하에서만 저리다. → **세로축 개시 · FS-0001 설치.**
- **사용 자재**: 우는 유물 / 임계 현현 / 골목 마수 제압 / 1인 용병단 이명.
- **이식**: 기존 1화 만짐 루프 → '읽음' 단발 전환. C의 want 자백 문단 + 시그니처 기억읽기 장면 이식. **B의 케이스 완결 골격 위에 얹음.**

### 2화 — 케이스2a (사라진 의뢰인)

- **스파인**: 나침반 맡긴 의뢰인 실종. 사무소 다른 유물 기억을 읽어 행방 추적 개시. **목소리가 기억을 번역해 단서 제공(순정보).**
- **보상**: 추적 실마리 + 뒷골목 규칙 한 조각(마수 유인 원리).
- **훅**: (4) 흔적이 암시장으로 이어지고 그 방향이 **나침반 지하 방향과 일치.**
- **규율(C)**: '몰라' 금지 — 정체(결말)만 유보하고 **경고 순정보 부착**('저 마수들이 네 부풀음 냄새를 맡고 온다').
- **이식**: 기존 2화 재탕 회차(1화 반복) 폐기 → 추적 케이스로 재구성.

### 3화 — 케이스2b 완결 (암시장 잠입)

- **스파인**: 자유도시 이중구조(합법 거래소 vs 이단법 암시장)를 **사건으로 통과.** 나침반이 '누가 애타게 찾던 물건'임을 해제.
- **보상**: 나침반 내력 일부 · 판돈 · 세력 이면.
- **훅**: (2) 목소리 첫 자기균열 — '저 방향, 나랑 닮았어. 왜인지는 나도 몰라'(**결말만 유보, 정보는 지급** · FS-0001 강화).
- **사용 자재**: 암시장 추적.

### 4화 — 케이스3a (골든마르크 감정 의뢰)

- **스파인**: 대경매장 유물 감정 요청. 뒷골목 맨얼굴론 문턱도 못 넘어 **페르소나 라인하르트 첫 가동**(위장=능동 카드).
- **보상**: 페르소나01 자산 획득 · 금융 허브 소개.
- **훅**: (4) 거래소 게시판 '지하 방향 유물 고가 매입' 공고 — 누군가 **나침반의 그곳을 이미 사들이는 중.**
- **사용 자재**: 페르소나01 라인하르트 카스테로 · 골든마르크 무대.
- **규율(C)**: 목소리 정곡('남의 얼굴 쓰는 게 뭐가 다르냐') 반박에 관계 이득 + 선긋기('이건 내가 고른 것').

### 5화 — 케이스3b 완결 (법이 도시마다 다르다) · **[반복 재미 단위 각인 완료 지점]**

- **스파인**: 골든마르크선 합법인 거래가 인접 구역선 '사령 유물 밀거래'로 낙인되는 혼란을 몸으로 통과·해결.
- **보상**: **큰 수임료 + 세계 문법 획득('법=계약=금화')** · '1인 용병단' 이명 몸으로 성립. **대가로 편두통.**
- **훅**: (6) 매입 배후 큰손의 그림자, 나침반 견인 심화.
- **점검**: 여기까지 '결핍→작은 돌파→보상→다음 결핍' 진동 **5회 노출 완료** → 독자 공식 학습.

### 6화 — 케이스4 (남의 기억) · **[중형 보상 삽입 起]**

- **스파인**: 감정 중 **낯선 기억 한 조각**이 에반의 어제 기억과 경계가 흐려짐. 목소리 '그거, 네 거 아니야'(순정보).
- **보상**: **능력 성장(타인 기억 판독)** · '나는 누구인가' 근거리 의문 점화(오독→정정 2단).
- **훅**: (6) 마수 소동이 '이상한 힘 쓰는 감정사' 소문으로 번짐 · 의뢰 노파가 흔적째 소멸(**FS-0003 설치·강화**).
- **사용 자재**: 남의 기억 조각.

### 7화 — 케이스5 (소문의 방문자)

- **스파인**: 소문 듣고 온 **회색 인물**의 감정 의뢰. 우호적 얼굴 뒤 다른 셈.
- **보상**: 세력 이면 정보 · 페르소나 운용 심화 · **미션 좌표 가시화**(견인 방향=회수 목표로 번역, 추상 want→행동 목표).
- **훅**: (4) 방문자가 나침반 방향과 마수 소동을 **이미 다 알고 있다** — 누구 편인지 모른다.
- **사용 자재**: 회색 방문자 / 낯선 사내.

### 8화 — 케이스6a (역학이라 부르는 자)

- **스파인**: 방문자가 흘린 정보 — 아크브릿지 하부에 소환술을 **'흑마술 아닌 역학'**이라 부르는 이가 있다. 나침반 코트 최심부 정착.
- **보상**: 능력 재프레이밍 예고 · **'견인을 읽는 법' 전수(성장 보상)** · 촉매 개선 기대.
- **훅**: (2) 목소리 자기정체 첫 균열('왜 저 방향을 알지'는 결말만 유보) → 나침반 견인이 **아크브릿지 협곡과 연결됨을 지목**(다음 무대 견인).

### 9화 — 케이스6b (성국의 창끝)

- **스파인**: 사건에 얽혀 소환술이 **'사령술'로 낙인될 위험.** 신성 계약단(성국 사냥개)의 존재를 처음 감지.
- **보상**: 위험 지도 · 세계 축(신앙 vs 자유) 첫 냄새.
- **훅**: (7) '저 이교도는 누구 손에 쥐어져야 하는가'라는 물음의 **첫 그림자.**

### 10화 — 케이스 결착 · 미션 점화 · **[부분 재집필]**

- **스파인**: 하룻밤 매듭. 나침반을 **'고장'으로 봉인하되 곁에 둔다(스스로 선택).** want→미션 번역: 지하가 가리키는 **'이름 없는 유물'**(※이시스-아스트라 계보 무명/진안 계열 — **고유명 확정은 작가 확인**)을 기어이 감정·회수하겠다고 **자기 입으로 선언 = 구체 미션 드라이브 명문화.**
- **보상**: **미션 드라이브 확정.**
- **훅**: (3→4) 목소리가 처음으로 방향을 지목 + '예전에 가본 적 있어'(장거리 떡밥·결말만 유보). 세 세력이 동시에 '누구 손에'를 묻기 시작.
- **이식**: 기존 '고장으로 넘긴다' 장면 재배치 + 결착 지문 개조.

## 2. 11~100화 파급 지도

> **원칙**: A1(1~10화)만 '케이스 퍼 에피소드'로 전면 재편. A2~A8(11~100화) **골격 대부분 보존**, 나침반 세로축을 through-line으로 명시적 오버레이 + C 규율 3종 승격.

| 구간 | 아크 | 처리 | 내용 |
|---|---|---|---|
| 11~25화 | A2 (페르소나 잠입) | **수정(콘티 재라벨 + 중형보상 실삽입)** | 페르소나 잠입 콘티를 '의뢰→감정→해결' 케이스 단위로 재라벨(사건·좌표 보존, 프레임만 케이스화) + 세로축 나침반 진동 비트 삽입. **11~13화 페르소나 운용 성공을 사이다로 명시 · 14화 청동 원반 획득(기존) · 15화 대조 실험 승리를 중형 보상으로 승격** → 5→22화 대가 단독 누적 구간 해소. |
| 26~40화 | A3 (역학 재해석) | **경미 수정(주석)** | 8~9화 예고 씨앗 연결 주석 수준. '역학 재해석' 진입이 8~9화 예고로 매끄러워짐. 26화 최초 이름 호명 규칙 무변경. |
| 41~100화 | A4~A8 | **오버레이(경미)** | 화별 콘티 골격 보존. 나침반 세로축 through-line 주석 + '케이스 클러스터' 라벨만 삽입. |

### 2-a. 전 구간 규율 승격 3건 (C)

1. **루미에 순정보 불변식 → 스토리-상태 장부 전 구간 규칙 승격**: 11~100화 모든 '몰라/모르겠어'를 '정체(결말)는 유보하되 번역·해제·경고 순정보 ≥1' 규격으로 전수 치환. **장부 스키마에 'net-gain' 열 추가.**
2. **중형 보상 삽입 A2까지 연장**: 아크당 소환 운용 비트 1회와 병렬로 6~24화 가뭄 본체 정면 해소.
3. **want→미션 전환(10화 앵커)** → A2~A8 나침반 견인 추적 전체가 '지하 방향 회수 미션 드라이브'로 재프레임(추상 기억회복→행동 목표).

### 2-b. 그대로 유지(무변경)

- 세력 순회 순서(골든마르크→아크브릿지→카르델라→포지게이트→성국, **성국 우선**) · 무대 실존 캐논 · 페르소나01 라인하르트 운용 · 240화 봉인 · 클리프행어 7종 로테이션 · 시간선 B안(1~6화 첫날밤/7~10화 이튿날밤) · 26화 최초 호명.
- **R3-1 보상 위계**: 소환 중보상은 A1 첫 현현 1회 정점, 이후 기존 진용 재활용.

## 3. 재작업 견적

| 유형 | 화수 | 내용 |
|---|---|---|
| **신규 재집필(전면)** | 1화 | 1화 — 케이스 1건 + 첫 소환 현현 + 나침반 접수 + want 자백을 한 화 압축(재미 패널 1화 압축 처방과 정합). |
| **부분 재집필** | 1화 | 10화 — want→미션 전환 명문화 · 결착 지문 개조. |
| **실질 재집필(전면 케이스화)** | 8화 | 2~9화 — 슬로우번(만짐 루프 3회→현현 3화→제압 5화)을 '화당 케이스 완결' 구조로 전면 재구성. |
| **수정(콘티 재라벨 + 중형보상 실삽입)** | ~15화 | 11~25화 A2 — '의뢰→감정→해결' 케이스 단위 재라벨(사건·좌표 보존) + 세로축 비트 + 중형 보상 승격. |
| **경미 수정(주석)** | — | 26~40화 A3 — 8~9화 예고 씨앗 연결 주석. |
| **오버레이(주석)** | ~60화 | 41~100화 A4~A8 — through-line 주석 + 케이스 클러스터 라벨. |
| **전 구간 규율 승격** | 전건 | net-gain 열 추가 · '몰라' 전수 치환. |

- **총량**: 신규 재집필 10화(1~10) + 실질 수정 약 15화(11~25) + 주석 오버레이 약 60화(41~100).
- **리스크**: 봉인·FS·세력 순회 불변이므로 **구조 리스크 낮음.** C의 장부 규칙 승격이 하류 정합 파동을 규격화해 재봉합 리스크 추가 완화.

## 5. 콘티 문법 승격 규정 (집필 게이트)

> 아래 4규정을 콘티 문법으로 승격. 리모델링 원고는 화별로 전건 준수 — 미달 화는 재집필. (기획서 §3 체크리스트 12항의 집행 규격.)

### 5-a. 화당 보상 규정 (진단③ 집행)

- 각 화 말미에 **소보상(획득·성장·사이다) 규칙 배치.** 영구 대가(냉기·수명·기억)는 **소보상보다 앞서거나 최소 동반** — **대가 단독 누적 화 금지.**
- R3-1 위계: (소)아이템=장 말미 / (중)소환 현현=부 단위(A1 첫 현현 1회 정점) / (대)서사 진전=Act 단위.
- **11~25화 중형 보상 실비트 의무**(11~13 페르소나 성공·14 원반·15 대조 실험) — 재라벨만으로 흘리기 금지.

### 5-b. 대화 공전 금지 규정 (진단① 집행)

- 매 문답이 **번역·해제·경고 중 순정보 ≥1**을 남긴다(net-gain). '몰라/모르겠어' 단독 응답 금지.
- **유보 대상은 '결말'뿐** — 정체(누구인가/왜 아는가)만 유보하고 정보(무엇/어떻게/경고)는 지급.
- 스토리-상태 장부에 **'net-gain' 열** 추가, 화별 교환마다 순정보 1줄 기록.

### 5-c. want 점화 규정 (진단② 집행)

- **1화 온스크린 want**: 에반의 능동 욕망을 자기 입/자기 동기로 명시(시그니처 라인). '원해지는 자' 등장 금지.
- **10화 미션 결정화**: 추상 want(기억 회복)를 손에 잡히는 구체 미션(이름 없는 유물 회수)으로 자기 선언 번역.
- 제압은 **도망 아닌 능동 제압**(첫 승리 1~2화).

### 5-d. 톤 감시 게이트 (잔여 리스크 관리)

- 케이스 중심이 서정·미스터리 좌표를 흐리지 않도록 **매화 문체 정본 C 리트머스** 적용: 4행 초과 장문단은 감각·전환 한정 / 설정 덤프 0 / 시스템창 산문 노출 0 / 냉소 유머로 무게 감싸기 / 오독→정정 2단 화 완결.

## 6. 결재 요청

1. **1~10화 화별 뼈대(스파인·보상·훅·자재·이식)**를 승인하십니까?
2. **11~100 파급 처리(A2 재라벨+중형보상 실삽입 / A3 주석 / A4~A8 오버레이 / 규율 3종 승격)**를 승인하십니까?
3. **FS 재배치표(소멸 0)** 및 **콘티 문법 승격 4규정**을 게이트로 확정하십니까?
4. **10화 미션 유물 고유명**(※이시스-아스트라 계보 무명/진안 계열) — 확정 지시 필요(발명 0 게이트).

> 4개 항목 승인 시 → **집필 착수.** 승인 전 집필 금지.

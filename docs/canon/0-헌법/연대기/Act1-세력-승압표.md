# 📈 Act 1 세력 승압표 — 누가 언제 어떻게 올라오는가

> 원칙: 세력은 **씨앗(이름·시선 한 컷) → 접촉(대면·거래) → 개입(이해관계 충돌) → 본편(부 주역)** 4단으로 승압한다.
> 한 단계를 건너뛰고 등장하는 세력이 없게 — "갑자기 나타난 세력"은 이 표 위반이다.

## 성장영웅 존재감 3층 (한백무림서 운용 — 작가 문답 2026-07-14)

| 층 | 인원 | 방식 |
|---|---|---|
| 직접 교차 | Act당 2~3인 | 자기 목표로 움직이다 에반과 스침·충돌, 그 Act 안에서 각자 매듭(R2-6) |
| 간접 존재감 | 소수 | 소문·전적·현상금 명단 — 세계가 에반 없이도 움직인다는 신호 (화당 0~1회, 억지 금지) |
| 문서 층 | 나머지 전원 | 설정집 대기 = 후속 캐스팅 풀. 미등장은 낭비가 아니라 세계의 두께 |

> 전원 등장 금지 — 한백무림서의 주역 군상도 소수였고, 라이트 독자 인지 부하가 실증 경고됨.

## 타대륙 언급 3원칙 (작가 문답 2026-07-14)

1. **질감 언급 권장**: 교역품·항로 소식·이주민의 입 — 대륙의 존재감만 (자유도시=허브라 자연 채널).
2. **세력 연장선 제한**: 캐논 실존 관계만 (예: 성국-솔라리안 성전 전선 — 추적 전이에 기배선).
3. **예고편 금지**: 타 대륙 Act의 본편 내용(아르카나 비밀·결빙 유산 등) 사전 소진 금지 — Act별 완결·진입 재소개(R1)가 있으므로 불필요. 예외는 Act 말 인계 고리 단 하나.

## 집행 규칙

1. **2장 콘티 생성 시 이 표가 구속력** — 각 세력의 해당 단계 비트를 콘티에 의무 배선, 게이트3 검사 축에 "승압표 준수" 추가.
2. 1장 잔여(68~100)는 씨앗 단계만 — 접촉·개입을 앞당기지 말 것 (본편 텐션 소진 방지).
3. 이미 쓴 1~67화는 소급 수정하지 않는다 — 도입부의 성국 집중은 설계상 정상 (수정은 씨앗 전진 배치로 충분).
4. 세력 등장 시 캐논 정합: 각 세력의 외교 정합본(표면/이면)과 일치해야 하며 게이트3이 검사한다.

#  예언과 미래 시나리오 (Prophecies & Future)

> [!NOTE] 🌘 종말 예언 — 아래 신들의 묘사는 신화시대 신앙 기록이며 현세 사실이 아니다. [[신들의종말EndofGods|신들의종말]] 이후 신은 부재·침묵. 현재 규칙은 [[세계의-헌법#5. 신과 인간 — 세계의 테마 ★|헌법 §5]].

> *"예언은 단 하나의 미래를 보여주지만, 선택은 무수한 가능성을 만든다."*

---

##  주요 예언

| 예언명 | 내용 | 관련 세력 |
|-------|------|----------|
| **태양이 검게 물들리라** | 태양이 어둠으로 덮이고, 성국이 심연의 군단과 전쟁을 벌일 것 | 성국, 심연의 군단 |
| **마지막 제국의 몰락** | 크림슨 대륙의 최강국이 붕괴하고, 새로운 세력이 등장 | 솔라리안 제국, 왕국연합 |
| **심연의 문이 열리리라** | 심연의 존재들이 현실 세계로 쏟아져 나오는 대재앙 발생 | 심연의 군단, 마법협회 |
| **영웅과 악마의 춤** | 한 시대를 끝낼 영웅과 악마가 등장하여 세계의 운명을 가름 | 모든 주요 세력 |
| **시간의 고리** | 과거의 실수가 반복되며, 세계는 다시 멸망의 위기를 맞이 | 마도제국 잔재 |

---

##  가능한 미래 시나리오

| 시나리오 | 설명 |
|----------|------|
| **심연의 대침공** | 심연의 군단이 차원의 균열을 통해 대륙 전역을 침공 |
| **제국의 붕괴** | 현재 강대국들이 내부 분열과 외부 위협으로 몰락 |
| **금단 마법의 부활** | 봉인된 강력한 금단 마법이 다시 발견 |
| **차원의 융합** | 차원의 균열이 확대, 다른 차원들이 현실 세계와 합쳐짐 |
| **운명의 전쟁** | 운명신 테네브리스의 신탁이라 전해지는 예언을 좇아, 모든 존재가 하나의 미래를 향해 움직인다는 믿음이 세계를 휩쓺 (신의 직접 개입이 아닌 인간의 예언 해석) |

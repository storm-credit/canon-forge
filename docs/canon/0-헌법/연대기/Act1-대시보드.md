## 2. 세력·인물 요약

- **에반 × 루미에**(나침반 목소리) — 페어 티키타카가 무기 (재미 패널 4인 일치)
- **추적자**: 성국 이단심문·신성 계약단 (1-B부터, Act 2로 "확장하나 좌절" 전이)
- **무대**: 자유도시연합 4도시 (베르나·카르델라·아크브릿지·포트 넥서스)
- **폭발축**: 마법협회 ↔ 정령연합 (표면 동맹·이면 원한 → 1-C)
- **히로인**: 세레나 루미나스·루미에르 아르카나 (에테르 2인만)
- **그림자**: 감시자 — 1-D 12장 첫 지문만, 정체는 시리즈 봉인 (FS-0002)

## 3. 큰 사건 축 (부별)

1-A(240화) 첫 현현·마수전 → 1-B(300화) 고해록 폭로·성국 정조준 → 1-C(300화) 협회-정령 폭발·균열 요동 → 1-D(360화) 지워진 역사 폭로·여명의 잔광 대면·감시자 지문 → "기억을 지켜주는 자"

# 📥 2장 콘티 반영 대장

## A. 의무 배선 (승압표·감사 이월)
1. **성장영웅 착화** — 루시엔 감정대 스침 비트 의무 배선(승압표 2장 교차 1~2회) + 자유도시발 씨앗(아크브릿지 협곡 균열 소식·게시판 채널 재활용)
2. 활용도 금맥 waypoint — 기억의 도서관·잊혀진 서고단(살생부 원본 대조 훅)·철의금융·숨겨진 학원
3. 성국 지휘관 대의 장면화 — '인간 크기의 순간' 1회 (또는 인터루드 — 작가 결재)

## B. 연출 지시 (모티브 정련 6건)
4. 아포리즘 배분 — 소환영웅·루미에르에게 자기소유 통찰 라인 상향, 아크당 A밀도 서정 블록 하한 1
5. 경이 팔레트 승급 — 방죽·등불·저림은 003 전용 예약, 2장 대형 피크는 새 감각 계열
6. 진동 리듬 — 소환 운용 비트 아크당 1회 이상·'나도 몰라' 반복 금지·화말 훅=세계 조각 지급
7. 회계 은유 경계 — 셈/값은 에반의 자기기만 보이스로만, 세계 형이상학화 금지(業=흉 착지)
8. 페어 리듬 — 페르소나 화에도 루미에 반말 위트 되받기 1~2비트, 연역은 산문 이관
9. 존대 슬립 이격 — 에반=반사 즉시정정 / 루미에=누출 잔류로 결 분리

## C. 리모델링 후속 (1~10 재집필과 별개)
10. **11화 콘티 본문에 '환영의 서 첫 가동 = 루미에 첫 완전 발화(FS-0001 언어 설치)' 비트 실삽입** — 장부·콘티 §2는 갱신 완료(2026-07-15), §3 11화 본문만 잔여. 11~100 만담 밀도 개조 배치에 포함할 것.

## 3. 채점 (4축 균등 가중, 각 10점)

| 시안 | 시장성 | 모티브 충실 | 보존 | 비용 | **총점** |
|---|---|---|---|---|---|
| **B** | 9.0 | 8.5 | 7.5 | 7.0 | **32.0** |
| **C** | 6.5 | 9.0 | 9.5 | 9.5 | **34.5** |
| **하이브리드** | — | — | — | — | **두 시안 각각 상회(정성)** |

### 3-a. 시안 B 리스크

- (a) A1 전면 재편의 A2~A8 재봉합을 주석 60화로만 낙관 → **정합 파동 견적 초과 소지.**
- (b) 케이스 퍼 에피소드가 사건 중심으로 기울어 **서정·미스터리 좌표(문체 정본 C) 이탈** 위험 → 톤 게이트 필수.
- (c) 매화 케이스 완결이 세로축 장기 미스터리와 **자원 경쟁 → 궁금증 희석.**
- (d) 케이스 유물·의뢰 **신규 발명이 발명 0 게이트 저촉** → 작가 결재 필수.
- **강점**: 작가 3대 요구(명확한 미션·약천마식 리듬·10화 재미)를 **구조로 충족하는 유일 시안.**

### 3-b. 시안 C 리스크

- **치명 리스크**: 효과 상한이 **구조적으로 낮음** — 본인 정직 견적 "미션이 전설의 보검식 구체 목표에 미달, 견인 방향 추상성 잔존, 미션 드라이브 소프트."
- 진단② want 능동화 = **부분·최약**(구조 동결로 에반이 여전히 견인·조사·방어. 본인 자백 'want 능동화는 근원 해결 아닌 앞당김').
- 1~5화가 기존 마수 농성 시퀀스라 **반복 재미 각인이 B보다 약함.**
- **강점**: 보존·비용·리스크 최소, 발명 0 완전 준수, 균등 가중 총점 최고(34.5).

## 4. PM 추천 — 하이브리드 ('B-엔진 위에 C-규율')

### 4-a. 왜 단독 채택이 비권장인가

- **C 단독 불가**: 균등 총점은 C(34.5)>B(32)로 C 우세(보존·비용·대화공전 압도). 그러나 이 작품은 **몰입 부재가 치명 판정된 정체작**이고 작가 명시 요구는 명확한 미션·약천마식 돌파·10화 확실한 재미인데, C는 이 셋을 **'구조적으로 못 낸다'고 스스로 자백.** 진단 최상류 원인(감정사-장부 태세→회차당 수지 적자)을 C의 구조 동결은 **못 건드리고 하류만 흑자화.**
- **B 단독 불가**: 3대 사각지대 — 하류 대화공전 잔존·6~24화 가뭄 본체 미삽입·유물 신규 발명 리스크.

### 4-b. 왜 하이브리드가 두 시안 각각을 상회하는가

**결함이 상호 배타적이라 조합 시 상쇄한다.**

- **B의 케이스 엔진**이 진단 3결함과 작가 3대 요구를 **상류 태세 전환**으로 해결.
- **C의 전구간 순정보 불변식·중형보상 명시 삽입·발명 0 규율**이 B의 3대 사각지대를 **정확히 봉합.**

### 4-c. 하이브리드 설계 요지 ('B-엔진 위에 C-규율')

- **1화 (C 자백형)**: B 골격(케이스1 완결+첫 소환 현현+나침반 접수 한 화 압축) 위에 **C의 want 자백 콜드오픈** 이식 — 시그니처 라인을 능력 목격 직전 배치 → 진단② 1페이지에서 응원 훅 점화.
- **엔진 (B)**: 1~10화 화당~2화당 케이스 완결(의뢰-읽기[목소리 번역=순정보]-소환/기지 제압-보상+대가 미세). 5화 내 공식 각인. 10화 이름 없는 유물 회수 미션 결정화.
- **하류 규율 3종 (C, B 사각지대 봉합)**:
  1. **순정보 불변식을 11~100화 장부 전 구간 승격**(net-gain 열) → B의 하류 대화공전 확정 치유.
  2. **중형 보상을 11~25화에 실비트로 명시**(페르소나 성공·원반·대조 실험 승격) → B가 재라벨로 흘릴 6~24화 가뭄 본체 봉합.
  3. **발명 0 게이트 + 봉인 좌표 유지** → 케이스 유물은 기존 A1 자재 재배열로 조달.

### 4-d. 3결함 최종 판정 (하이브리드)

| 결함 | 판정 |
|---|---|
| ① 대화 공전 | **완전 치유** |
| ② 주인공 무욕 | **강 치유**(1화 자백+제압 능동성+상류 태세 전환) |
| ③ 보상 가뭄 | **완전 치유**(케이스 매화 흑자+본체 중형 실삽입) |

- **작가 3대 요구 전부 충족.** 잔여 관리는 **톤 이탈 하나**(케이스 중심이 서정·미스터리 좌표를 흐릴 위험) → **감시 게이트 병행**(문체 정본 C 리트머스 매화 적용).
- **비용**: 신규 10화 + 수정 15화 + 주석 60화. C의 장부 규칙 승격이 하류 정합 파동을 규격화해 재봉합 리스크를 낮춘다.

## 4. PM 추천 — 하이브리드 ('B-엔진 위에 C-규율')

### 4-a. 왜 단독 채택이 비권장인가

- **C 단독 불가**: 균등 총점은 C(34.5)>B(32)로 C 우세(보존·비용·대화공전 압도). 그러나 이 작품은 **몰입 부재가 치명 판정된 정체작**이고 작가 명시 요구는 명확한 미션·약천마식 돌파·10화 확실한 재미인데, C는 이 셋을 **'구조적으로 못 낸다'고 스스로 자백.** 진단 최상류 원인(감정사-장부 태세→회차당 수지 적자)을 C의 구조 동결은 **못 건드리고 하류만 흑자화.**
- **B 단독 불가**: 3대 사각지대 — 하류 대화공전 잔존·6~24화 가뭄 본체 미삽입·유물 신규 발명 리스크.

### 4-b. 왜 하이브리드가 두 시안 각각을 상회하는가

**결함이 상호 배타적이라 조합 시 상쇄한다.**

- **B의 케이스 엔진**이 진단 3결함과 작가 3대 요구를 **상류 태세 전환**으로 해결.
- **C의 전구간 순정보 불변식·중형보상 명시 삽입·발명 0 규율**이 B의 3대 사각지대를 **정확히 봉합.**

### 4-c. 하이브리드 설계 요지 ('B-엔진 위에 C-규율')

- **1화 (C 자백형)**: B 골격(케이스1 완결+첫 소환 현현+나침반 접수 한 화 압축) 위에 **C의 want 자백 콜드오픈** 이식 — 시그니처 라인을 능력 목격 직전 배치 → 진단② 1페이지에서 응원 훅 점화.
- **엔진 (B)**: 1~10화 화당~2화당 케이스 완결(의뢰-읽기[목소리 번역=순정보]-소환/기지 제압-보상+대가 미세). 5화 내 공식 각인. 10화 이름 없는 유물 회수 미션 결정화.
- **하류 규율 3종 (C, B 사각지대 봉합)**:
  1. **순정보 불변식을 11~100화 장부 전 구간 승격**(net-gain 열) → B의 하류 대화공전 확정 치유.
  2. **중형 보상을 11~25화에 실비트로 명시**(페르소나 성공·원반·대조 실험 승격) → B가 재라벨로 흘릴 6~24화 가뭄 본체 봉합.
  3. **발명 0 게이트 + 봉인 좌표 유지** → 케이스 유물은 기존 A1 자재 재배열로 조달.

### 4-d. 3결함 최종 판정 (하이브리드)

| 결함 | 판정 |
|---|---|
| ① 대화 공전 | **완전 치유** |
| ② 주인공 무욕 | **강 치유**(1화 자백+제압 능동성+상류 태세 전환) |
| ③ 보상 가뭄 | **완전 치유**(케이스 매화 흑자+본체 중형 실삽입) |

- **작가 3대 요구 전부 충족.** 잔여 관리는 **톤 이탈 하나**(케이스 중심이 서정·미스터리 좌표를 흐릴 위험) → **감시 게이트 병행**(문체 정본 C 리트머스 매화 적용).
- **비용**: 신규 10화 + 수정 15화 + 주석 60화. C의 장부 규칙 승격이 하류 정합 파동을 규격화해 재봉합 리스크를 낮춘다.

> **A 반영 재심사는 PM 몫** — §4-A(보완 생성 시안 A)의 강점(1화 즉발 미션·"위작에는 죽은 자가 없다" 시그니처)을 하이브리드에 흡수할지, A를 독립 채택안으로 승격할지의 재심사 판정은 PM이 별도로 수행한다.

## 4-A. 시안 A (보완 생성 2026-07-15) — 전설의 보검 진위 감정 미션형

> [!IMPORTANT] 보완 생성 배경
> §1 비교표에서 시안 A는 '스펙 미입력 — 하이브리드에서 C의 1화 want 자백형으로 복원'으로만 처리됐다. 작가 제안(전설의 보검 미션형)의 A 시안이 최초 생성에 실패해, 본 절로 **동일 규격 보완 생성**한다. 기존 §1~§5 무삭제, §3 채점표에 A 행 추가, §4 PM 추천 절에 재심사 위임 1줄만 삽입했다. **불변 제약 전량 준수**(봉인 규정·발명 0·FS-0001~0006 보존·문체 정본 C·루미에 순정보).

### ① 로그라인

> **"위작에는 죽은 자가 없다. 진짜에는 반드시 누군가의 마지막이 남아 있다 — 나는 그 마지막을 읽어, 남이 지운 삶을 값 매긴다."**
> — 감정(鑑定)=**죽은 자의 기억 읽기**를 1화 클라이맥스 시그니처로 전면화. 시리즈 로그라인("세상이 지워버린 것들을, 나는 기억해서 다시 일으켜 세운다")의 **직업적·미션형 번역** — '읽기(감정)'를 손에 잡히는 의뢰로 앞세우고, '세우기(소환)'는 미션 수행 중 터진다.
> 시그니처 라인(FS-0006)은 그대로 설치: *"남의 인생은 또렷한데 내 인생은 흐릿하다. 장사치로는 최악의 재능이고, 감정사로는 최고의 재능이었다."*

### ② 엔진 구조 (가로축 / 세로축)

- **가로축 (미션 엔진 = 반복 재미 단위)**: **"전설의 보검 진위 감정" 의뢰**. 자칭 이시스-아스트라 계보 추정품(발명 0 — 계보 총람 §미등재·비정통 전승 개체 프레임: '가문 구전으로 이름만 물려받은 무명 장검'·'출처를 밝히지 않는 호신구' 등 캐논이 이미 인정한 진위 불명 개체를 감정 대상으로 인용)의 진짜·위작을 가린다. 감정 시그니처 = **만지면 죽은 자의 기억이 흘러든다**(에반 체질, 기존 1화 방해 비트 승격). 진품엔 옛 주인의 마지막이 남아 있고, 위작엔 아무도 없다 — "위작에는 죽은 자가 없다." 매 아크가 새 감정 의뢰(유물·보검)로 열리고, **읽기=순정보 배당**(루미에 정보 이득 의무 자동 충족)·**진위 판정=매 미션 사이다**·**보검을 노리는 세력 경쟁=긴장**을 한 단위에 묶는다.
- **세로축 (through-line)**: **부러진 나침반(FS-0001, 루미에 잔파편)**. 의뢰인이 보검과 함께 맡긴 '잡동사니' 속 나침반이 미션 도중 **지하 방향으로만 떤다** — 미션에 '끼어드는' 세로축. 감정으로 읽은 죽은 자의 기억이 **에반 자기 기억과 경계가 흐려지는**(FS-0003) 순간, 가로축(보검)과 세로축(정체)이 처음 엮인다. 나침반의 견인·루미에 목소리·에반 정체 미스터리(FS-0001·0003)는 미션 물체에 **얹혀** 상승한다.
- **다섯 깃발 (승압표 씨앗 단계 준수)**: 진품 보검이 부르는 복수 세력 — 골든마르크 큰손 / 블랙 수트 카르텔(마르코 블랙잭) / 신성 계약단(성국) / 암시장 세력 / 자유도시 평의회 = 기존 A2~A8 세력 로스터. 1장에선 **전면 등장 금지·씨앗만**(매입 공고·정보상 소문·단일 첩자·그림자). 보검을 향한 '누구 손에 쥐어져야 하는가'가 A8 '저 이교도는 누구 손에'(Act 명제)의 **축소 리허설**로 기능 → 승압표 위계 그대로.

### ④ 11~100화 파급

- **가로 엔진 상속**: 아크마다 **새 감정 의뢰**(유물·보검 진위 감정, 발명 0 — 이시스-아스트라 계보 색인 65자루·루미에파편 도감·기존 유물 도감에서만 인용)로 개막. '죽은 자의 기억 읽기=순정보'가 **순정보 불변식**을 매 아크 자동 충족(진단① 완전 치유). 진위 판정=매 아크 사이다, 의뢰 보수=중형 보상 실비트(진단③ 치유).
- **세력 승압**: 보검이 부른 다섯 깃발이 A2(골든마르크 큰손)→A5(카르델라 블랙 수트)→A7(성국 신성 계약단)→A8(세 세력 동시 물음)으로 씨앗→반전→전면 순차 상승. 기존 A2~A8 무대·세력 로스터 그대로 상속(좌표 이동, 소멸 0).

### ⑧ B / C / 하이브리드 대비 정직 비교 (적대 검토)

**강점**
- **미션 드라이브 최강·최조기**: 1화부터 손에 잡히는 '전설의 보검 진위 감정' = 작가 요구 ①(명확한 미션)을 **1화에** 충족. B·하이브리드는 10화 결정화 → A가 9화 앞선다.
- **want 즉점화 + 응원 훅**: 보수(인생역전) want가 1페이지에 자기 입으로 선다 → 진단②(무욕)를 하이브리드의 '서정 자백'보다 **더 능동적으로** 점화(응원 훅이 호기심 훅보다 앞선다).
- **시그니처의 극적 승격**: "위작에는 죽은 자가 없다"가 감정 체질을 **수동 체질→극적 클라이맥스**로 전환. 감정=죽은 자 기억 읽기가 로그라인('지워진 것을 읽어 세운다')과 직접 공명.

**약점 · 적대 검토 — "보검 미션이 나침반 세로축과 정체성 미스터리를 희석하는가?"**
- **판정: 예, 구조적으로 그렇다(부분).** 1화에 **경쟁하는 두 미스터리 물체**가 선다 — 보검(진위) vs 나침반(목소리·루미에 origin). 보검이 더 시끄럽다(인생역전·세력 경쟁·클라이맥스). 그런데 **세로축·정체성 미스터리의 진짜 담체는 나침반**(FS-0001=루미에 origin, FS-0003=에반 정체)이다. 보검이 스포트라이트를 훔치면, 나침반은 자신을 설치해야 할 바로 그 아크에서 **배경으로 강등**된다 → 이는 §3-a 시안 B 리스크(c) '매화 케이스가 세로축과 자원 경쟁, 궁금증 희석'의 A판 발현.
- **완화책과 그 비용**: 6화(보검의 죽은 자 기억→에반 자기 기억 흐려짐)·8화(나침반 견인→보검 출처)에서 두 물체를 **braid**로 엮으면, 보검 미션이 정체 미스터리를 **끌어올리는 매개**가 되어 희석이 통합으로 뒤집힌다. **단 이 braid는 매 아크 유지해야 하는 상시 부하**이고(⑥), 실패 시 곧바로 희석으로 되돌아간다.
- **톤 리스크(모티브)**: 보수(돈)·인생역전·세력 경쟁 프레임이 문체 정본 C(서정·미스터리)를 사이다·거래 쪽으로 당긴다 → 시안 D(노선 폐기) 방향으로의 미끄러짐 위험. B(8.5)보다 모티브 충실 반점(8.0). 심층 want 2단(6화)과 톤 게이트로 억제 가능하나 상시 감시 필요.
- **보존 반점**: 세로축(FS-0001) 자원 경쟁 + 1화 전면 재설계 + A1 재편으로 보존 7.0(B 7.5 하회). 발명은 계보 미등재 개체 인용으로 0 유지 가능.

**결정적 비교 — 하이브리드와의 관계**: 하이브리드는 **나침반 자체를 1화 감정 대상**으로 삼아 A의 최고 강점(1화 즉발 미션 드라이브)을 **더 싸게** 얻는다 — 단일 물체가 미션이자 세로축이므로 A의 이중 물체 희석·braid 상시 부하가 **원천 소거**된다. 즉 **A의 강점(미션 조기화·"위작에는 죽은 자가 없다" 시그니처)은 하이브리드에 표적 이식 가능한 반면, A의 약점(세로축 희석)은 하이브리드에 없다.**
- **정직한 총점**: A 31.5 < B 32.0 < C 34.5. A는 시장성 최고(9.5)이나 세로축 희석·톤 리스크로 보존·모티브 실점.
- **PM 관점 권고(재심사 대상)**: A를 **독립 채택**하기보다, A의 두 자산 — ⓐ 1화 "감정 의뢰=미션" 프레이밍, ⓑ "위작에는 죽은 자가 없다" 감정 클라이맥스 시그니처 — 를 **하이브리드의 1화(나침반 감정)에 표적 이식**하면, 세로축 희석 없이 미션 드라이브를 1화로 앞당길 수 있다. 최종 판정은 §4 PM 재심사 위임 1줄에 따라 PM이 별도 수행.

## §6 PM 최종 재심사 결론 (2026-07-15) — 결재 대상

**최종 추천안 = 「하이브리드 + A 이식」**

- 골격: 하이브리드 "B-엔진 위에 C-규율" (§4) 그대로 — 케이스 가로 엔진·순정보 불변식·중형 보상 실삽입·발명 0
- A에서 이식하는 자산 2개 (§4-A 권고 채택):
  1. **1화 = '감정 의뢰가 곧 미션' 프레이밍** — 다만 감정 대상을 보검이 아니라 **부러진 나침반 그 자체**로: "값을 매길 수 없는 물건의 진위·정체를 감정하라"가 1화부터 미션이 된다 (이중 미스터리 희석 원천 소거, A의 미션 조기화 효과는 그대로)
  2. **"위작에는 죽은 자가 없다. 진짜에는 반드시 누군가의 마지막이 남아 있다"** — 1화 클라이맥스 시그니처 라인·감정 철학으로 채택
- 보검(이시스-아스트라 자재)은 폐기가 아니라 **후속 케이스 유물**(2~10화 케이스 중 1건 + 10화 미션 결정화 후보)로 예약 — 작가 확인 칸 유지
- 판정 근거: A 자가 채점 31.5 < B 32.0 < C 34.5 이나 세 안 모두 단독 결함 상호 배타 — 조합이 전 결함 상쇄 (§4·§4-A 적대 검토 일치)

**결재 요청 (작가)**: ① 이 최종안 승인 여부 ② 승인 시 설계안(1~10 화별 뼈대)을 이 결론으로 갱신 후 재집필 배치 개시 — 뼈대 재확인을 원하시면 갱신본 먼저 올림.

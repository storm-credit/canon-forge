# 🏛️ 고대 문명 → 현세 계승표

> **[편찬 원칙]** 6대 고대 문명과 현세 세력·인물 간의 기술·유물·계보 연결을 정리한다.  
> `✅ 확정`: 원문 명시됨. `⚠️ 미확정`: 연결 불명확, 집필 시 결정 필요. `❌ 단절`: 알려진 계승자 없음.

## 1. 아르카나 제국 (마법공학 문명)

| 구분 | 내용 | 계승 여부 |
|------|------|-----------|
| **기술 계승** | 마법공학(Magitech), 룬 기계 공학, 공중 도시 설계 | ✅ **아르카나 잔재** 세력이 크림슨 대륙에서 계속 연구 |
| **세력 계승** | 마도제국(후기) → 마도제국 멸망 후 **아르카나 잔재** 결성 | ✅ 10대전쟁사 명시 |
| **유물 현황** | 부유 도시 메인 코어 청사진, 골렘 제조법, 기록 보관용 안구 이식체 | ✅ 마법협회가 탈취를 노림 |
| **에반 연결** | 에반의 소환술 루미에 연산 공식과 아르카나 연산 체계의 관련성 | ⚠️ 미확정 — 루미에 원천 문명 지정 필요 |

## 2. 잃어버린 황금 제국 (연금술 문명)

| 구분 | 내용 | 계승 여부 |
|------|------|-----------|
| **기술 계승** | 금속 연금술, 황금 이종 합금 | ❌ 계승 세력 미지정 |
| **세력 계승** | 제국 완전 소멸 — 잔재 세력 없음 | ❌ |
| **유물 현황** | 황금 이종 합금 조각(기억의 씨앗), 7대 연금 성물 | ✅ 약점 노출 거울 등 일부 유물 현존 기록 있음 |
| **에반 연결** | — | ❌ 미연결 |
| **⚠️ 보강 필요** | 황금제국 기술을 잇는 현세 세력 없음 — 마법협회 또는 철의 금융동맹 계승 검토 권장 | |

## 3. 태고의 정령왕국 (대자연 문명)

|------|------|-----------|
| **기술 계승** | 정령 교신, 자연 마법 | ✅ 에테르 대륙 **정령연합** 및 **시조의 엘프** |

| **에반 연결** | — | ❌ 미연결 |

## 4. 운명의 사원 문명 (신권정치 문명)

| 구분 | 내용 | 계승 여부 |
|------|------|-----------|
| **기술 계승** | 예언 연산 체계, 신탁 해석법 | ✅ **운명의 교단 (Temple of Fate)** — 왜곡된 형태로 계승 |
| **세력 계승** | 사원 문명 → 성국 내 운명의 교단 | ✅ (교리 오독·왜곡 포함) |
| **유물 현황** | 7대 저주 유물 (예언자의 찢겨진 눈물 등) | ✅ 현재 유적 내 정체된 시간존에 보존 |
| **에반 연결** | 모이라이(에반의 미등록 오류)와 운명 연산 체계 관련성 | ⚠️ 미확정 — 에반이 사원 문명의 예언 체계에 등록되지 않은 이유 명시 필요 |

| **에반 연결** | 에반의 이계 출신과 차원 균열의 관계 | ⚠️ 미확정 |

| **⚠️ 보강 필요** | 계승자 전무 — 해양 대륙 심해 세력 또는 이단파와 연결 검토 권장 | |

# 연대기 (Timeline & Cataclysms)

> *"역사는 결코 조용하지 않았다. 하나의 사건이 세계의 흐름을 바꾸었다."*

> Phase 0 · 2차 정리 · 2026-06-16  
> 원본: `01-16. 연대기/` + `01-9. 전쟁사/` (총 19개 파일)

---

## §1. 시대 구분

| 시대 | 기간 | 핵심 사건 | 신의 개입 |
|---|---|---|---|
| **창세기** | 태초 | 창세전쟁·심연 생성·세계 형성 | ✅ 신들이 직접 전쟁 |
| **신화시대** | 태초~-5000년 | 신들의 전쟁·고대 문명 번영 | ✅ 신들이 직접·간접 개입 |
| **멸망과 재건** | -5000~-500년 | 제2·3·4대격변·고대 문명 연속 붕괴 | 🟡 약화 중 |
| **현대** | -500년~현재 | 성국·왕국연합·자유도시·솔라리안 성립 | ❌ 침묵 |
| **황혼기 (임박)** | 현재~미래 | 제5대격변 조짐·차원 균열 확산 | ❌ 인간만 남겨짐 |

## §2. 5대 격변 연표

### 제1대격변 — 태초의 붕괴 (창세전쟁)
- **원인**: 태초의 신들 간 전쟁. 네메시스의 몸이 찢겨 심연 생성
- **결과**: 최초의 세계 붕괴 → 현재 아스트라리스 형성
- **잔재**: 창세전쟁의 상흔이 대지 깊숙이 남아 차원 균열의 근원이 됨

---

### 제2대격변 — 마도제국의 멸망 (-5000년)
- **원인**: 아르카나 제국의 금단 마법 — 역공압 흡수 프로토콜로 시민 300만의 생명력 강제 추출
- **결과**: 제국 붕괴. 서쪽 대륙 전역이 도넛 모양 크레이터로. **크림슨 대륙 황무지화**
- **잔재**: 아르카디아 폐허 = 현재도 마나 진공 지대·학살 기계 작동 중. 유물이 지상으로 튕겨져 나옴
- **연계**: [[고대문명]] §아르카나 제국

---

### 제3대격변 — 심연의 침공 (-2000년)
- **원인**: 심연의 군단이 차원의 틈을 열어 대륙 침공
- **결과**: 수많은 왕국 멸망. **오벨리스크 대륙 생성** (심연 에너지의 지형 변형)
- **잔재**: 심연 방벽 체계 수립. 봉인 수호단 창설 기원. 현재도 오벨리스크에는 심연 에너지 잔류

---

### 제4대격변 — 천공의 전쟁 (-1000년, 신들의 내전)
- **원인**: 은빛 옥좌의 신들이 남은 영맥을 독식하기 위해 우주에서 상잔
- **결과**: 일부 신 사망·봉인. **신들의 힘 대규모 약화 → 신들의 종말 시대 진입**
- **잔재**: 성전이 벌어진 국경 지대는 현재도 물리법칙이 뒤집히는 왜곡 지대 (탐험 던전)
- **의미**: 이 전쟁 이후 신은 인간의 기도에 답하지 않는다

---

### 제5대격변 — 조짐 (현재 진행)
- **전조**: 차원 균열 증가·심연 활동 활발·성국 내 이단 급증
- **예측 시나리오**:
  - 심연의 대침공 (확률 높음)
  - 차원의 균열 확산 (확률 높음)
  - 신들의 전면전 (중간)
  - 마법 대폭발 (중간)
- **핵심 변수**: 소환사 에반 라크라시스 — 모이라이 시스템 미등록 변칙 존재

## §3. 3대 기술 참사

### 마도 반란 (Magitech Rebellion)
- **배경**: 순혈 마법사와 기공사의 기술 패권 대립
- **사건**: 아르카나 요새 폭발로 마법사 토벌대 4천 명 증발
- **결과**: 마력 광해병 창궐. 아르카나 공학회 설립. 기공 기술 규제 시작

### 마력 전쟁 (Mana War)
- **배경**: 성국의 십자군 5만 명 vs 자유도시 마법과학 연합군
- **사건**: 딥 포지 참사 — 자아 융합 마도포의 제어 코어 붕괴. 하룻밤에 10만 명 전사
- **결과**: 자동화 병기 영구 금지 조약. 마도포 폐기. 마법과 무기 기술 분리 원칙 확립

### 어비스 대재앙 (Abyssal Catastrophe)
- **배경**: 네크로폴리스에서 차원 마도 융합로 실험 강행
- **사건**: 에텔론 붕괴 — 차원문 역류, 심연의 마수 대량 침입. 연구원 3천 명 전멸
- **결과**: 영구적 차원의 흉터 형성. 해당 지역 현재도 심연 마수 출몰 지대

## §4. 주요 전쟁사 (3시대 20+개)

### 창세와 분쟁의 시대 (태초~제국력 1150년)

| 전쟁 | 교전 세력 | 결과 |
|---|---|---|
| 창세 대전 | 원초 신들 간 전쟁 | 세계 형성·심연 탄생 |
| 고대 문명 대분열 | 6대 고대 문명 내부 붕괴 | 각 문명 연속 멸망 |
| 운명의 전쟁 | 신탁 세력 vs 자유의지 세력 | 운명의 사원 문명 삭제 |
| 고대 엘프 전쟁 | 고대 엘프 vs 인간 연합 | 엘프 대륙 분리·은둔화 |
| 심해 성전 | 육상 세력 vs 심해 생체군집 | 심해 군집 붕괴·산호 미궁 고립 |
| 용의 전쟁 | 용족 vs 인간 마법사 연합 | 드래곤 세력 대폭 축소 |
| 왕국연합 통합 전쟁 | 에테르 대륙 소왕국들 | 왕국연합 성립 |
| 마도제국의 멸망 | 아르카나 내부 자멸 | 제2대격변 = 크림슨 황무지화 |
| 그림자 첩보 대전 | 비밀 조직 간 정보전 | 진실의 눈·밤의 계약단 등 비밀 조직 성립 |
| 금화 전쟁 | 황금제국 잔재 세력 vs 신흥 세력 | 배금주의 국가 체제 붕괴 |

### 대륙 격돌과 확장의 시대 (제국력 1200~1800년)

| 전쟁 | 결과 |
|---|---|
| 오크 대침공 | 왕국연합 방벽 체계 확립 |
| 마법사 해방 전쟁 | 마법사 계급 독립 |
| 솔라리안 내전 | 솔라리안 제국 현 형태 확립 |
| 피의 전쟁 | 오벨리스크 대륙 세력 재편 |
| 제2차 성전 | 성국 교단 국경 확장 |
| 해상 패권 전쟁 | 황금 함대 해상 패권 장악 |
| 제1차 심연 대전 | 봉인 수호단 결성 |
| 천명제국 정복 전쟁 | 솔라리안 크림슨 대륙 통일 시작 |
| 망자의 전쟁 | 오벨리스크 망자의 왕국 영토 확장 |

### 심연의 대침공과 전란의 시대 (제국력 1800년~현재)

- 해골왕의 반란
- 제2차 심연 대침공
- 현재 7개 국지전 동시 다발:
  - 사막 전쟁 (크림슨)
  - 정령 전쟁 (에테르)
  - 지하 전쟁 (프로스트 드워프)
  - 해상 패권 전쟁 (해양)
  - 서부 전선 (왕국연합 vs 성국)
  - 솔라리안 황위 계승전
  - 심연 침식 전쟁 (전 대륙)

## §5. 헌터 길드 3대 재앙

### 검은 태양의 밤 (300년 전)
- 정예 헌터 1,000명 투입 vs 검은 태양
- 생환자 12명. 흑월단(黑月團) 설립 → 성국에 파문당함
- 현재도 흑월단은 성국의 이단 명단에 등재

### 심연의 용 토벌전 (200년 전)
- 마스터 헌터 5명 투입. 용 토벌 성공 후 전원 저주로 절명
- "신화의 파괴자(Myth Breaker)" 호칭 확립. 최상위 위험 등급 레이드 분류 시작

### 죽음의 군단 사건 (기록 유실기)
- 망자의 사냥단 투입. 신성 불꽃과 진동 마법으로 종결
- 언데드 전문 헌터 직책 신설. 오벨리스크 대륙 언데드 격리 구역 지정

## §7. 5대 예언

1. **태양이 검게 물들리라** — 성국과 심연 군단의 최종 전쟁 예고
2. **마지막 제국의 몰락** — 크림슨 대륙 최강국 붕괴 예고
3. **심연의 문이 열리리라** — 심연 존재들의 현실 전면 침입
4. **영웅과 악마의 춤** — 시대를 끝낼 영웅과 악마의 등장
5. **시간의 고리** — 과거의 실수 반복, 멸망의 위기

## §8. 숨겨진 역사

신들이 공식 역사에서 감춘 것들:

- **태초의 신들의 전쟁** — 공식 기록보다 훨씬 치열했음. 신들 다수 영구 소멸
- **심연 신들의 배신** — 은빛 옥좌 신들 중 일부가 심연과 거래
- **마도제국의 몰락 배후** — 신들이 에너지 추출 기술을 의도적으로 방치했다는 의혹
- **잊혀진 종족들** — 인간·엘프·드워프 이전에 존재한 고대 종족 기록 말소

**현존 비밀 조직들**:
| 조직 | 목적 |
|---|---|
| 진실의 눈 | 역사의 진실 조사·복원 |
| 밤의 계약단 | 금단 마법 연구 |
| 운명의 감시자 | 운명의 서 해독 |
| 황혼의 서고 | 신들이 숨긴 기록 보관 |

## §9. 세력별 현재 전략

### 에테르 대륙
- **성국**: 빛의 방벽 조약 (왕국연합), 면죄부 무역 협정 (자유도시). 십자군 준비 중
- **왕국연합**: 강철과 금화의 맹약 (자유도시). 성국과 위태로운 군사 연대
- **자유도시연합**: 세계의 채권자. 양측 무기 판매로 중립 유지
- **마법협회**: 성국과 냉전. 잊혀진 자들과 비밀 협약
- **정령연합**: 훼손된 지맥 수복 집중. 비정규적 자연의 분노 응징

### 크림슨 대륙
- **솔라리안 제국**: 무자비한 정복지 통제. 고대 마력로 강탈 시도
- **아르카나 제국 잔재**: 결사항전의 마법 요새화. 금기를 넘어선 힘 개방
- **붉은 사막 부족**: 게릴라 전술. 블랙 마켓 거점화
- **잊혀진 서고단**: 은폐된 심연 속 폐관 수련. 지식의 무기화

### 외곽 대륙 (프로스트·오벨리스크·해양)
- **딥 포지 왕국 (드워프)**: 용광로 요새화. 기계갑 양산
- **심연의 군단**: 잠행과 내부 침식. 변이 흑마법 진화 중
- **망자의 왕국**: 죽음의 포식과 영토 팽창. 맹독의 안개 살포
- **봉인의 수호자**: 뼈를 깎는 봉인 유지. 극단적 자원 소모

---

## §10. 5대 미래 전란 시나리오

1. **서부 전선의 격돌** — 왕국연합 vs 성국. 현재 팽팽한 소모전
2. **크림슨의 마력 폭주 전쟁** — 솔라리안 제국 vs 아르카나 잔재. 마나 역류 현상 발생
3. **대해상 패권의 날** — 해적 연합 vs 황금 함대 vs 심연. 글로벌 경제 마비
4. **대자연의 피눈물** — 정령·설원 vs 기계 제국. 세계 마나 고갈 위협
5. **최후의 종언** — 심연의 여왕 릴리스 vs 인류 연합. 라그나로크 클라이맥스

---

## §11. 연대기 속 탐험의 역할

매 대격변마다 **세계는 변형되고 탐험 불가 지대가 생겨났다**:

| 탐험지 | 격변 | 현재 위험 |
|---|---|---|
| 아르카디아 폐허 | 제2대격변 | 마나 진공·학살 기계 |
| 어비스 대재앙 흉터 지대 | 3대 기술 참사 | 심연 마수 상시 출몰 |
| 신들의 전쟁터 | 제4대격변 | 물리법칙 왜곡 지대 |
| 차원 균열 지대 | 현재 진행 | 시공간 불안정 |

금단의 땅들을 탐험하는 것이 곧 세계의 진실에 가까워지는 유일한 방법. **역사를 직접 걸어야 역사를 바꿀 수 있다.**

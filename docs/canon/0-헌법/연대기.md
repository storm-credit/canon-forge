## §1-A. 신화: 최초의 사냥과 황혼의 창 (The First Hunt)

> 창세전쟁 직후, 헌터 길드와 몬스터 경제의 신화적 기원. → [[직업체계]] 몬스터 헌터

- **원시의 포식 (The Primal Devouring)**: 창세전쟁 직후 찢어진 하늘의 틈새에서 아스트라리스 최초의 돌연변이들이 쏟아진 최초의 몬스터 웨이브 시대. 신들은 질서(Ordos) 수립에 몰두해 필멸자들의 학살을 관망했다. 당시 마법은 조악했고 철제 무기는 거수 가죽에 흠집도 못 냈다.
- **황혼의 창 카이론 (Kairon)**: 방벽 위에 홀로 선 이름 없는 전사. 마지막 숨을 몰아쉴 때 운명의 신 **테네브리스**가 현현해 치유의 축복 대신 저주받은 창을 내렸다.
- **황혼의 창 (Spear of Twilight)**: 찌르는 순간 짐승의 **존재 확률 자체를 왜곡해 소멸시키는** 병기이자, 술자의 생명력을 연료로 태우는 저주받은 매개체. 테네브리스의 신탁은 축복이 아닌 형벌 — *"균형을 갉아먹는 자들의 목을 잘라라. 영원히."*
- **제1군주 《어둠을 뒤덮는 자》 척살**: 카이론의 첫 사냥감은 반경 100km의 생명력을 흡수하던 심연의 첫 번째 군주. 7일간의 사투 끝에 심방을 꿰뚫자 최초의 균열이 닫혔다. 몬스터가 **극단적 물리력으로 파괴될 수 있음**을 증명한 사건이자, 몬스터의 피·코어(Core)가 인간 경제에 떨어진 첫 기원.
- **헌터 길드의 기원**: 카이론은 신성한 영웅이 아니라 **'학살의 기술'**을 인간에게 전수했다 — 가죽으로 방어구를, 뼈로 무기를 만드는 법. 신에 의존하지 않고 전술로 짐승을 도살하는 **몬스터 헌터** 집단의 시초. 카이론 본인은 훗날 황혼의 창과 함께 흔적 없이 사라졌다.

> **에픽 훅**: 황혼의 창은 현재 [[신앙체계|성국]]과 마도제국 잔재가 혈안이 되어 찾는 궁극의 병기. 재출현 시 헌터 길드와 대륙 패권 지도가 뒤집힌다. ⚠️ 이단설(아나테아의 반박): 창이 테네브리스가 아닌 심연 자체의 산물이라는 가설도 존재.

## §2. 5대 격변 연표

### 제1대격변 — 태초의 붕괴 (창세전쟁)
- **원인**: 태초의 신들 간 전쟁. 네메시스의 몸이 찢겨 심연 생성
- **결과**: 최초의 세계 붕괴 → 현재 아스트라리스 형성
- **잔재**: 창세전쟁의 상흔이 대지 깊숙이 남아 차원 균열의 근원이 됨

---

### 제2대격변 — 마도제국의 멸망 (-5000년)
- **원인**: 아르카나 제국의 금단 마법 — 역공압 흡수 프로토콜로 시민 300만의 생명력 강제 추출
- **결과**: 제국 붕괴. 서쪽 대륙 전역이 도넛 모양 크레이터로. **크림슨 대륙 황무지화**
- **잔재**: 아르카디아 폐허 = 현재도 마나 진공 지대·학살 기계 작동 중. 유물이 지상으로 튕겨져 나옴
- **연계**: [[고대문명]] §아르카나 제국

---

### 제3대격변 — 심연의 침공 (-2000년)
- **원인**: 심연의 군단이 차원의 틈을 열어 대륙 침공
- **결과**: 수많은 왕국 멸망. **오벨리스크 대륙 생성** (심연 에너지의 지형 변형)
- **잔재**: 심연 방벽 체계 수립. 봉인 수호단 창설 기원. 현재도 오벨리스크에는 심연 에너지 잔류

---

### 제4대격변 — 천공의 전쟁 (-1000년, 신들의 내전)
- **원인**: 은빛 옥좌의 신들이 남은 영맥을 독식하기 위해 우주에서 상잔
- **결과**: 일부 신 사망·봉인. **신들의 힘 대규모 약화 → 신들의 종말 시대 진입**
- **잔재**: 성전이 벌어진 국경 지대는 현재도 물리법칙이 뒤집히는 왜곡 지대 (탐험 던전)
- **의미**: 이 전쟁 이후 신은 인간의 기도에 답하지 않는다

---

### 제5대격변 — 조짐 (현재 진행)
- **전조**: 차원 균열 증가·심연 활동 활발·성국 내 이단 급증
- **예측 시나리오**:
  - 심연의 대침공 (확률 높음)
  - 차원의 균열 확산 (확률 높음)
  - 신들의 전면전 (중간)
  - 마법 대폭발 (중간)
- **핵심 변수**: 소환사 에반 라크라시스 — 모이라이 시스템 미등록 변칙 존재

### 대격변을 둘러싼 세력

대격변은 **폭주 물결(Flux Tide)**이 극대화될 때 발생한다. 이를 두고 두 진영이 대립한다.

| 입장 | 세력 | 목적·활동 |
|---|---|---|
| **막으려는 자** | 성국 | 신성 마법 강화·신탁 조사로 격변 저지 |
| | 마법협회 | 마법 폭주 방지·차원 균열 연구·금단 마법 봉인 |
| | 왕국연합 | 방어 요새 건설·군사 동맹으로 심연 침공 저지 |
| | **천상의 서고** | 과거 대격변 기록을 해석해 해결책 탐색 (cf. 비밀조직 '황혼의 서고'와 별개) |
| **조장하는 자** | 심연의 군단 | 심연 부활·차원 균열 확장·어둠의 마법 확산 |
| | **로토스를 따르는 자들** | 신들의 질서 파괴·신전 파괴·종교 박멸로 새 세계 건설 |
| | **황금 손아귀** | 세계 붕괴 시 최대 이익 노림. 경제 조작·전쟁 조장 |

## §3. 3대 기술 참사

### 마도 반란 (Magitech Rebellion)
- **배경**: 순혈 마법사(혈통주의)와 기공사(기계주의)의 기술 패권 대립. 기공사들은 끓는 물과 마석을 캡슐에 압축한 동력 기관 **마법 증기 엔진**을 발명, 순혈 마법사 일족은 이들을 "신성한 마나를 모독하는 천민"으로 규정해 공방을 산 채로 불태움.
- **사건**: 기공사 최대 거점 아르카나 요새 함락 — 포위·전원 처형 선고를 받은 기공사들이 투항 대신 자폭. 메인 방벽 코어 마력 네트워크에 에테르 폐기물을 역류시켜 수소 융합 폭발. 순혈 마법사 토벌대 4천 명이 피부·내장이 녹아 증발.
- **결과**: 마력 광해병(Mana Sickness) 창궐(오염 마나 낙진 → 폐부에 검은 마력 결정). **이사크 폰 벨렌**(아르카나 공학회 초대 회장)이 지하 벙커로 도주해 세계의 흑막 **아르카나 공학회** 설립. 톱니바퀴 소지자조차 효수하는 광적 기계 혐오 합법화. 지하 암시장에서 강제 인체 연성·키메라 제조(→ 마르코프 생체 폭주의 기원) 시작.

### 마력 전쟁 (Mana War)
- **배경**: 마도 반란 수십 년 후, 비대해진 마법과학 진영을 위협으로 본 **성국**의 성기사단이 전 대륙 마도 공방 극비 타격 작전 **「레오니스 포고령」**(레오니스 3세 발동)을 선포. 성국 십자군 5만 명 vs 자유도시 마법과학 연합군.
- **사건**: 크림슨 마도 요새 공방전 — 자유도시군이 **자아 융합 마도포**와 **AI 무장 군단(Auto-Magi-Warrior)**을 최초 실전 배치. 3일째 밤, 학살당한 성기사들의 피·원념이 통제 코어에 과부하를 일으켜 메인 프레임이 모든 생명체의 혈류·마나를 오염물로 재규정 → 포신이 거꾸로 돌아 아군 사령부를 난사한 **딥 포지 참사**. 하룻밤에 양측 10만 명이 자기 강철에 도살됨.
- **결과**: 자아 갱신형 지능형 자동 병기 생산 영구 금지 국제법. 참사 터는 자가 조립 고철 언데드가 굴러다니는 **크림슨 데스존**(최악의 금지구역)으로. 이후 병기는 반드시 인간이 뇌파 접속으로 직접 조종하는 조종사 강제 연결 페널티 유지.

### 어비스 대재앙 (Abyssal Catastrophe)
- **배경**: 자원 한계에 부딪힌 거대 상단·학자들이 공허 차원의 무한 에너지를 펌핑하는 **차원 마도 융합로(Dimensional Core)** 연구 강행. 밀수한 고농축 **심연의 파편(Fragment of Abyss)**을 촉매로 차원벽에 바늘구멍을 냄. 실험 장소는 불사·생체연금이 성행하던 타락 구역 **네크로폴리스(Necropolis)**.
- **사건**: 출력을 극한으로 올린 순간 융합로 내부 압력이 역전, 좌표계가 붕괴하며 수십 미터짜리 **에텔론 Rift**(검은 우주 균열)가 찢어짐. 차원의 틈이 포식자들에게 현실의 좌표를 송신 → 이계 진흙 괴사체와 눈먼 도살자 떼가 쇄도. 돔 내부 연구원 3천 명이 산 채로 내장이 파먹힘.
- **결과**: 균열은 성국 정화 기사단·대마도사들이 목숨으로 봉인할 때까지 일주일간 열려 대륙 절반에 돌연변이 가스를 토함. **마법공학+심연 역학 결합 연구는 즉결 처형 1순위**로 등록. 봉인 구역은 중력이 뒤집히는 사망 지대(Dead Zone)로 잔존. 심연 가스 피폭 생존자(비늘화·영혼 융해)는 **심연 피폭자 격리 정책**으로 수용소에 박해됨.

## §5. 헌터 길드 3대 재앙

### 검은 태양의 밤 (300년 전)
- 정예 헌터 1,000명 투입 vs 검은 태양
- 생환자 12명. 흑월단(黑月團) 설립 → 성국에 파문당함
- 현재도 흑월단은 성국의 이단 명단에 등재

### 심연의 용 토벌전 (200년 전)
- 마스터 헌터 5명 투입. 용 토벌 성공 후 전원 저주로 절명
- "신화의 파괴자(Myth Breaker)" 호칭 확립. 최상위 위험 등급 레이드 분류 시작

### 죽음의 군단 사건 (기록 유실기)
- 망자의 사냥단 투입. 신성 불꽃과 진동 마법으로 종결
- 언데드 전문 헌터 직책 신설. 오벨리스크 대륙 언데드 격리 구역 지정

---

## §6. 현재 갈등 (동시 다발 8개)

| 갈등 | 교전 세력 | 현황 |
|---|---|---|
| 서부 전선 | 왕국연합 vs 성국 | 팽팽한 소모전 |
| 크림슨 재건 전쟁 | 솔라리안 제국 vs 아르카나 잔재 | 마나 역류 현상 발생 |
| 해상 패권 전쟁 | 해적 연합 vs 황금 함대 vs 심연 | 글로벌 경제 마비 위기 |
| 심연의 침공 | 전 대륙 연합 vs 심연 군단 | 점진적 심연 침식 |
| 신성 vs 자유 | 태양신 교단 vs 무신론 세력 | 이단 재판 급증 |
| 황위 계승전 | 솔라리안 제국 내부 | 암살·쿠데타 반복 |
| 드워프 왕국 단절 | 전통파 vs 개혁파 | 기계갑 도입 여부로 분열 |
| 차원 균열 확대 | 마법협회 vs 자연 세계 | 금단 마법 부활 시도 |

## §7. 5대 예언

1. **태양이 검게 물들리라** — 성국과 심연 군단의 최종 전쟁 예고
2. **마지막 제국의 몰락** — 크림슨 대륙 최강국 붕괴 예고
3. **심연의 문이 열리리라** — 심연 존재들의 현실 전면 침입
4. **영웅과 악마의 춤** — 시대를 끝낼 영웅과 악마의 등장
5. **시간의 고리** — 과거의 실수 반복, 멸망의 위기

## §8. 숨겨진 역사

신들이 공식 역사에서 감춘 것들 — 각 진실마다 이를 은폐한 신이 명시돼 있다:

| 숨겨진 역사 | 내용 | 감춘 신 |
|---|---|---|
| **태초의 신들의 전쟁** | 공식 기록보다 훨씬 치열했음. 신들 다수 영구 소멸 | 크로노스 · 오르도스 |
| **심연 신들의 배신** | 일부 신들이 인간을 배반하고 심연과 거래해 타락한 존재가 됨 | 테네브리스 · 모르디아 |
| **마도제국의 몰락 배후** | 단순한 전쟁이 아니라 신들의 의도적 개입으로 멸망 | 솔라리스 · 소피아 |
| **잊혀진 종족들** | 인간·엘프·드워프 이전에 존재한 초월적 고대 종족 기록 말소 | 녹스 · 마리노스 |

**현존 비밀 조직들**:
| 조직 | 목적 | 활동 지역 |
|---|---|---|
| 진실의 눈 | 역사의 진실 조사·복원 | 자유도시연합·마법협회 |
| 밤의 계약단 | 금단 마법 연구 | 마법협회·오벨리스크 대륙 |
| 운명의 감시자 | 운명의 서 해독 | 성국·마도제국 잔재 |
| 황혼의 서고 | 신들이 숨긴 기록 보관 | 불명 |

## §9. 세력별 현재 전략

### 에테르 대륙
- **성국**: 빛의 방벽 조약 (왕국연합), 면죄부 무역 협정 (자유도시). 십자군 준비 중
- **왕국연합**: 강철과 금화의 맹약 (자유도시). 성국과 위태로운 군사 연대
- **자유도시연합**: 세계의 채권자. 양측 무기 판매로 중립 유지
- **마법협회**: 성국과 냉전. 잊혀진 자들과 비밀 협약
- **정령연합**: 훼손된 지맥 수복 집중. 비정규적 자연의 분노 응징

### 크림슨 대륙
- **솔라리안 제국**: 무자비한 정복지 통제. 고대 마력로 강탈 시도
- **아르카나 제국 잔재**: 결사항전의 마법 요새화. 금기를 넘어선 힘 개방
- **붉은 사막 부족**: 게릴라 전술. 블랙 마켓 거점화
- **잊혀진 서고단**: 은폐된 심연 속 폐관 수련. 지식의 무기화

### 외곽 대륙 (프로스트·오벨리스크·해양)
- **딥 포지 왕국 (드워프)**: 용광로 요새화. 기계갑 양산
- **심연의 군단**: 잠행과 내부 침식. 변이 흑마법 진화 중
- **망자의 왕국**: 죽음의 포식과 영토 팽창. 맹독의 안개 살포
- **봉인의 수호자**: 뼈를 깎는 봉인 유지. 극단적 자원 소모

## §10. 5대 미래 전란 시나리오

1. **서부 전선의 격돌** — 왕국연합 vs 성국. 현재 팽팽한 소모전
2. **크림슨의 마력 폭주 전쟁** — 솔라리안 제국 vs 아르카나 잔재. 마나 역류 현상 발생
3. **대해상 패권의 날** — 해적 연합 vs 황금 함대 vs 심연. 글로벌 경제 마비
4. **대자연의 피눈물** — 정령·설원 vs 기계 제국. 세계 마나 고갈 위협
5. **최후의 종언** — 심연의 여왕 릴리스 vs 인류 연합. 라그나로크 클라이맥스

## §11. 연대기 속 탐험의 역할

매 대격변마다 **세계는 변형되고 탐험 불가 지대가 생겨났다**:

| 탐험지 | 격변 | 현재 위험 |
|---|---|---|
| 아르카디아 폐허 | 제2대격변 | 마나 진공·학살 기계 |
| 어비스 대재앙 흉터 지대 | 3대 기술 참사 | 심연 마수 상시 출몰 |
| 신들의 전쟁터 | 제4대격변 | 물리법칙 왜곡 지대 |
| 차원 균열 지대 | 현재 진행 | 시공간 불안정 |

금단의 땅들을 탐험하는 것이 곧 세계의 진실에 가까워지는 유일한 방법. **역사를 직접 걸어야 역사를 바꿀 수 있다.**

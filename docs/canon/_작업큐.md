## 불변 원칙 (전 세션 공유)

1. **줄이지 말 것**: 보일러플레이트(YAML frontmatter·해시태그·에반 섭리 블록)만 제거, 본문 전수 보존
2. **3중 게이트**: ① 축약점검 ≥85% · ② 원본대조 누락 0 · ③ `[[직업체계|` 등 체계오연결 grep 0
3. **NAV_ONLY 가드**: 스크립트 재실행 시 캐논 파일 덮어쓰기 방지
4. **[D] 정독**: grep 키워드 체크 아님 — 에이전트 4개 병렬, 실제 본문 정독, 모순 즉시 해소
5. **슬러그 사전**: `링크복원.build_canon_slugs()` 608개 슬러그 자동 매칭 (`[[A/B|C]]` 복원)

- 티켓 #3·#5 실행 시: 수선 sonnet(어휘·문장 수술) → 기계 게이트 → 게이트3 표본 → 1~7화 미니 패널 재독 → 커밋. 복선 장부 무영향(사건 보존).
- 일괄 실행 시점: 작가 "고쳐" 한마디 (통독 끝까지 모아서 해도, 지금 해도 무방).
